# 经济学知识体系

> 面向成年人自学的经济学大类知识体系
> 
> 覆盖范围：经济学、金融学，以及与现实决策高度相关的会计、产业、政策、数据分析、商业与制度基础

---

## 一、定位说明

这份知识体系不是给考试备考用的“教材目录”，而是给成年人自学用的**理解型学习地图**。

成年人学经济学，最常见的问题不是“找不到概念”，而是：

- 学了很多术语，却不会解释现实问题；
- 学了很多财经新闻词汇，却不知道它们之间是什么关系；
- 微观、宏观、金融、投资、会计、政策各学一点，但始终拼不成整体框架；
- 知道一些结论，却不清楚这些结论在什么条件下成立。

因此，本体系按“**决策—市场—增长—金融—企业—政策—数据**”的主线组织，而不是简单照搬大学教材章节。

学习目标分三层：

1. **会解释现实**：能用经济学语言解释价格、收入、通胀、利率、失业、资产涨跌、企业决策等常见现象。
2. **会做判断**：面对财经话题时，能区分事实、机制、假设和立场，不被表面叙事带着走。
3. **会继续深入**：学完主干后，可以自然进入投资、公司金融、产业研究、政策分析、行为经济学等专题。

---

## 二、学习总路线

建议按下面的顺序学习：

1. **经济学思维起步**：先建立稀缺、选择、激励、机会成本、边际分析这些最基础的判断工具。
2. **微观主线**：理解个体、家庭、企业和市场如何做决策，价格如何形成，市场为什么有时有效、有时失灵。
3. **宏观主线**：理解总产出、通胀、失业、利率、财政、货币、汇率与经济周期之间的关系。
4. **金融主线**：理解货币、银行、债券、股票、基金、估值、风险和资产配置。
5. **企业与会计主线**：理解财务报表、盈利模式、现金流、融资决策和企业价值。
6. **政策与制度主线**：理解政府、央行、监管、税收、社保、国际贸易和产业政策。
7. **数据与研究主线**：学会看指标、读图表、识别样本偏差、理解相关与因果。

## 三、经济学思维线

### E1 稀缺、选择与机会成本

- 核心问题：资源有限时，为什么一切选择都有代价？
- 关键词：稀缺性、取舍、机会成本、显性成本、隐性成本
- 对应学案：[[经济学1-稀缺性与机会成本]]

### E2 激励、边际与理性选择

- 核心问题：人为什么会改变行为？为什么很多决策要看“多一单位值不值”？
- 关键词：激励、边际收益、边际成本、理性选择、约束条件
- 对应学案：[[经济学2-激励与边际决策]]

### E3 交易、分工与比较优势

- 核心问题：为什么合作和交换能提高总福利？为什么不是“谁更强谁就一定更占优”？
- 关键词：分工、交换、专业化、比较优势、贸易收益
- 对应学案：[[经济学3-分工交换与比较优势]]

## 四、微观经济学主线

### M1 需求、供给与价格机制

### M3 消费者选择

- 核心问题：收入有限时，消费者怎样在不同商品之间分配预算？

- 核心问题：企业怎样决定产量、价格、扩张和退出？

### M5 市场结构与竞争

- 核心问题：完全竞争、垄断、寡头、垄断竞争分别意味着什么？
- 对应学案：[[经济学8-市场结构与竞争]]

### M6 市场失灵与政府作用

## 十、成年人自学建议

### 1. 先学框架，再补细节

不要一开始就死抠模型推导。成年人自学更重要的是先建立：

- 谁在决策；
- 面临什么约束；
- 激励如何变化；
- 价格、数量、利率、收入、风险怎样传导。

### 2. 每学一个概念，都要问四个问题

1. 它解释什么现实问题？
2. 它成立需要什么条件？
3. 它最常见的误用是什么？
4. 它和哪些指标、政策、资产价格有关？

### 3. 经济学不是“永远正确的答案”，而是“有条件的解释框架”

很多争论不是谁懂不懂，而是：

- 假设不同；
- 时间尺度不同；
- 关注的群体不同；
- 对公平和效率的权衡不同。

### 4. 建议学习节奏

- 第一阶段：`1 → 2 → 3 → 4 → 7 → 9`
- 第二阶段：`10 → 11 → 12 → 13 → 14`
- 第三阶段：`15 → 16 → 17 → 18 → 19 → 20`
- 第四阶段：`21 → 22 → 23 → 24`
- 第五阶段：`25 → 26 → 27 → 28 → 29 → 30 → 31`

## 十一、提高路线课程

主线课程解决的是“先搭框架、能解释现实”的问题；提高路线解决的是“把框架推深，形成更强的理论、分析与研究能力”。

### A. 理论进阶线

- [[提高1-价格机制福利与一般均衡]]：从局部供需走向福利与一般均衡
- [[提高2-博弈论与策略互动]]：理解策略互动、价格战与谈判结构
- [[提高3-信息不对称与机制设计]]：从逆向选择走向制度设计

### B. 宏观与金融进阶线

- [[提高4-增长波动与政策规则]]：把长期增长、短期波动和政策规则放到统一框架里
- [[提高5-货币政策传导与金融周期]]：理解货币政策为什么有时有效、有时失灵
- [[提高6-固定收益与利率期限结构]]：系统理解债券、收益率曲线和利率风险
- [[提高7-估值建模与自由现金流]]：从贴现思想进入系统估值建模

### C. 商业与研究进阶线

- [[提高8-竞争优势商业模式与战略分析]]：从行业观察走向长期竞争优势判断
- [[提高9-计量思维与因果识别]]：从看数据走向评估证据质量

### 提高路线建议顺序

- 若你偏好理论：提高1 → 提高2 → 提高3
- 若你偏好宏观金融：提高4 → 提高5 → 提高6 → 提高7
- 若你偏好商业研究：提高8 → 提高9
- 若你希望形成完整分析能力：建议在主线完成后，至少选修 提高4、提高7、提高8、提高9

## 十二、扩展方向

在主干学完后，可以继续向下列方向扩展：

- 更形式化的中级微观与中级宏观；
- 金融工程、固定收益、衍生品；
- 会计分析与估值建模；
- 公共政策分析；
- 城市经济学、劳动经济学、行为经济学；
- 博弈论、计量经济学、因果推断；
- 中国经济专题、全球宏观专题、行业研究专题。

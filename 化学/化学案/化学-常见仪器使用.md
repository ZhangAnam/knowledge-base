# 化学学案：常见仪器使用

> 高中化学 | 实验方法线 E1

---

## 本课学什么

- 认识高中化学常见仪器的名称、用途和基本操作要求。
- 会区分可直接加热与不可直接加热的仪器。
- 能根据实验任务选择合适仪器，并避免常见操作错误。

## 原理

1. 化学实验仪器的使用既关系实验结果，也关系实验安全。
2. 不同仪器对应不同功能：反应、加热、量取、分离、收集、滴加、称量等。
3. 仪器选择错误，往往会导致测量偏差、样品污染甚至安全事故。

## 条件

| 仪器 | 主要用途 | 使用要点 |
|---|---|---|
| 试管 | 少量物质反应、加热 | 液体不超过容积 `1/3`，加热时管口勿对人 |
| 烧杯 | 配制、溶解、较大量反应 | 一般不能直接精准量取体积 |
| 量筒 | 量取液体体积 | 不能加热，读数看凹液面最低处 |
| 容量瓶 | 配制一定体积一定浓度溶液 | 不能溶解固体，需定容 |
| 胶头滴管 | 少量滴加液体 | 不能伸入容器内接触液体 |
| 分液漏斗 | 分液、控制滴加 | 活塞要先检查是否漏液 |

## 现象

正确使用仪器时：

- 量取体积读数清楚；
- 加热安全、受热均匀；
- 配制溶液浓度更准确；
- 反应与分离过程更规范。

## 方程式 / 操作关系

常见实验操作链：

```text
称量 → 溶解 → 转移 → 洗涤 → 定容 → 摇匀
```

## 易错点

- **量筒不能用来配制溶液**。
- **容量瓶不能直接加热，也不能长期贮液**。
- **胶头滴管不能倒置后平放**。
- **酒精灯熄灭不能用嘴吹，应使用灯帽盖灭**。

## 递进例题

### 例1：体积量取

量取 `20 mL` 稀盐酸，优先选什么仪器？

**答案**：量筒。

### 例2：配制溶液

配制 `250 mL` 一定浓度 NaCl 溶液，关键仪器是什么？

**答案**：250 mL 容量瓶。

### 例3：滴加液体

向试管中逐滴加入试剂，常用什么仪器？

**答案**：胶头滴管。

### 例4：分离液体

分离互不相溶的两种液体，常用什么仪器？

**答案**：分液漏斗。

## 练习

1. 量筒能否加热？
2. 容量瓶最核心的用途是什么？
3. 胶头滴管使用时能否伸入试管？
4. 熄灭酒精灯应怎样操作？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 不能 |
| 2 | 配制一定体积的一定浓度溶液 |
| 3 | 一般不能 |
| 4 | 用灯帽盖灭 |

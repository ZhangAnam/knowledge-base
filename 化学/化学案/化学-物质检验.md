# 化学学案：物质检验

> 高中化学 | 实验方法线 E2

---

## 本课学什么

- 掌握常见气体、常见离子和典型物质的检验思路。
- 会根据“试剂—现象—结论”建立规范表达。
- 能区分检验与鉴别，避免只记现象不看干扰条件。

## 原理

1. 物质检验是利用某些特征反应，通过明显现象判断物质或离子是否存在。
2. 检验要关注三要素：**所加试剂、实验现象、得到结论**。
3. 可靠检验不仅要看到现象，还要排除其他物质造成相同现象的可能。

## 条件

| 检验对象 | 常用试剂 | 典型现象 |
|---|---|---|
| `CO2` | 澄清石灰水 | 变浑浊 |
| `H2` | 点燃 | 发出淡蓝色火焰，罩冷干燥烧杯有水珠 |
| `NH3` | 湿润红色石蕊试纸 | 试纸变蓝 |
| `Cl^-` | 稀硝酸、`AgNO3` | 生成白色沉淀 |
| `SO4^{2-}` | 稀盐酸、`BaCl2` | 生成白色沉淀且不溶于酸 |

## 现象

- 白色沉淀、蓝色沉淀、红褐色沉淀；
- 产生有刺激性气味气体；
- 试纸变色；
- 火焰颜色或燃烧现象变化。

## 方程式

```chemical
CO2 + Ca(OH)2 = CaCO3↓ + H2O
Ag^+ + Cl^- = AgCl↓
Ba^{2+} + SO4^{2-} = BaSO4↓
NH3 + H2O ⇌ NH3·H2O
```

## 易错点

- **检验 `Cl^-` 时常先加稀硝酸酸化**，防止 `CO3^{2-}` 等干扰。
- **白色沉淀不一定都是同一种离子**，要看后续条件。
- 检验与鉴别不同：检验是确认有没有，鉴别是在多种物质中区分谁是谁。
- 现象描述要客观，不能直接把结论写成现象。

## 递进例题

### 例1：检验二氧化碳

如何检验某无色气体是否为 `CO2`？

**答案**：通入澄清石灰水，若变浑浊，说明可能含 `CO2`。

### 例2：检验氯离子

检验溶液中是否含 `Cl^-`，应如何操作？

**答案**：先加稀硝酸，再加 `AgNO3` 溶液，若生成白色沉淀，则说明含 `Cl^-`。

### 例3：检验铵根离子

检验 `NH4^+` 常用什么方法？

**答案**：加碱并加热，若放出刺激性气味气体且使湿润红色石蕊试纸变蓝，则说明有 `NH4^+`。

### 例4：区分检验和鉴别

“判断某瓶液体是否含硫酸根”和“区分三瓶无色溶液分别是什么”属于同一任务吗？

**答案**：不是，前者是检验，后者是鉴别。

## 练习

1. 检验 `SO4^{2-}` 常用哪类试剂？
2. 检验 `NH4^+` 时为什么常要加热？
3. 检验 `Cl^-` 前为什么要先酸化？
4. 现象和结论书写时应怎样区分？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 含 `Ba^{2+}` 的溶液，并常先酸化 |
| 2 | 促使 `NH3` 更易逸出 |
| 3 | 排除碳酸根等干扰 |
| 4 | 先写看到的现象，再下结论 |

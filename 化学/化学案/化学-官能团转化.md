# 化学学案：官能团转化

> 高中化学 | 有机化学线 O3

---

## 本课学什么

- 理解有机推断题中常见官能团转化关系。
- 会从“结构—性质—反应类型”角度梳理醇、醛、酸、酯之间的联系。
- 能用官能团变化分析合成路线中的关键步骤。

## 原理

1. 官能团是决定有机物性质和反应方向的核心结构单元。
2. 有机推断题常通过官能团的引入、消去、氧化、还原、加成、水解等实现转化。
3. 抓住“反应前后官能团怎样变”比死记整条路线更重要。

## 条件

| 转化方向 | 常见反应类型 | 典型提示 |
|---|---|---|
| 烯烃 → 醇 | 加成 | 与水加成 |
| 醇 → 醛/酸 | 氧化 | 条件常有氧化剂、催化加热 |
| 酸 + 醇 → 酯 | 酯化 | 浓硫酸、加热 |
| 酯 → 酸/醇 | 水解 | 酸性或碱性条件 |

## 现象

官能团转化题多以结构式和条件呈现，实验现象不是唯一重点，但常伴随：

- 溴水褪色；
- 银镜反应；
- 新制氢氧化铜出现砖红色沉淀；
- 酯香味等。

## 方程式 / 转化链

```text
烯烃 → 醇 → 醛 → 酸
酸 + 醇 ⇌ 酯 + 水
酯 + 水 → 酸 + 醇
```

## 易错点

- **把官能团转化理解成元素种类完全改变**，核心是结构单元变化。
- 醇氧化不一定一步到酸，要看条件和醇的种类。
- 酯化和水解互为逆过程，但条件不完全相同。
- 有机推断要重视“已知现象”和“分子式约束”。

## 递进例题

### 例1：链式转化

写出“乙烯 → 乙醇”所属反应类型。

**答案**：加成反应。

### 例2：氧化关系

乙醇可氧化生成什么，再进一步氧化生成什么？

**答案**：先生成乙醛，再生成乙酸。

### 例3：酯化本质

乙酸与乙醇反应生成乙酸乙酯，本质上是哪类官能团之间的转化？

**答案**：羧基和羟基共同参与形成酯基。

### 例4：推断题抓手

有机推断题中看到银镜反应，应优先联想到什么官能团？

**答案**：醛基。

## 练习

1. 烯烃和水加成常生成什么类型物质？
2. 酯水解后通常得到什么？
3. 醇、醛、酸之间常见什么关系？
4. 银镜反应常提示哪类官能团？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 醇 |
| 2 | 酸和醇 |
| 3 | 常可通过氧化逐步转化 |
| 4 | 醛基 |

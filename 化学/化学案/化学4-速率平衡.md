# 化学学案 4：速率平衡综合

> 高中化学 | 反应规律线 R6

---

## 本课学什么
- 把 **化学反应速率、化学平衡、图像分析** 放到同一框架中理解。
- 会判断条件变化后速率怎样变、平衡怎样移、最终组成怎样变。
- 会处理“先变快还是先移动”“瞬时变化与最终结果”这类综合题。

## 原理
1. 可逆反应开始后，正逆反应同时存在；开始时通常逆反应速率为 0，随后逐渐增大。
2. 达到平衡时，**正反应速率 = 逆反应速率**，但两者一般都不为 0。
3. 外界条件改变后，正逆反应速率会先发生 **瞬时改变**，再通过平衡移动达到新的相等状态。
4. 综合题的统一分析顺序是：**先看速率瞬时变化，再看平衡移动方向，最后看浓度、转化率、图像终态**。

## 条件
| 条件变化 | 速率瞬时变化 | 平衡移动判断 |
|---|---|---|
| 增大反应物浓度 | 正反应速率先增大 | 向正反应方向移动 |
| 减小生成物浓度 | 逆反应速率先减小 | 向正反应方向移动 |
| 增大压强（气体总数不同） | 正逆反应速率都增大 | 向气体总数减小方向移动 |
| 升高温度 | 正逆反应速率都增大 | 结合吸放热方向判断 |
| 加催化剂 | 正逆反应速率都增大 | 平衡不移动，只更快达到平衡 |

## 现象
| 体系 | 条件变化 | 现象或图像特征 |
|---|---|---|
| `2NO2 ⇌ N2O4` | 降温 | 棕色变浅，说明平衡向生成 `N2O4` 方向移动 |
| 合成氨体系 | 加压 | 氨的平衡含量增加，图像中重新建立新平衡 |
| `Fe^{3+} + SCN^- ⇌ Fe(SCN)^{2+}` | 加入 `Fe^{3+}` | 溶液血红色加深 |
| 平衡图像题 | 加催化剂 | 两条速率曲线同时升高并更快相交 |

## 方程式
```chemical
v = Δc / Δt
N2 + 3H2 ⇌ 2NH3
2NO2 ⇌ N2O4
Fe^{3+} + SCN^- ⇌ Fe(SCN)^{2+}
```

## 易错点
- 速率增大 **不等于** 平衡一定向正反应方向移动。
- 条件改变后先看 **瞬时速率变化**，不要直接跳到“最终浓度变化”。
- 催化剂能让体系更快达到平衡，但 **不能改变平衡组成**。
- 压强对平衡的影响只在 **气体总物质的量两边不同** 时才有明确方向性。

## 递进例题
### 例1：速率与平衡的区别
可逆反应达到平衡时，正逆反应是否停止？

**分析**：平衡是动态的，只是正逆反应速率相等。

**答案**：没有停止，正逆反应仍在进行。

### 例2：浓度改变
在 `N2 + 3H2 ⇌ 2NH3` 平衡体系中，突然增大 `H2` 浓度，正逆反应速率如何变化？平衡怎样移动？

**分析**：`H2` 是反应物，增大后正反应速率立即增大，逆反应速率瞬时不变；随后平衡向右移动。

**答案**：正反应速率先增大，逆反应速率开始瞬时不变后再增大；平衡向右移动。

### 例3：温度改变
反应 `2NO2 ⇌ N2O4` 正反应放热。若升高温度，颜色如何变化？

**分析**：升温有利于逆向吸热过程，`NO2` 增多。

**答案**：颜色加深，平衡逆向移动。

### 例4：催化剂图像题
某平衡体系达到平衡后加入催化剂，图像中正逆反应速率曲线应如何变化？

**分析**：正逆反应速率同时增大，但平衡组成不变。

**答案**：两条曲线同时上升，并较快再次达到相等；平衡位置不变。

## 练习
1. 判断：平衡时 `v正 = v逆 = 0`。
2. 在 `2SO2 + O2 ⇌ 2SO3` 中，增大 `SO2` 浓度后，平衡如何移动？
3. 对于正反应放热的可逆反应，升高温度后正逆反应速率都怎样变化？平衡向哪边移动？
4. 加入催化剂后，转化率是否一定改变？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | 错；平衡时正逆反应速率相等且通常不为 0 |
| 2 | 向正反应方向移动 |
| 3 | 都增大；平衡向逆反应方向移动 |
| 4 | 不一定；一般不改变平衡转化率，只改变达到平衡的快慢 |

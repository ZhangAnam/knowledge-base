# 化学学案：氯气及其化合物

> 高中化学 | 元素化合物线 E4

---

## 本课学什么
- 抓住氯气的 **强氧化性、氯水成分、漂白消毒、实验室制取**。
- 会区分 **干燥氯气、氯水、次氯酸盐** 的性质差异。
- 会根据条件判断氯气与金属、水、碱的反应产物。

## 原理
1. 氯原子最外层有 7 个电子，氯气易得电子，因而具有较强氧化性。
2. 氯气溶于水后部分发生反应：`Cl2 + H2O ⇌ HCl + HClO`，所以氯水中含有多种微粒。
3. 次氯酸 `HClO` 具有 **强氧化性、漂白性和杀菌消毒作用**，但不稳定，见光易分解。
4. 氯气与碱反应属于歧化反应，生成氯化物和次氯酸盐（或氯酸盐）。

## 条件
| 反应或性质 | 条件 | 典型结论 |
|---|---|---|
| `Cl2` 与金属 | 点燃或加热 | 生成高价金属氯化物较常见 |
| `Cl2` 与 `H2` | 光照或点燃 | 剧烈反应生成 `HCl` |
| `Cl2` 与水 | 常温 | 生成盐酸和次氯酸 |
| `Cl2` 与冷 NaOH | 常温 | 生成 `NaCl`、`NaClO` |
| `Cl2` 与热浓 NaOH | 加热浓碱 | 可生成 `NaCl`、`NaClO3` |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| 氯气本身 | 黄绿色，有刺激性气味，有毒 | 注意尾气处理 |
| 氯气通入湿润有色布条 | 布条褪色 | 体现 `HClO` 漂白性 |
| 干燥氯气通入干燥有色布条 | 一般不褪色 | 干燥氯气本身漂白性弱 |
| 实验室制氯气 | 生成黄绿色气体 | 常用浓盐酸与 `MnO2` 共热 |

## 方程式
```chemical
2Fe + 3Cl2 = 2FeCl3（点燃）
H2 + Cl2 = 2HCl（光照）
Cl2 + H2O ⇌ HCl + HClO
Cl2 + 2NaOH = NaCl + NaClO + H2O
3Cl2 + 6NaOH = 5NaCl + NaClO3 + 3H2O（热浓）
MnO2 + 4HCl(浓) = MnCl2 + Cl2↑ + 2H2O（加热）
```

## 易错点
- 使有色布条褪色的主要是 **`HClO`**，不是干燥氯气本身。
- 氯水成分复杂，既有分子 `Cl2`、`HClO`，也有离子 `H^+`、`Cl^-`、`ClO^-` 等。
- 铁与氯气反应通常生成 **`FeCl3`**，不是 `FeCl2`。
- 实验室制氯气要用 **浓盐酸**，并进行尾气吸收，不能直接排放。

## 递进例题
### 例1：漂白判断
为什么干燥氯气不能使干燥有色布条褪色，而湿润布条可以？

**分析**：只有在有水条件下才能生成具有漂白性的 `HClO`。

**答案**：因为湿润条件下氯气与水反应生成 `HClO`，干燥条件下这一过程不能有效发生。

### 例2：氯水成分
新制氯水中主要含有哪些分子和离子？

**分析**：由溶解和平衡关系判断。

**答案**：分子主要有 `Cl2`、`H2O`、`HClO`；离子主要有 `H^+`、`Cl^-`、`ClO^-`。

### 例3：碱液吸氯
氯气通入冷 NaOH 溶液时生成什么？

**分析**：氯元素一部分被还原成 `Cl^-`，一部分被氧化成 `ClO^-`。

**答案**：生成 `NaCl`、`NaClO` 和 `H2O`。

### 例4：定量关系
标准状况下 11.2 L `Cl2` 通入足量 NaOH 溶液，理论上生成多少 mol `NaClO`？

**分析**：`n(Cl2)=0.5 mol`，由方程式 `Cl2 + 2NaOH = NaCl + NaClO + H2O` 可知 1 mol `Cl2` 生成 1 mol `NaClO`。

**答案**：`0.5 mol`。

## 练习

2. 判断：新制氯水中只含 `Cl2` 和 `H2O`。
3. 湿润淀粉 KI 试纸遇氯气会出现什么现象？
4. 冷 NaOH 溶液吸收氯气的产物中，具有漂白和消毒作用的主要成分是什么？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `MnO2 + 4HCl(浓) = MnCl2 + Cl2↑ + 2H2O（加热）` |
| 2 | 错 |
| 3 | 先变蓝，后可能褪色 |
| 4 | `NaClO`（在水中对应次氯酸盐体系） |

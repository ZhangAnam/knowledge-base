# 高中化学：氯气

> 高中化学 | 元素化合物

---

## 📌 氯气的性质

### 物理性质
黄绿色刺激性气味，有毒

### 化学性质
- 与金属：2Fe + 3Cl₂ → 2FeCl₃
- 与非金属：H₂ + Cl₂ → 2HCl
- 与水：Cl₂ + H₂O ⇌ HCl + HClO
- 与碱：Cl₂ + 2NaOH → NaCl + NaClO + H₂O

---

## 📌 次氯酸

### 性质
- 不稳定性：2HClO → 2HCl + O₂↑
- 强氧化性：漂白、消毒
- 弱酸性：比碳酸弱

---

## 📌 氯气制备

### 实验室
MnO₂ + 4HCl → MnCl₂ + Cl₂↑ + 2H₂O

### 工业
2NaCl + 2H₂O → 2NaOH + H₂↑ + Cl₂↑

---

## 📝 典型例题

### 例1：计算
标准状况下，11.2L Cl₂ 通入足量NaOH溶液，生成NaCl和NaClO各多少mol？

> 📋 答案
> n(Cl₂) = 11.2/22.4 = 0.5 mol
> n(NaCl) = 0.5 × 2 = 1 mol
> n(NaClO) = 0.5 × 2 = 1 mol

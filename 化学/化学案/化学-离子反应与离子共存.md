# 化学学案：离子反应与离子共存

> 高中化学 | 反应规律线 R2

---

## 本课学什么
- 会判断电解质在水溶液中哪些要拆成离子、哪些不能拆。
- 会根据 **沉淀、气体、弱电解质、氧化还原** 判断离子能否大量共存。
- 会正确书写并检查离子方程式。

## 原理
1. 离子反应研究的是 **溶液中实际参加反应的微粒**，因此要先判断物质在溶液中的存在形式。
2. **强酸、强碱、可溶性盐** 一般拆成离子；**单质、氧化物、沉淀、气体、弱酸、弱碱、水** 一般不拆。
3. 两组离子若混合后能生成 **沉淀、气体、弱电解质**，或发生明显 **氧化还原反应、络合反应**，则不能大量共存。
4. 检查离子方程式时要同时看 **原子守恒、电荷守恒、得失电子守恒**。

## 条件
| 任务 | 条件 | 判断方法 |
|---|---|---|
| 是否拆离子 | 水溶液中、强电解质、可溶 | 写成离子 |
| 是否保留化学式 | 弱电解质、沉淀、气体、单质、水 | 不拆 |
| 判断共存 | 是否会反应 | 看沉淀、气体、弱电解质、氧化还原 |
| 判断酸碱环境 | 溶液中是否存在 `H^+`、`OH^-` | 注意隐含条件 |

## 现象
| 离子组合 | 现象 | 结论 |
|---|---|---|
| `Ag^+` 与 `Cl^-` | 生成白色沉淀 | 不能大量共存 |
| `Ba^{2+}` 与 `SO4^{2-}` | 生成白色沉淀 | 不能大量共存 |
| `H^+` 与 `CO3^{2-}` | 产生无色气泡 | 不能大量共存 |
| `Fe^{3+}` 与 `SCN^-` | 溶液显血红色 | 发生特征反应 |
| `NH4^+` 与 `OH^-` | 加热产生刺激性气味气体 | 生成 `NH3` |

## 方程式
```chemical
Ag^+ + Cl^- = AgCl↓
Ba^{2+} + SO4^{2-} = BaSO4↓
2H^+ + CO3^{2-} = CO2↑ + H2O
H^+ + OH^- = H2O
NH4^+ + OH^- = NH3↑ + H2O（加热）
Fe^{3+} + 3OH^- = Fe(OH)3↓
```

## 易错点
- `CaCO3`、`BaSO4`、`Cu(OH)2` 这类 **沉淀不能拆**。
- `HAc`、`NH3·H2O`、`H2O` 属于弱电解质或水，**离子方程式中保留化学式**。
- 题目若写“无色透明溶液”，就要先排除有色离子和会生成沉淀的组合。
- “大量共存”看的是 **混合后整体是否稳定存在**，不是只看任意两种离子之间是否反应。

## 递进例题
### 例1：哪些要拆
写出 `Na2CO3` 与盐酸反应的离子方程式。

**分析**：`Na2CO3` 和 `HCl` 都是强电解质且可溶，要拆；生成的 `CO2` 和 `H2O` 不拆。

**答案**：`2H^+ + CO3^{2-} = CO2↑ + H2O`。

### 例2：离子共存
下列离子能大量共存的是哪一组？
A. `K^+`、`NO3^-`、`Na^+`、`Cl^-`
B. `Ba^{2+}`、`K^+`、`SO4^{2-}`、`NO3^-`

**分析**：A 组不发生反应；B 组中 `Ba^{2+}` 与 `SO4^{2-}` 生成沉淀。

**答案**：A 组能大量共存。

### 例3：纠错题
判断下列离子方程式是否正确：
`Fe + 2H^+ = Fe^{3+} + H2↑`

**分析**：铁与非氧化性酸反应通常生成 `Fe^{2+}`，不是 `Fe^{3+}`。

**答案**：错误，正确应为 `Fe + 2H^+ = Fe^{2+} + H2↑`。

### 例4：隐含条件
向含有 `NH4^+`、`Na^+`、`Cl^-` 的溶液中加入足量 NaOH 并加热，有什么现象？

**分析**：`NH4^+` 与 `OH^-` 反应生成 `NH3`，加热更有利于逸出。

**答案**：有刺激性气味气体放出，可使湿润红色石蕊试纸变蓝。

## 练习
1. 写出 `Ba(OH)2` 溶液与稀硫酸反应的离子方程式。
2. 判断 `Na^+`、`K^+`、`NO3^-`、`CO3^{2-}` 在酸性溶液中能否大量共存，并说明理由。
3. 写出 `FeCl3` 溶液与 NaOH 溶液反应的离子方程式。
4. 下列离子方程式是否正确：`CO3^{2-} + 2H^+ = CO2↑ + H2O`。

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `Ba^{2+} + 2OH^- + 2H^+ + SO4^{2-} = BaSO4↓ + 2H2O` |
| 2 | 不能；酸性条件下 `H^+` 与 `CO3^{2-}` 反应生成 `CO2` |
| 3 | `Fe^{3+} + 3OH^- = Fe(OH)3↓` |
| 4 | 正确 |

# 高中化学：钠及其化合物

> 高中化学 | 元素化合物

---

## 📌 钠的性质

### 物理性质
- 银白色金属
- 质软
- 密度小于水

### 化学性质
- 与水：2Na + 2H₂O → 2NaOH + H₂↑
- 与氧气：4Na + O₂ → 2Na₂O
- 与氯气：2Na + Cl₂ → 2NaCl

---

## 📌 钠的化合物

### Na₂O（氧化钠）
- 碱性氧化物
- 与水：Na₂O + H₂O → 2NaOH

### Na₂O₂（过氧化钠）
- 过氧化物
- 与水：2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
- 与CO₂：2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂

### NaOH（氢氧化钠）
- 强碱
- 四大碱的性质

### Na₂CO₃（碳酸钠）
- 纯碱
- 与酸反应

### NaHCO₃（碳酸氢钠）
- 小苏打
- 受热分解：2NaHCO₃ → Na₂CO₃ + CO₂↑ + H₂O

---

## 📝 典型例题

### 例1：计算
13.8g Na₂CO₃·10H₂O 溶于水配成1L溶液，浓度？

> 📋 答案
> n = 13.8/286 = 0.0483 mol
> c = 0.0483/1 = 0.0483 mol/L

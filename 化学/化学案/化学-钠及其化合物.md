# 化学学案：钠及其化合物

> 高中化学 | 元素化合物线 E1

---

## 本课学什么
- 把握钠的 **活泼金属性** 以及 `Na2O`、`Na2O2`、`Na2CO3`、`NaHCO3` 的典型性质。
- 会从 **结构/价态 → 性质 → 反应 → 实验现象** 的链条理解钠元素专题。
- 会处理钠与水、过氧化钠、纯碱和小苏打的比较题。

## 原理
1. 钠位于第ⅠA族，最外层只有 1 个电子，**易失电子**，所以金属性强、化学性质活泼。
2. 钠与水反应本质是钠失电子，水中的 `H^+` 得电子生成 `H2`，同时形成碱。
3. `Na2O2` 中氧为 -1 价，兼具氧化性和特殊转化规律，因此与水、`CO2` 反应都会放出 `O2`。
4. `Na2CO3` 与 `NaHCO3` 的差别可从 **热稳定性、与酸反应、与碱反应、溶解性** 来比较。

## 条件
| 物质 | 条件 | 典型结论 |
|---|---|---|
| Na 与 `O2` | 常温 | 主要生成 `Na2O` |
| Na 与 `O2` | 加热或点燃 | 易生成 `Na2O2` |
| Na 与水 | 常温即可 | 剧烈反应生成 `NaOH` 和 `H2` |
| `NaHCO3` | 加热 | 分解生成 `Na2CO3`、`CO2`、`H2O` |
| `Na2CO3`、`NaHCO3` 与酸 | 酸足量 | 都放出 `CO2` |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| Na 投入水中 | 浮、熔、游、响，滴酚酞后溶液变红 | 反应放热，生成碱和氢气 |
| `Na2O2` 与水反应 | 有气泡放出，放热 | 生成 `O2` |
| `Na2CO3` 溶液与澄清石灰水 | 生成白色沉淀 | 生成 `CaCO3` |
| 加热 `NaHCO3` 固体 | 有气体放出，残留固体可使酚酞变红 | 分解生成 `Na2CO3` |

## 方程式
```chemical
2Na + 2H2O = 2NaOH + H2↑
4Na + O2 = 2Na2O
2Na + O2 = Na2O2（加热）
2Na2O2 + 2H2O = 4NaOH + O2↑
2Na2O2 + 2CO2 = 2Na2CO3 + O2
Na2CO3 + 2HCl = 2NaCl + CO2↑ + H2O
2NaHCO3 = Na2CO3 + CO2↑ + H2O（加热）
```

## 易错点
- 钠与水反应生成的是 **`NaOH` 和 `H2`**，不是 `O2`。
- `Na2O2` 与 `CO2` 反应既不是简单酸碱反应，也不是普通氧化物反应，要注意生成 `O2`。
- 比较 `Na2CO3` 与 `NaHCO3` 时，不要只背名称，要抓 **热稳定性和与酸碱反应差异**。
- 纯碱是 `Na2CO3`，虽然名叫“碱”，但属于盐。

## 递进例题
### 例1：钠与水
把一小块金属钠投入滴有酚酞的水中，会观察到哪些现象？

**分析**：钠密度比水小，反应放热使其熔成小球，并迅速移动；生成碱使酚酞变红。

**答案**：浮在水面、熔成小球、四处游动、伴随响声，溶液变红。

### 例2：过氧化钠性质
`Na2O2` 分别与 `H2O`、`CO2` 反应有什么共同点？

**分析**：两者都生成 `O2`，体现过氧化钠中氧的特殊价态。

**答案**：共同点是都放出 `O2`。

### 例3：纯碱与小苏打比较
区别 `Na2CO3` 和 `NaHCO3` 的一种简便方法是什么？

**分析**：可采用加热法或与相同浓度盐酸反应速度比较。

**答案**：加热时 `NaHCO3` 易分解而 `Na2CO3` 不分解；也可比较与酸反应快慢。

### 例4：综合计算
标准状况下，11.2 L `CO2` 与足量 `Na2O2` 反应，生成多少 mol `O2`？

**分析**：`n(CO2)=0.5 mol`，由 `2Na2O2 + 2CO2 = 2Na2CO3 + O2` 可知 2 mol `CO2` 生成 1 mol `O2`。

**答案**：`0.25 mol`。

## 练习
1. 写出钠在空气中点燃的主要反应方程式。
2. 判断：`Na2O2` 属于碱性氧化物。
3. `NaHCO3` 受热分解后剩余固体是什么？
4. `Na2CO3` 溶液中滴加稀盐酸，有什么明显现象？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `2Na + O2 = Na2O2（加热）` |
| 2 | 错；它是过氧化物 |
| 3 | `Na2CO3` |
| 4 | 有气泡放出，生成 `CO2` |

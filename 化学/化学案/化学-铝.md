# 高中化学：铝

> 高中化学 | 元素化合物

---

## 📌 铝的性质

### 物理性质
银白色金属，密度小

### 化学性质
- 与氧气：4Al + 3O₂ → 2Al₂O₃
- 与酸：2Al + 6H⁺ → 2Al³⁺ + 3H₂↑
- 与碱：2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑

---

## 📌 铝的化合物

### Al₂O₃（氧化铝）
两性氧化物

### Al(OH)₃
两性氢氧化物
- 与酸：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
- 与碱：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O

### 明矾
KAl(SO₄)₂·12H₂O
- 净水原理：Al³⁺ + 3H₂O → Al(OH)₃ + 3H⁺

---

## 📝 典型例题

### 例1：铝热反应
铝粉与Fe₂O₃反应方程式。

> 📋 答案
> 2Al + Fe₂O₃ → Al₂O₃ + 2Fe

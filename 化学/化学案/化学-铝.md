# 化学学案：铝及其化合物

> 高中化学 | 元素化合物线 E2

---

## 本课学什么
- 理解铝的 **两性、钝化、铝热反应** 这三条主线。
- 会判断 `Al`、`Al2O3`、`Al(OH)3` 在酸、碱条件下的反应。
- 会把铝盐净水原理和实验现象联系起来。

## 原理
1. 铝位于第ⅢA族，较易失去 3 个电子，常见化合价为 +3。
2. 铝表面易形成致密 `Al2O3` 薄膜，所以常温下具有一定 **抗腐蚀性**，这叫钝化。
3. `Al2O3` 和 `Al(OH)3` 都具有 **两性**：既能与酸反应，也能与强碱反应。
4. 铝热反应本质是铝作还原剂，把金属氧化物中的金属还原出来，同时放出大量热。

## 条件
| 物质 | 条件 | 典型结论 |
|---|---|---|
| Al 与稀酸 | 常温 | 生成铝盐和 `H2` |
| Al 与强碱溶液 | 常温 | 生成偏铝酸盐和 `H2` |
| `Al2O3`、`Al(OH)3` | 酸性或碱性条件 | 都表现两性 |
| 铝热反应 | 高温点燃 | 剧烈放热，可冶炼金属 |
| 明矾净水 | 水中水解 | 生成胶状 `Al(OH)3` 吸附杂质 |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| 铝片投入稀盐酸 | 有气泡放出 | 生成 `H2` |
| 铝片投入 NaOH 溶液 | 也有气泡放出 | 铝与碱反应生成 `H2` |
| 向铝盐溶液中滴加 NaOH | 先有白色沉淀，后继续加碱沉淀溶解 | `Al(OH)3` 具有两性 |
| 铝热反应 | 发出耀眼光和大量热，生成熔融金属 | 反应放热剧烈 |

## 方程式
```chemical
2Al + 6HCl = 2AlCl3 + 3H2↑
2Al + 2NaOH + 2H2O = 2NaAlO2 + 3H2↑
Al2O3 + 6HCl = 2AlCl3 + 3H2O
Al2O3 + 2NaOH = 2NaAlO2 + H2O
Al(OH)3 + 3HCl = AlCl3 + 3H2O
Al(OH)3 + NaOH = NaAlO2 + 2H2O
2Al + Fe2O3 = Al2O3 + 2Fe（高温）
```

## 易错点
- 铝能与酸反应，也能与强碱反应，这是 **铝及其氧化物、氢氧化物两性特征** 的常考点。
- 铝与浓硝酸、浓硫酸常温下常表现为 **钝化**，不是“永远不反应”。
- `Al(OH)3` 是白色胶状沉淀，不是两性“气体”或“可溶盐”。
- 偏铝酸盐在高中常写 `NaAlO2`，做题时注意与离子形式 `AlO2^-` 的转换。

## 递进例题
### 例1：两性判断
为什么铝既能与盐酸反应，又能与 NaOH 溶液反应？

**分析**：铝及其化合物具有两性相关特征，铝本身能在碱性条件下转化为偏铝酸盐。

**答案**：因为铝既能在酸性环境中失电子生成 `Al^{3+}`，又能在强碱性条件下生成偏铝酸盐并放出 `H2`。

### 例2：沉淀变化
向 `AlCl3` 溶液中逐滴加入 NaOH 溶液直到过量，会看到什么现象？

**分析**：先生成 `Al(OH)3` 沉淀，继续加碱沉淀溶解。

**答案**：先出现白色胶状沉淀，后沉淀逐渐消失。

### 例3：铝热反应
写出铝粉与 `Fe2O3` 的反应方程式，并指出铝在其中的作用。

**分析**：铝失电子，还原氧化铁。

**答案**：`2Al + Fe2O3 = Al2O3 + 2Fe（高温）`；铝作还原剂。

### 例4：净水原理
明矾净水的主要原理是什么？

**分析**：`Al^{3+}` 水解形成胶体 `Al(OH)3`，可吸附悬浮杂质。

**答案**：利用 `Al(OH)3` 胶体的吸附作用净水，不是消毒杀菌。

## 练习
1. 判断：`Al2O3` 只属于碱性氧化物。
2. 写出 `Al(OH)3` 与 NaOH 反应的方程式。
3. 常温下铝遇浓硝酸有什么典型特点？
4. 明矾净水属于杀菌消毒还是胶体吸附？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | 错；`Al2O3` 是两性氧化物 |
| 2 | `Al(OH)3 + NaOH = NaAlO2 + 2H2O` |
| 3 | 易钝化，表面形成致密氧化膜 |
| 4 | 胶体吸附 |

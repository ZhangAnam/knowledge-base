# 化学学案：浓度计算

> 高中化学 | 计算模型线 C2

---

## 本课学什么

- 理解物质的量浓度的定义和单位。
- 会进行 `c、n、V` 的基本换算，能处理稀释、混合等基础题。
- 能分清“溶液体积”“溶剂体积”和“加水体积”等易混量。

## 原理

1. 物质的量浓度表示单位体积溶液中所含溶质的物质的量。
2. 定义式：

```text
c = n/V
```

其中 `V` 用 `L` 作单位。

3. 若已知质量，可先通过 `n = m/M` 求物质的量，再代入浓度公式。
4. 稀释前后溶质的物质的量通常不变，因此常有：

```text
c1V1 = c2V2
```

## 条件

| 任务 | 使用条件 | 关键提醒 |
|---|---|---|
| `c = n/V` | 已知溶质物质的量与溶液体积 | `V` 必须是溶液总体积 |
| `c1V1 = c2V2` | 稀释前后溶质未损失 | 适用于只加水不反应的情形 |
| 质量转浓度 | 先求 `n` | 注意相对分子质量和单位换算 |

## 现象

浓度变化会影响：

- 反应快慢；
- 颜色深浅；
- 导电能力强弱；
- 滴定和配制实验中的结果。

## 方程式 / 关系式

```text
c = n/V
n = cV
c1V1 = c2V2
n = m/M
```

## 易错点

- **浓度公式中的体积是溶液体积，不是溶剂体积**。
- 稀释题中前后不变的是 **溶质的物质的量**，不是浓度也不是体积。
- `mL` 与 `L` 换算容易出错。
- 配制一定浓度溶液时，定容后总体积才等于目标体积。

## 递进例题

### 例1：直接计算浓度

`0.5 mol` NaCl 配成 `0.25 L` 溶液，浓度是多少？

**解答**：

```text
c = 0.5/0.25 = 2 mol/L
```

### 例2：质量求浓度

将 `5.85 g` NaCl 配成 `500 mL` 溶液，求浓度。

**解答**：

```text
M(NaCl)=58.5 g/mol
n=5.85/58.5=0.1 mol
V=0.500 L
c=0.1/0.5=0.2 mol/L
```

### 例3：稀释计算

把 `100 mL 1 mol/L` 的盐酸稀释到 `500 mL`，新浓度是多少？

**解答**：

```text
c1V1 = c2V2
1×0.1 = c2×0.5
c2 = 0.2 mol/L
```

### 例4：逆向求体积

需要配制 `0.2 mol/L` NaOH 溶液 `250 mL`，需 NaOH 多少 mol？

**解答**：

```text
n = cV = 0.2×0.25 = 0.05 mol
```

## 练习

1. `0.3 mol` 溶质配成 `0.6 L` 溶液，浓度是多少？
2. `1 mol/L` 溶液 `200 mL` 中含多少 mol 溶质？
3. `2 mol/L` 溶液 `50 mL` 稀释到 `200 mL`，新浓度是多少？
4. 配制溶液时定容前能否直接把体积当最终体积？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `0.5 mol/L` |
| 2 | `0.2 mol` |
| 3 | `0.5 mol/L` |
| 4 | 不能 |

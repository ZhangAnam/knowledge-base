# 化学学案：化学平衡

> 高中化学 | 反应规律线 R4

---

## 本课学什么
- 理解化学平衡的建立、本质和特征。
- 会用勒夏特列原理判断浓度、压强、温度变化对平衡移动的影响。
- 会区分 **平衡移动、速率变化、平衡常数变化** 这三个常考点。

## 原理
1. 对于可逆反应，在一定条件下，当 **正反应速率等于逆反应速率** 时，体系达到动态平衡。
2. 平衡时各组分浓度保持不变，但反应并没有停止，正逆反应仍在进行。
3. 改变浓度、压强、温度等条件后，体系会向 **削弱这种改变的方向** 移动，这就是勒夏特列原理。
4. **平衡常数只受温度影响**；浓度、压强的改变只会引起平衡移动，不会直接改变 K。

## 条件
| 改变条件 | 对平衡的影响 | 适用提醒 |
|---|---|---|
| 增大某反应物浓度 | 平衡向正反应方向移动 | 固体、纯液体一般不列入平衡表达式 |
| 减小生成物浓度 | 平衡向正反应方向移动 | 常见于移走产物 |
| 增大压强 | 向气体总物质的量减小方向移动 | 只对有气体参与的反应适用 |
| 升高温度 | 吸热方向有利 | 先判断正反应吸放热 |
| 使用催化剂 | 不改变平衡移动方向 | 仅同时改变正逆反应速率 |

## 现象
| 体系 | 条件变化 | 现象/结论 |
|---|---|---|
| `2NO2 ⇌ N2O4` | 降温 | 颜色变浅，平衡向生成 `N2O4` 方向移动 |
| 合成氨 `N2 + 3H2 ⇌ 2NH3` | 加压 | 氨的平衡含量增大 |
| `Fe^{3+} + SCN^- ⇌ Fe(SCN)^{2+}` | 加 `Fe^{3+}` | 血红色加深 |
| 上述体系 | 加入 `F^-` 络合 `Fe^{3+}` | 血红色变浅，平衡逆向移动 |

## 方程式
```chemical
N2 + 3H2 ⇌ 2NH3
2NO2 ⇌ N2O4
Fe^{3+} + SCN^- ⇌ Fe(SCN)^{2+}
K = [生成物浓度幂之积] / [反应物浓度幂之积]
```

## 易错点
- **平衡时浓度不变** 不等于浓度相等，也不等于反应停止。
- 增大压强对 **无气体体系** 或反应前后气体总量不变的体系，平衡移动影响可能不明显。
- 催化剂只改速率，不改变平衡组成。
- 只有温度改变时，才需要考虑 **平衡常数 K 是否变化**。

## 递进例题
### 例1：平衡建立
为什么可逆反应达到平衡时，反应仍然没有停止？

**分析**：平衡是动态平衡，正逆反应同时进行且速率相等。

**答案**：因为平衡时正逆反应都在继续，只是速率相等，所以各组分浓度保持不变。

### 例2：浓度影响
在平衡体系 `Fe^{3+} + SCN^- ⇌ Fe(SCN)^{2+}` 中，加入少量 `KSCN` 溶液，颜色如何变化？

**分析**：`SCN^-` 浓度增大，平衡向右移动。

**答案**：血红色加深。

### 例3：压强影响
反应 `N2 + 3H2 ⇌ 2NH3` 在恒温下增大压强，平衡向哪边移动？为什么？

**分析**：气体总物质的量由 4 变 2，增大压强有利于气体总数减小的一侧。

**答案**：向右移动，因为右侧气体总物质的量更少。

### 例4：温度与 K
反应 `2NO2 ⇌ N2O4` 的正反应放热。升高温度时，平衡如何移动？K 如何变化？

**分析**：升温有利于吸热方向，即逆向；放热正反应对应的 K 减小。

**答案**：平衡逆向移动，K 减小。

## 练习
1. 判断：平衡时正反应速率和逆反应速率都等于 0。
2. 反应 `2SO2 + O2 ⇌ 2SO3` 中，若移走 `SO3`，平衡怎样移动？
3. 反应 `H2 + I2 ⇌ 2HI` 中，若反应前后气体总物质的量相等，改变压强对平衡移动有何影响？
4. 对于吸热反应，升高温度后平衡常数如何变化？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | 错；平衡时正逆反应速率相等且通常不为 0 |
| 2 | 向正反应方向移动 |
| 3 | 一般不引起平衡移动 |
| 4 | 增大 |

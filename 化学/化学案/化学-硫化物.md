# 高中化学：硫化物

> 高中化学 | 元素化合物

---

## 📌 硫的性质

### 物理性质
黄色固体，不溶于水

### 化学性质
- 与氧气：S + O₂ → SO₂
- 与金属：Fe + S → FeS

---

## 📌 硫的化合物

### SO₂（二氧化硫）
- 刺激性气味，有毒
- 还原性：2SO₂ + O₂ → 2SO₃
- 漂白性：与有色物质结合

### SO₃（三氧化硫）
- 无色固体
- 与水：SO₃ + H₂O → H₂SO₄

### H₂S（硫化氢）
- 臭鸡蛋气味
- 还原性：2H₂S + SO₂ → 3S + 2H₂O

---

## 📌 硫酸

### 性质
- 吸水性：作干燥剂
- 脱水性：蔗糖碳化
- 强氧化性：与Cu反应

### 浓硫酸稀释
酸入水，沿杯壁，慢慢搅

---

如何鉴别SO₂和CO₂？

> 📋 答案

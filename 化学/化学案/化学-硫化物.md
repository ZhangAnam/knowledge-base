# 化学学案：硫及其化合物

> 高中化学 | 元素化合物线 E5

---

## 本课学什么
- 把握硫专题中的 **`SO2` 性质、浓硫酸特性、硫化氢还原性**。
- 会区分 `SO2` 的 **酸性氧化物性质、还原性、漂白性**。
- 会结合实验现象判断硫及其常见化合物的反应类型。

## 原理
1. 硫元素化合价变化范围广，常见有 `-2`、`0`、`+4`、`+6` 价，因此易出现氧化还原转化。
2. `SO2` 中硫为 +4 价，既可被氧化成 +6 价，也可表现一定还原性，同时具有漂白性。
3. 浓硫酸除了强酸性外，还具有 **吸水性、脱水性、强氧化性**。
4. `H2S` 中硫为 -2 价，还原性强，易被氧化生成硫单质或更高价硫化合物。

## 条件
| 物质 | 条件 | 典型结论 |
|---|---|---|
| S 与 `O2` | 点燃 | 生成 `SO2` |
| `SO2` 与品红 | 常温 | 品红褪色，加热可恢复 |
| `SO2` 与酸性高锰酸钾 | 常温 | 紫色褪去，体现还原性 |
| Cu 与浓硫酸 | 加热 | 生成 `SO2` |
| `H2S` 与 `SO2` | 常温 | 生成硫单质和水 |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| 硫在空气中燃烧 | 发出淡蓝色火焰，有刺激性气味气体 | 生成 `SO2` |
| `SO2` 通入品红溶液 | 品红褪色 | 体现漂白性 |
| `SO2` 通入酸性高锰酸钾 | 紫色褪去 | 体现还原性 |
| Cu 与浓硫酸加热 | 产生刺激性气味气体，溶液变蓝 | 生成 `SO2` 和 `CuSO4` |

## 方程式
```chemical
S + O2 = SO2（点燃）
2SO2 + O2 ⇌ 2SO3（催化剂、加热）
SO3 + H2O = H2SO4
Cu + 2H2SO4(浓) = CuSO4 + SO2↑ + 2H2O（加热）
2H2S + SO2 = 3S↓ + 2H2O
```

## 易错点
- `SO2` 能使品红褪色，但这是 **可逆漂白**；与 `HClO`、`O3` 的氧化漂白不同。
- `SO2` 既是酸性氧化物，又有还原性，做题时要看反应对象。
- 稀硫酸主要体现酸性，**浓硫酸** 才常考吸水性、脱水性和强氧化性。
- 稀释浓硫酸时必须 **酸入水**，不能把水倒入浓硫酸。

## 递进例题
### 例1：硫燃烧
硫在空气中燃烧的现象和产物是什么？

**分析**：硫与氧气反应生成 `SO2`。

**答案**：发出淡蓝色火焰，生成有刺激性气味的 `SO2`。

### 例2：`SO2` 性质判断
如何用实验区分 `SO2` 的漂白性和还原性？

**分析**：用品红检验漂白性，用酸性高锰酸钾检验还原性。

**答案**：通入品红溶液可褪色，通入酸性高锰酸钾溶液可使其紫色褪去。

### 例3：浓硫酸与铜
铜为什么不能与稀硫酸反应，却能与浓硫酸在加热条件下反应？

**分析**：稀硫酸无足够氧化性，浓硫酸在加热下表现强氧化性。

**答案**：因为浓硫酸兼有强氧化性，可把铜氧化；稀硫酸主要只显酸性。

### 例4：综合转化
`H2S` 与 `SO2` 反应中，谁是氧化剂，谁是还原剂？

**分析**：`H2S` 中硫由 -2 价升到 0 价，作还原剂；`SO2` 中硫由 +4 价降到 0 价，作氧化剂。

**答案**：`SO2` 是氧化剂，`H2S` 是还原剂。

## 练习
1. 写出 Cu 与浓硫酸加热反应的方程式。
2. 判断：`SO2` 使酸性高锰酸钾褪色，体现的是漂白性。
3. 浓硫酸稀释时应怎样操作？
4. `2H2S + SO2 = 3S↓ + 2H2O` 中生成的黄色固体是什么？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `Cu + 2H2SO4(浓) = CuSO4 + SO2↑ + 2H2O（加热）` |
| 2 | 错；体现的是还原性 |
| 3 | 将浓硫酸沿器壁慢慢注入水中并不断搅拌 |
| 4 | 硫单质 |

# 化学学案：乙醇与乙酸

> 高中化学 | 有机化学线 O2

---

## 本课学什么

- 认识乙醇、乙酸的结构简式和主要官能团。
- 理解乙醇的氧化、乙酸的酸性，以及酯化反应的基础。
- 会从官能团角度理解性质差异。

## 原理

1. 乙醇含有羟基，兼具可燃、可被氧化等性质。
2. 乙酸含有羧基，表现出酸性，并可与乙醇发生酯化反应。
3. 有机物性质变化的核心抓手是 **官能团**。

## 条件

| 物质 | 官能团 | 典型性质 |
|---|---|---|
| 乙醇 | `-OH` | 可燃、可氧化、可与活泼金属反应 |
| 乙酸 | `-COOH` | 具有酸性、可与醇酯化 |

## 现象

- 乙醇燃烧产生淡蓝色火焰；
- 乙酸有刺激性气味；
- 乙醇催化氧化生成乙醛时可能有颜色变化；
- 酯化反应常闻到香味。

## 方程式

```chemical
C2H5OH + 3O2 = 2CO2 + 3H2O
2C2H5OH + O2 → 2CH3CHO + 2H2O
CH3COOH + C2H5OH ⇌ CH3COOC2H5 + H2O
CH3COOH + NaOH = CH3COONa + H2O
```

## 易错点

- **乙醇不是酸**，虽然含羟基，但与无机碱的反应性和酸不同。
- 乙酸酸性比碳酸弱，但能使指示剂变色并与碱反应。
- 酯化反应是可逆反应，常需浓硫酸作催化剂和吸水剂。
- 结构简式、分子式、官能团表达要规范。

## 递进例题

### 例1：官能团判断

乙醇和乙酸分别含什么官能团？

**答案**：乙醇含羟基，乙酸含羧基。

### 例2：酸性判断

乙酸能否与 NaOH 反应？

**答案**：能，因为乙酸具有酸性。

### 例3：酯化反应

乙酸和乙醇反应生成什么？

**答案**：乙酸乙酯和水。

### 例4：性质比较

为什么乙酸和乙醇性质明显不同？

**答案**：因为官能团不同，导致结构与性质不同。

## 练习

1. 乙酸的官能团是什么？
2. 乙醇催化氧化可生成什么？
3. 酯化反应常需要哪种酸？
4. 乙酸能否使紫色石蕊变红？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 羧基 |
| 2 | 乙醛 |
| 3 | 浓硫酸 |
| 4 | 能 |

# 化学学案：氧化还原反应

> 高中化学 | 反应规律线 R1

---

## 本课学什么
- 理清 **化合价变化、电子转移、氧化剂、还原剂** 之间的对应关系。
- 会判断一个反应是不是氧化还原反应，并能找出谁被氧化、谁被还原。
- 会用 **化合价法和电子守恒** 处理基础配平与推断题。

## 原理
1. **有电子转移的反应就是氧化还原反应**，电子转移可以是“真正得失电子”，也可以表现为“共用电子对偏移”。
2. **失电子或化合价升高** 的物质被氧化，是还原剂；**得电子或化合价降低** 的物质被还原，是氧化剂。
3. 氧化还原反应中一定同时发生氧化和还原，且 **得失电子总数相等**。
4. 判断顺序通常是：**先看元素化合价是否变化，再判电子转移方向，最后定氧化剂和还原剂**。

## 条件
| 判断任务 | 关键条件 | 操作抓手 |
|---|---|---|
| 判断是否为氧化还原反应 | 反应前后至少有一种元素化合价变化 | 先标变价元素 |
| 判断氧化剂、还原剂 | 有电子得失或偏移 | 升价找还原剂，降价找氧化剂 |
| 书写配平思路 | 先守恒电子，再守恒原子 | 常结合酸性、碱性条件补 `H^+`、`OH^-`、`H_2O` |
| 判断强弱 | 常见强氧化剂、强还原剂参与 | 结合反应条件和产物价态判断 |

## 现象
| 典型反应 | 现象 | 说明 |
|---|---|---|
| 铁与硫酸铜反应 | 铁表面析出红色铜，溶液由蓝色逐渐变浅绿 | `Fe` 还原 `Cu^{2+}` |
| 铜与浓硝酸反应 | 产生红棕色气体，溶液变蓝 | `Cu` 被氧化，生成 `NO_2` |
| 氢气还原氧化铜 | 黑色粉末变红，试管口有水珠 | `H_2` 作还原剂 |
| 氯气与碘化钾反应 | 溶液变棕黄，可使淀粉变蓝 | `Cl_2` 氧化 `I^-` |

## 方程式
```chemical
Fe + CuSO4 = FeSO4 + Cu
Cu + 4HNO3(浓) = Cu(NO3)2 + 2NO2↑ + 2H2O
H2 + CuO = Cu + H2O（加热）
Cl2 + 2KI = 2KCl + I2
2FeCl2 + Cl2 = 2FeCl3
```

## 易错点
- **有元素化合价变化** 才是氧化还原反应，酸碱中和通常不是。
- “氧化产物”“还原产物”说的是 **生成物**，不是反应物。
- 同一种物质可能既是氧化剂又是还原剂，如 `Cl2` 与碱反应发生歧化。
- 配平时不能只看系数凑整数，要先保证 **电子守恒**，再看原子守恒、电荷守恒。

## 递进例题
### 例1：基础判断
判断下列反应是否属于氧化还原反应：
1. `NaOH + HCl = NaCl + H2O`
2. `Fe + CuSO4 = FeSO4 + Cu`

**分析**：第 1 个反应中各元素化合价不变；第 2 个反应中 `Fe` 由 0 价变成 +2 价，`Cu` 由 +2 价变成 0 价。

**答案**：1 不是；2 是。

### 例2：判断氧化剂和还原剂
反应 `2FeCl2 + Cl2 = 2FeCl3` 中，指出氧化剂和还原剂。

**分析**：`Fe^{2+}` 变为 `Fe^{3+}`，化合价升高，被氧化；`Cl2` 中氯由 0 价变为 -1 价，被还原。

**答案**：氧化剂是 `Cl2`，还原剂是 `FeCl2`。

### 例3：电子守恒
在反应 `Cu + 4HNO3(浓) = Cu(NO3)2 + 2NO2↑ + 2H2O` 中，1 mol Cu 失去多少 mol 电子？生成多少 mol `NO2`？

**分析**：铜由 0 价到 +2 价，1 mol Cu 失 2 mol 电子；方程式显示 1 mol Cu 对应生成 2 mol `NO2`。

**答案**：失去 2 mol 电子，生成 2 mol `NO2`。

### 例4：综合判断
已知 `Cl2` 能与 `FeBr2` 溶液反应。若通入少量 `Cl2`，优先被氧化的是 `Fe^{2+}` 还是 `Br^-`？

**分析**：比较常见还原性强弱，`Fe^{2+}` 更易被氧化为 `Fe^{3+}`，少量 `Cl2` 先氧化 `Fe^{2+}`。

**答案**：优先氧化 `Fe^{2+}`。

## 练习
1. 判断 `Zn + 2HCl = ZnCl2 + H2↑` 中被氧化的物质和被还原的微粒。
2. 反应 `2Na2O2 + 2H2O = 4NaOH + O2↑` 中，氧元素是否都只发生一种变化？
3. 写出 `Fe2O3 + 3CO = 2Fe + 3CO2（高温）` 中氧化剂与还原剂。
4. 已知 `MnO2 + 4HCl(浓) = MnCl2 + Cl2↑ + 2H2O（加热）`，每生成 1 mol `Cl2`，转移多少 mol 电子？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `Zn` 被氧化，`H^+` 被还原 |
| 2 | 不是；过氧化钠中氧一部分由 -1 价升到 0 价，一部分由 -1 价降到 -2 价 |
| 3 | 氧化剂是 `Fe2O3`，还原剂是 `CO` |
| 4 | 2 mol |

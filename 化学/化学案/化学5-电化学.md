# 化学学案 5：电化学综合

> 高中化学 | 反应规律线 R5 / 计算模型线 C5

---

## 本课学什么
- 在电化学基础上，进一步掌握 **判极、写电极反应、分析离子移动、处理简单计算**。
- 会区分原电池题、电解池题、电镀题和金属腐蚀题。
- 会把“电子守恒”用于电极反应和定量关系。

## 原理
1. 原电池依靠自发氧化还原反应供电，核心是 **负极氧化、正极还原**。
2. 电解池依靠外加电源驱动反应，核心是 **阳极氧化、阴极还原**。
3. 电化学计算本质仍是 **电子守恒**：转移电子数与析出、溶解物质的量成固定比例。
4. 金属防护的常见思路有 **牺牲阳极保护法** 和 **外加电流保护法**。

## 条件
| 题型 | 条件抓手 | 核心判断 |
|---|---|---|
| 原电池 | 是否自发反应、是否闭合回路 | 活泼金属多作负极 |
| 电解池 | 是否接外加电源 | 电源正接阳极，负接阴极 |
| 电镀 | 镀层金属作阳极，被镀物作阴极 | 阴极析出镀层金属 |
| 定量计算 | 已知电量、时间、物质的量 | 用电子守恒联立 |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| Zn-Cu 原电池 | 锌片溶解、铜片析铜 | Zn 为负极 |
| 电解饱和食盐水 | 阳极放出黄绿色气体，阴极放出无色气体 | 生成 `Cl2` 和 `H2` |
| 铜电极电解 `CuSO4` | 阳极铜溶解，阴极析出铜 | 电解精炼与电镀模型 |
| 铁连接锌块防护 | 锌先被腐蚀 | 牺牲阳极保护铁 |

## 方程式
```chemical
Zn - 2e^- = Zn^{2+}
Cu^{2+} + 2e^- = Cu
2Cl^- - 2e^- = Cl2↑
2H2O + 2e^- = H2↑ + 2OH^-
Cu - 2e^- = Cu^{2+}
Zn^{2+} + 2e^- = Zn
```

## 易错点
- 原电池中的“正极”与电解池中的“阳极”不是同一个概念，不能混写。
- 电镀题中被镀物一定作 **阴极**，不是阳极。
- 电解池中若是活性电极，阳极材料本身可能失电子溶解。
- 计算时不要漏掉电子数与物质的量的倍数关系，如 `Cu^{2+} + 2e^- = Cu`。

## 递进例题
### 例1：判装置类型
某装置中 Zn 与 Cu 用导线连接并插入各自盐溶液，电流表发生偏转。该装置属于什么？

**分析**：无需外加电源且能自发产生电流。

**答案**：原电池。

### 例2：写电极反应
电解饱和食盐水时，写出阳极和阴极的电极反应式。

**分析**：阳极 `Cl^-` 放电，阴极水得电子。

**答案**：阳极：`2Cl^- - 2e^- = Cl2↑`；阴极：`2H2O + 2e^- = H2↑ + 2OH^-`。

### 例3：电镀判断
用锌给铁制品镀层时，铁制品应连接电源哪一极？

**分析**：铁制品表面要析出锌，需在阴极得电子。

**答案**：连接电源负极，作阴极。

### 例4：电子守恒计算
若阴极析出 `0.10 mol` Cu，需要转移多少 mol 电子？

**分析**：每 1 mol Cu 需要 2 mol 电子。

**答案**：`0.20 mol` 电子。

## 练习
1. 判断：原电池中阳离子向负极移动。
2. 电解 `CuCl2` 溶液时，阴极析出什么物质？
3. 用锌块保护铁管道，哪种金属先被腐蚀？
4. 阴极析出 `0.25 mol` Zn 时，转移电子多少 mol？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | 错；原电池中阳离子向正极移动 |
| 2 | 铜 |
| 3 | 锌先被腐蚀 |
| 4 | `0.50 mol` |

# 化学学案：电化学基础

> 高中化学 | 反应规律线 R5 / 计算模型线 C5

---

## 本课学什么
- 区分 **原电池** 与 **电解池** 的工作原理。
- 会判断正负极、阴阳极，能写常见电极反应式。
- 会结合电子流向和离子移动分析简单电化学现象。

## 原理
1. **原电池**：把自发氧化还原反应中的化学能转化为电能。较活泼金属通常作负极，发生氧化。
2. **电解池**：在外加电源作用下发生非自发氧化还原反应，把电能转化为化学能。
3. 两类装置中都满足：**氧化发生在失电子的一极，还原发生在得电子的一极**。
4. 电子沿外电路从原电池负极流向正极；电解池中电子由电源负极送到阴极。

## 条件
| 装置 | 条件 | 关键判断 |
|---|---|---|
| 原电池 | 两极材料、电解质溶液、形成闭合回路、自发反应 | 负极氧化，正极还原 |
| 电解池 | 外加直流电源、电解质、闭合回路 | 阳极氧化，阴极还原 |
| 电极方程式书写 | 先判反应物来自哪里 | 注意溶液环境和放电顺序 |
| 离子移动 | 阴离子向阳极，阳离子向阴极 | 用于判断电荷守恒 |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| Zn-Cu 原电池 | 锌片逐渐溶解，铜片质量增加，电流表偏转 | Zn 为负极，Cu 为正极 |
| 电解饱和食盐水 | 阳极有黄绿色气体，阴极有无色气体，溶液碱性增强 | 生成 `Cl2`、`H2` 和 `NaOH` |
| 铜电极电解硫酸铜 | 阳极铜溶解，阴极析出铜 | 电解精炼和电镀基础 |
| 铁制品镀锌 | 铁制品表面形成锌层 | 阴极发生 `Zn^{2+}` 得电子 |

## 方程式
```chemical
Zn - 2e^- = Zn^{2+}
Cu^{2+} + 2e^- = Cu
2Cl^- - 2e^- = Cl2↑
2H2O + 2e^- = H2↑ + 2OH^-
Cu - 2e^- = Cu^{2+}
Zn + CuSO4 = ZnSO4 + Cu
```

## 易错点
- 原电池中 **正极不是“带正电的金属”**，而是发生还原反应的一极。
- 电解池中 **阳极、阴极名称由电源正负极决定**，不是由溶液酸碱性决定。
- 写电极反应时要区分是 **活性电极参与反应** 还是 **惰性电极只导电**。
- 电极反应式既要守恒原子，也要守恒电荷。

## 递进例题
### 例1：原电池判极
在 Zn-Cu 原电池中，Zn 插入 ZnSO4 溶液，Cu 插入 CuSO4 溶液。哪一极是负极？

**分析**：锌更活泼，更易失电子。

**答案**：Zn 极是负极，发生氧化反应。

### 例2：写电极反应
写出 Zn-Cu 原电池两极的电极反应式。

**分析**：负极失电子，正极得到电子。

**答案**：负极：`Zn - 2e^- = Zn^{2+}`；正极：`Cu^{2+} + 2e^- = Cu`。

### 例3：电解池判极
电解饱和食盐水时，连接电源正极的一端产生何种气体？

**分析**：连接电源正极的是阳极，`Cl^-` 在阳极失电子。

**答案**：产生氯气。

### 例4：综合应用
铁制品镀锌时，铁制品应作阴极还是阳极？

**分析**：要让锌离子在铁制品表面得到电子并析出金属锌。

**答案**：铁制品作阴极。

## 练习
1. 判断：原电池中电子通过溶液由负极流向正极。
2. 电解饱和食盐水时，阴极生成什么物质？
3. 铜电极电解 `CuSO4` 溶液时，阳极和阴极质量如何变化？
4. 写出铁制品镀锌时阴极的电极反应式。

## 答案
| 题号 | 答案 |
|---|---|
| 1 | 错；电子只沿外电路流动 |
| 2 | `H2`，同时溶液中生成 `OH^-` |
| 3 | 阳极减小，阴极增大 |
| 4 | `Zn^{2+} + 2e^- = Zn` |

# 化学学案 1：物质的量

> 高中化学 | 物质的量

---

## 📌 核心公式

### 物质的量 n
n = N/NA = m/M = V/Vm

### 气体摩尔体积 Vm
- 标准状况：22.4L/mol
- 标况：0℃，101kPa

### 物质的量浓度
c = n/V

---

## 📝 典型例题

### 例题 1
求 11.2L CO₂ 标准状况物质的量

**解法**：n = V/Vm = 11.2/22.4 = 0.5 mol

**答案**：0.5 mol

### 例题 2
9g H₂O 物质的量

**解法**：M = 18g/mol，n = 9/18 = 0.5 mol

**答案**：0.5 mol

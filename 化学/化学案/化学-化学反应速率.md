# 化学学案：化学反应速率

> 高中化学 | 反应规律线 R3

---

## 本课学什么
- 理解化学反应速率的含义，会用 `v = Δc/Δt` 进行基础计算。
- 掌握温度、浓度、压强、催化剂、固体表面积对反应速率的影响。
- 会分析速率图像和“控制变量”类实验题。

## 原理
1. 化学反应速率表示 **单位时间内反应物浓度减少或生成物浓度增加的快慢**。
2. 微观上，速率取决于 **单位时间内有效碰撞次数**；有效碰撞越多，反应越快。
3. 升高温度、增大反应物浓度、增大气体压强、增大固体表面积、使用合适催化剂，通常都能提高有效碰撞频率或活化分子比例。
4. 同一反应中，不同物质表示的速率数值可能不同，但它们之间满足 **化学计量数之比**。

## 条件
| 影响因素 | 条件变化 | 对速率的常见影响 |
|---|---|---|
| 温度 | 升高温度 | 速率增大 |
| 浓度 | 增大反应物浓度 | 速率增大 |
| 压强 | 只对有气体参加的反应明显 | 压强增大，速率一般增大 |
| 催化剂 | 合适催化剂参与 | 改变反应路径，通常加快反应 |
| 固体表面积 | 固体磨细或粉末化 | 接触面积变大，速率增大 |

## 现象
| 情境 | 现象 | 反映的速率变化 |
|---|---|---|
| 镁条与盐酸反应，升高温度 | 产生气泡更快 | 反应速率增大 |
| 过氧化氢中加入 `MnO2` | 迅速放出大量气泡 | 催化作用使分解加快 |
| 大理石粉末与稀盐酸反应 | 比块状大理石放气更快 | 表面积增大 |
| 加压合成氨体系 | 单位时间内碰撞增多 | 气相反应速率增大 |

## 方程式
```chemical
v = Δc / Δt
2H2O2 = 2H2O + O2↑（MnO2作催化剂）
Zn + 2HCl = ZnCl2 + H2↑
CaCO3 + 2HCl = CaCl2 + CO2↑ + H2O
N2 + 3H2 ⇌ 2NH3
```

## 易错点
- 速率一定写成 **正值**，不写负号。
- 用不同物质表示速率时，要先按 **化学计量数换算**，不能直接比较数字大小。
- 增大压强只对 **有气体参与且体积变化明显的反应体系** 讨论更有意义。
- 催化剂能改变速率，但 **不能改变平衡转化率和反应总量的理论极限**。

## 递进例题
### 例1：基础计算
某反应中，2 s 内 `H2` 浓度由 `1.0 mol/L` 降到 `0.6 mol/L`，求以 `H2` 表示的平均反应速率。

**分析**：`Δc = 1.0 - 0.6 = 0.4 mol/L`，`Δt = 2 s`。

**答案**：`v(H2) = 0.4 / 2 = 0.20 mol/(L·s)`。

### 例2：计量换算
反应 `N2 + 3H2 ⇌ 2NH3` 中，若 `v(H2) = 0.30 mol/(L·s)`，则 `v(NH3)` 是多少？

**分析**：按化学计量数比 `H2 : NH3 = 3 : 2` 换算。

**答案**：`v(NH3) = 0.20 mol/(L·s)`。

### 例3：影响因素判断
等质量块状 CaCO3 和粉末 CaCO3 分别与足量稀盐酸反应，哪一个反应更快？为什么？

**分析**：粉末表面积更大，与酸接触更充分。

**答案**：粉末 CaCO3 更快，因为表面积更大。

### 例4：图像判断
某可逆反应开始后，正反应速率逐渐减小，逆反应速率逐渐增大，最终两者相等。这说明什么？

**分析**：说明体系由开始反应逐步达到动态平衡。

**答案**：说明反应正在建立化学平衡，达到平衡时正逆反应速率相等且不为 0。

## 练习
1. 5 s 内某反应物浓度由 `0.80 mol/L` 变为 `0.55 mol/L`，平均速率是多少？
2. 反应 `2SO2 + O2 ⇌ 2SO3` 中，若 `v(O2) = 0.10 mol/(L·s)`，则 `v(SO3)` 是多少？
3. 为什么实验室制取 `CO2` 常用块状大理石而不用大理石粉末？
4. 判断正误：加入催化剂后，反应物的平衡转化率一定增大。

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `0.05 mol/(L·s)` |
| 2 | `0.20 mol/(L·s)` |
| 3 | 粉末反应过快，不利于控制；块状更便于平稳收集气体 |
| 4 | 错；催化剂通常只改变达到平衡的快慢 |

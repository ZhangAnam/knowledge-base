# 化学学案：甲烷、乙烯与乙炔

> 高中化学 | 有机化学线 O1

---

## 本课学什么

- 比较甲烷、乙烯、乙炔的结构特点和主要性质。
- 理解饱和烃与不饱和烃在取代、加成方面的差异。
- 能把结构、键型和反应类型联系起来。

## 原理

1. 甲烷属于烷烃，分子中只有单键，结构稳定，主要发生取代反应。
2. 乙烯和乙炔分别含双键、三键，属于不饱和烃，较易发生加成反应。
3. 有机物性质差异往往来自官能团和成键方式差异。

## 条件

| 物质 | 结构特点 | 典型反应 |
|---|---|---|
| 甲烷 | 只含单键 | 取代、燃烧 |
| 乙烯 | 含碳碳双键 | 加成、加聚、燃烧 |
| 乙炔 | 含碳碳三键 | 加成、燃烧 |

## 现象

- 乙烯、乙炔常能使溴水褪色；
- 乙烯可使酸性高锰酸钾溶液褪色；
- 甲烷通常不使溴水褪色；
- 燃烧时火焰特征和黑烟情况有所差异。

## 方程式

```chemical
CH4 + 2O2 = CO2 + 2H2O
C2H4 + Br2 → C2H4Br2
C2H4 + H2 → C2H6
2CH2=CH2 → (-CH2-CH2-)n
C2H2 + 2H2 → C2H6
```

## 易错点

- **溴水褪色不等于一定发生取代反应**，对乙烯、乙炔多是加成。
- 甲烷和氯气的取代反应通常需要 **光照** 等条件。
- 乙烯能加聚，甲烷不能直接加聚。
- 结构简式、分子式、最简式要分清。

## 递进例题

### 例1：结构判断

甲烷、乙烯、乙炔中，哪两种属于不饱和烃？

**答案**：乙烯和乙炔。

### 例2：反应类型

乙烯与溴发生反应，属于什么反应？

**答案**：加成反应。

### 例3：稳定性比较

为什么甲烷通常比乙烯更稳定？

**答案**：甲烷只含单键，结构更稳定。

### 例4：性质鉴别

怎样初步区分甲烷和乙烯？

**答案**：通入溴水，乙烯可使溴水褪色，甲烷通常不能。

## 练习

1. 乙烯含什么键？
2. 甲烷主要发生哪类典型反应？
3. 乙烯为什么能加聚？
4. 乙炔属于饱和烃还是不饱和烃？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 碳碳双键 |
| 2 | 取代反应 |
| 3 | 因为含不饱和键可打开形成长链 |
| 4 | 不饱和烃 |

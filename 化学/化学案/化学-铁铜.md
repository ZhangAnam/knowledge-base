# 化学学案：铁、铜及其化合物

> 高中化学 | 元素化合物线 E3

---

## 本课学什么
- 抓住铁、铜专题中的 **变价、检验、置换、腐蚀** 四个高频考点。
- 会比较 `Fe^{2+}` 与 `Fe^{3+}` 的性质差异，掌握铜与氧化性酸的反应。
- 会用典型现象判断铁盐、铜盐和金属活动性顺序。

## 原理
1. 铁的常见价态是 +2 和 +3，因此铁及其化合物的核心是 **变价**。
2. 铜活动性比氢弱，一般不与非氧化性酸反应，但能与 **硝酸、浓硫酸** 等氧化性酸反应。
3. 金属置换反应本质是更活泼金属失电子，较不活泼金属离子得电子。
4. 金属腐蚀与电化学有关，铁在潮湿空气中易形成原电池而被腐蚀。

## 条件
| 物质或体系 | 条件 | 典型结论 |
|---|---|---|
| Fe 与稀酸 | 常温 | 生成 `Fe^{2+}` 和 `H2` |
| `Fe^{2+}` 遇氧化剂 | 如 `Cl2`、酸性高锰酸钾 | 可被氧化成 `Fe^{3+}` |
| Cu 与浓硝酸 | 常温 | 生成 `NO2` |
| Cu 与稀硝酸 | 常温 | 生成 `NO` |
| 铁的腐蚀 | 潮湿空气、电解质存在 | 腐蚀加快 |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| Fe 加入 `CuSO4` 溶液 | 铁表面析出红色铜，溶液由蓝色变浅绿色 | 发生置换反应 |
| 向 `FeCl3` 溶液滴加 KSCN | 溶液显血红色 | 检验 `Fe^{3+}` |
| `Fe(OH)2` 暴露空气 | 白色沉淀迅速变灰绿再变红褐 | 被氧化成 `Fe(OH)3` |
| Cu 与浓硝酸反应 | 溶液变蓝，产生红棕色气体 | 生成 `Cu^{2+}` 和 `NO2` |

## 方程式
```chemical
Fe + 2HCl = FeCl2 + H2↑
Fe + CuSO4 = FeSO4 + Cu
2FeCl2 + Cl2 = 2FeCl3
Fe^{3+} + 3OH^- = Fe(OH)3↓
4Fe(OH)2 + O2 + 2H2O = 4Fe(OH)3
Cu + 4HNO3(浓) = Cu(NO3)2 + 2NO2↑ + 2H2O
3Cu + 8HNO3(稀) = 3Cu(NO3)2 + 2NO↑ + 4H2O
Cu + 2H2SO4(浓) = CuSO4 + SO2↑ + 2H2O（加热）
```

## 易错点
- 铁与盐酸、稀硫酸反应通常生成 **`Fe^{2+}`**，不是 `Fe^{3+}`。
- 检验 `Fe^{3+}` 常用 **KSCN 溶液**，出现血红色；检验 `Fe^{2+}` 常结合酸性高锰酸钾褪色等现象。
- 铜不能与非氧化性酸反应，不要把铜和盐酸、稀硫酸乱写成放氢气。
- `Fe(OH)2` 颜色变化快，实验中常需隔绝空气观察。

## 递进例题
### 例1：置换反应
把铁钉放入 `CuSO4` 溶液中，有什么现象？

**分析**：铁比铜活泼，能置换出铜。

**答案**：铁表面析出红色铜，溶液蓝色变浅并逐渐显浅绿色。

### 例2：变价判断
为什么 `FeCl2` 溶液久置后往往会部分变黄？

**分析**：`Fe^{2+}` 容易被空气中的氧气氧化为 `Fe^{3+}`。

**答案**：因为 `Fe^{2+}` 被氧化成 `Fe^{3+}`，生成含三价铁的物质后颜色加深偏黄。

### 例3：离子检验
如何检验某溶液中含有 `Fe^{3+}`？

**分析**：加入 KSCN 溶液观察是否显血红色。

**答案**：滴加 KSCN 溶液，若显血红色，则说明含有 `Fe^{3+}`。

### 例4：铜与硝酸
铜分别与浓硝酸、稀硝酸反应，生成的气体有什么不同？

**分析**：浓硝酸通常生成 `NO2`，稀硝酸通常生成 `NO`。

**答案**：浓硝酸生成红棕色 `NO2`，稀硝酸生成无色 `NO`（遇空气变红棕）。

## 练习
1. 写出 Fe 与稀盐酸反应的方程式。
2. 判断：铜可以与盐酸反应放出 `H2`。
3. `Fe^{3+}` 的常用检验试剂是什么？现象是什么？
4. `Fe(OH)2` 在空气中最终变成什么颜色的沉淀？

## 答案
| 题号 | 答案 |
|---|---|
| 1 | `Fe + 2HCl = FeCl2 + H2↑` |
| 2 | 错 |
| 3 | KSCN 溶液；显血红色 |
| 4 | 红褐色 |

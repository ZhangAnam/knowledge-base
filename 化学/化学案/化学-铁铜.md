# 高中化学：铁铜

> 高中化学 | 元素化合物

---

## 📌 铁的性质

### 物理性质
银白色，有磁性

### 化学性质
- 与非金属：3Fe + 2O₂ → Fe₃O₄
- 与酸：Fe + 2H⁺ → Fe²⁺ + H₂↑
- 与Cu²⁺：Fe + Cu²⁺ → Fe²⁺ + Cu

---

## 📌 铁的化合物

### FeO（氧化亚铁）
黑色粉末

### Fe₂O₃（氧化铁）
红棕色

### Fe₃O₄（四氧化三铁）
黑色，有磁性

### Fe(OH)₂
白色 → 灰绿色 → 红褐色

### Fe(OH)₃
红褐色

---

## 📌 铜的性质

### 与非金属
Cu + Cl₂ → CuCl₂

### 与硝酸
Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

---

## 📝 典型例题

### 例1：Fe²⁺/Fe³⁺ 鉴别
用KSCN溶液鉴别Fe²⁺和Fe³⁺。

> 📋 答案
> Fe³⁺ + 3SCN⁻ → Fe(SCN)₃（血红色）
> Fe²⁺ 无此现象

# 高中化学：氮气氨气硝酸

> 高中化学 | 元素化合物

---

## 📌 氮气

### 性质
- 稳定：N₂ + 3H₂ ⇌ 2NH₃（工业合成氨）
- 与镁：3Mg + N₂ → Mg₃N₂

---

## 📌 氨气

### 性质
- 刺激性气味
- 极易溶于水：NH₃ + H₂O ⇌ NH₃·H₂O
- 碱性：NH₃ + HCl → NH₄Cl

### 制备
2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃↑ + 2H₂O

---

## 📌 硝酸

### 性质
- 不稳定性：4HNO₃ → NO₂↑ + O₂↑ + 2H₂O
- 强氧化性：与Cu反应

### 与Cu反应
Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

---

## 📝 典型例题

### 例1：喷泉实验
NH₃ 溶于水形成喷泉的原因。

> 📋 答案
> NH₃ 极易溶于水，瓶内压强迅速降低，大气压将水压入形成喷泉

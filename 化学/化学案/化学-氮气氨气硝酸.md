# 化学学案：氮气、氨气与硝酸

> 高中化学 | 元素化合物线 E6

---

## 本课学什么
- 抓住氮专题的三条主线：**`N2` 稳定性、`NH3` 碱性与喷泉实验、`HNO3` 强氧化性**。
- 会比较氮元素在 `0`、`-3`、`+5` 等价态下的典型性质。
- 会根据实验条件和现象判断合成氨、氨的制取和硝酸与金属反应。

## 原理
1. `N2` 分子中存在稳定的三键，因此常温下化学性质不活泼。
2. `NH3` 极易溶于水，并与水作用形成弱碱性体系，所以常见喷泉实验和碱性检验。
3. 硝酸除了酸性外，更重要的是 **强氧化性**，多数金属与硝酸反应不放出 `H2`。
4. 氮元素在不同价态间转化常伴随氧化还原过程，是氮专题综合题的核心。

## 条件
| 物质 | 条件 | 典型结论 |
|---|---|---|
| `N2 + H2` | 高温、高压、催化剂 | 工业合成氨 |
| 实验室制 `NH3` | 铵盐与强碱共热 | 生成氨气 |
| `NH3` 溶于水 | 常温 | 显碱性，形成喷泉 |
| Cu 与浓硝酸 | 常温 | 生成 `NO2` |
| Cu 与稀硝酸 | 常温 | 生成 `NO` |

## 现象
| 情境 | 现象 | 说明 |
|---|---|---|
| `NH3` 喷泉实验 | 水迅速进入烧瓶形成喷泉 | `NH3` 极易溶于水 |
| `NH3` 与 HCl 相遇 | 产生白烟 | 生成 `NH4Cl` |
| Cu 与浓硝酸反应 | 产生红棕色气体，溶液变蓝 | 生成 `NO2` 和 `Cu^{2+}` |
| Cu 与稀硝酸反应 | 先产生无色气体，遇空气变红棕 | 生成 `NO` |

## 方程式
```chemical
N2 + 3H2 ⇌ 2NH3
2NH4Cl + Ca(OH)2 = CaCl2 + 2NH3↑ + 2H2O（加热）
NH3 + H2O ⇌ NH3·H2O
NH3 + HCl = NH4Cl
Cu + 4HNO3(浓) = Cu(NO3)2 + 2NO2↑ + 2H2O
3Cu + 8HNO3(稀) = 3Cu(NO3)2 + 2NO↑ + 4H2O
```

## 易错点
- `N2` 稳定不等于“完全不反应”，在高温、高压、放电等条件下仍可反应。
- `NH3` 是 **碱性气体**，但其水溶液是弱碱，不要把它当成强碱。
- 金属与硝酸反应通常 **不放 `H2`**，因为硝酸的氧化性更强。
- 检验 `NH3` 常用湿润红色石蕊试纸变蓝，或与浓盐酸产生白烟。

## 递进例题
### 例1：氮气稳定性
为什么 `N2` 在常温下通常不活泼？

**分析**：氮氮三键键能大，难以断裂。

**答案**：因为 `N≡N` 键稳定，断键需要较高能量。

### 例2：氨气喷泉
喷泉实验中水为什么会迅速进入盛有 `NH3` 的烧瓶？

**分析**：`NH3` 极易溶于水，使瓶内压强迅速减小。

**答案**：因为 `NH3` 快速溶于水，瓶内压强降低，大气压把水压入烧瓶形成喷泉。

### 例3：硝酸与铜
铜与浓硝酸、稀硝酸反应时产生的气体各是什么？

**分析**：与浓硝酸生成 `NO2`，与稀硝酸生成 `NO`。

**答案**：浓硝酸生成 `NO2`，稀硝酸生成 `NO`。

### 例4：综合判断
实验室制氨气时，能否用向下排空气法收集？为什么？

**分析**：`NH3` 密度比空气小，且极易溶于水，不宜用排水法。

**答案**：能，用向下排空气法；因为 `NH3` 密度比空气小且易溶于水。

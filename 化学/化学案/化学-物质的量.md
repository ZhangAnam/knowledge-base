# 化学学案：物质的量

> 高中化学 | 计算模型线 C1

---

## 本课学什么

- 理解物质的量、阿伏加德罗常数、摩尔质量的含义。
- 会在粒子数、质量、物质的量、气体体积之间进行换算。
- 能判断标准状况、气体、溶液等条件是否满足对应公式的使用前提。

## 原理

1. **物质的量** 是连接微观粒子数与宏观质量、体积的桥梁，符号是 `n`，单位是 `mol`。
2. 1 mol 任意粒子都含有相同数目的微粒，这个数叫 **阿伏加德罗常数**，记作 `N_A`。
3. 常用关系式：

```text
n = m/M = N/N_A
```

4. 对气体而言，在标准状况下可用：

```text
n = V/22.4
```

单位中 `V` 取 `L`。

## 条件

| 任务 | 使用条件 | 关键提醒 |
|---|---|---|
| `n = m/M` | 已知质量和摩尔质量 | `M` 的数值常与相对分子质量一致，单位是 `g/mol` |
| `n = N/N_A` | 已知粒子数 | 粒子可指分子、原子、离子等 |
| `n = V/22.4` | **标准状况下气体** | 不是所有气体体积都可直接除以 22.4 |
| 溶液中求粒子数 | 先求物质的量 | 再结合化学式中粒子个数关系 |

## 现象

物质的量本身不是实验现象，但它常用于解释：

- 相同物质的量的不同物质，所含粒子数相同；
- 相同标准状况下的不同气体，若体积相同，则物质的量相同；
- 溶液反应中的系数关系，本质上常是物质的量关系。

## 方程式 / 关系式

```text
n = m/M
n = N/N_A
n = V/22.4（标准状况，气体）
N = nN_A
m = nM
```

## 易错点

- **物质的量不是“物质的数量”**，它是专门的物理化学量。
- `22.4 L/mol` 只适用于 **标准状况下的气体**，液体和固体不能直接用。
- 阿伏加德罗常数对应的是 **粒子个数**，不是原子数、电子数都能直接不加分析地套。
- 求离子、原子个数时，要先看 1 个化学式单位中含几个目标粒子。

## 递进例题

### 例1：质量求物质的量

`49 g H2SO4` 的物质的量是多少？

**解答**：

```text
M(H2SO4) = 98 g/mol
n = 49/98 = 0.5 mol
```

### 例2：物质的量求粒子数

`0.2 mol NaCl` 中含多少个 `NaCl` 粒子？

**解答**：

```text
N = nN_A = 0.2N_A
```

### 例3：标准状况气体换算

标准状况下 `11.2 L O2` 的物质的量是多少？

**解答**：

```text
n = 11.2/22.4 = 0.5 mol
```

### 例4：原子个数换算

`1 mol H2O` 中含多少 mol H 原子？

**解答**：1 个 `H2O` 含 2 个 H 原子，所以含 `2 mol` H 原子。

## 练习

1. `18 g H2O` 的物质的量是多少？
2. `0.5 mol CO2` 含多少 mol O 原子？
3. 标准状况下 `44.8 L N2` 的物质的量是多少？
4. `0.1 mol Na2SO4` 中含多少 mol `Na+`？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `1 mol` |
| 2 | `1 mol` |
| 3 | `2 mol` |
| 4 | `0.2 mol` |

# 化学知识体系（重构版）

> 适用范围：初高衔接到高中复习
> 
> 组织方式：不再按教材目录平铺，而按 **概念—结构—反应—实验—计算—推理** 六条主线组织，便于学案、题型和复习路径一一对应。

---

## 一、使用说明

1. **先找主线，再找专题**：先判断题目属于结构、反应、实验、计算还是推理，再回到对应学案。
2. **每份学案都要回答七件事**：概念、原理、条件、现象、方程式、易错点、练习。
3. **元素化合物不孤立记忆**：必须放回“结构决定性质、性质决定反应、反应落实到实验与计算”的链条中理解。

## 九、建议学习顺序

1. **结构起步**：原子结构 → 核外电子排布 → 周期律 → 化学键 → 分子间作用力 → 晶体。
2. **反应规则建模**：氧化还原 → 离子反应 → 速率 → 平衡 → 电化学。
3. **元素专题串联**：钠 → 铝 → 铁铜 → 氯 → 硫 → 氮。
4. **有机与实验补齐**：烃 → 乙醇乙酸 → 官能团转化 → 同分异构 → 聚合 → 推断。
5. **计算与推理收束**：物质的量 → 浓度 → 方程式计算 → 守恒差量 → 实验流程推断。

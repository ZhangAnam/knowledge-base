# 高中物理4-电容

> 高中物理｜电磁学主线 B1-6  
> 主题定位：研究电容器“容纳电荷能力”及其与电场、电压、能量的联系

## 本课学什么

本课围绕电容器展开，重点理解电容的物理意义，以及平行板电容器中 `C、Q、U、E` 的动态关系。  
学完本课，应能区分定义式和决定式，会分析“与电源相连”和“充电后断开”两类经典变化问题，并能理解电容器储能。

---

## 核心概念 / 规律 / 模型

### 一、电容器与电容

电容器由两个彼此绝缘又相互靠近的导体组成，用来储存电荷和电能。

电容定义：

```text
C = Q / U
```

含义：

- 电容越大，表示在相同电压下能容纳更多电荷；
- 或者说在带相同电荷时，两板间电压更小。

单位：

- `1 F = 10^6 μF = 10^12 pF`

### 二、定义式与决定式

#### 1. 定义式

```text
C = Q / U
```

用于说明电容的物理意义，但不能理解为电容会随着 `Q`、`U` 的改变而随意改变。

#### 2. 平行板电容器决定式

```text
C ∝ εS / d
```

即：

- 正对面积 `S` 越大，电容越大；
- 板间距离 `d` 越大，电容越小；
- 插入介质后，电容通常增大。

### 三、平行板电容器中的匀强电场模型

忽略边缘效应时，两板间近似为匀强电场：

```text
E = U / d
```

这使得电容问题常与电场问题结合。

### 四、两类经典动态模型

#### 模型1：充电后断开电源

断开后通常认为：

- `Q` 不变
- `C` 随板间结构变化而变
- 再由 `U = Q / C`、`E = U / d` 推其他量

对平行板电容器，若只改变板间距离：

- `d` 增大 → `C` 减小
- `Q` 不变 → `U` 增大
- `E = U/d` 可进一步判断

#### 模型2：始终接在电源两端

接电源时通常认为：

- `U` 不变
- `C` 改变
- 再由 `Q = CU`、`E = U/d` 推其他量

### 五、电容器储能

电容器储存的电能：

```text
W = 1/2 QU = 1/2 CU² = Q² / 2C
```

这说明电容器不仅“带电”，还会“储能”。

## 适用条件

| 内容 | 适用条件 | 说明 |
|---|---|---|
| `C = Q/U` | 任意电容器 | 是定义式，用于理解电容含义 |
| `C ∝ εS/d` | 平行板电容器 | 忽略边缘效应 |
| `E = U/d` | 板间电场近似匀强 | 常见于平行板模型 |
| 动态分析 | 先分清是否接电源 | “Q不变”与“U不变”不能混用 |

---

## 易错点

1. **误认为 `C = Q/U` 表示 C 随 Q、U 变化**  
   对同一个电容器，在结构和介质不变时，`C` 是定值。

2. **没先判断是否接电源**  
   这是电容动态题第一步，错了后面全错。

3. **把 `E = U/d` 和 `E = Q/...?` 混着用**  
   电场强度判断必须基于当前已知条件。

4. **板间距离变大就想当然说场强变大或变小**  
   要分“断电”还是“接电源”。

5. **忽视电容器储能**  
   很多综合题会把机械过程和能量变化联系起来。

## 递进例题

### 例1：基础——电容定义计算

电容器带电量 `Q = 4.0 × 10^-6 C`，两端电压 `U = 200 V`，求电容。

**解答**

```text
C = Q / U
  = 4.0 × 10^-6 / 200
  = 2.0 × 10^-8 F
```

### 例2：关键——接电源时的动态判断

平行板电容器始终接在电源两端，增大板间距离 `d`，判断 `C、Q、E` 如何变化。

**分析**

- 接电源：`U` 不变；
- `d` 增大 → `C` 减小；
- `Q = CU` → `Q` 减小；
- `E = U/d` → `E` 减小。

**结论**

`C` 减小，`Q` 减小，`E` 减小。

### 例3：易错——断开电源后的变化

平行板电容器充电后断开电源，再增大板间距离 `d`，判断 `Q、U` 如何变化。

**分析**

- 断开后：`Q` 不变；
- `d` 增大 → `C` 减小；
- `U = Q/C` → `U` 增大。

**结论**

`Q` 不变，`U` 增大。

### 例4：综合——储能计算

一个电容为 `5.0 × 10^-6 F` 的电容器，两端电压 `100 V`，求其储存的电能。

**解答**

```text
W = 1/2 CU²
  = 1/2 × 5.0 × 10^-6 × 100²
  = 2.5 × 10^-2 J
```

## 练习

1. 电容器两端电压 `50 V`，所带电量 `1.0 × 10^-5 C`，求电容。
2. 平行板电容器接电源后增大正对面积，`C` 如何变化？
3. 充电后断开电源的平行板电容器，减小板间距离，`U` 如何变化？
4. 电容器电容 `2.0 × 10^-6 F`，电压 `300 V`，求储能。

---

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `C = Q/U = 1.0 × 10^-5 / 50 = 2.0 × 10^-7 F` |
| 2 | 增大 |
| 3 | 减小 |
| 4 | `W = 1/2 CU² = 1/2 × 2.0 × 10^-6 × 300² = 9.0 × 10^-2 J` |

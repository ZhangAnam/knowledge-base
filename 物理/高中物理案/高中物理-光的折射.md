# 高中物理：光的折射

> 高中物理 | 光学 | 知识体系：C4-1 ~ C4-3

---

## 本课学什么

本课学习光的折射现象、折射定律、折射率和全反射的基础认识。学完后，应能判断光从一种介质进入另一种介质时传播方向怎样变化，理解“密到疏、疏到密”的偏折规律。

## 核心概念 / 规律 / 模型

### 一、光的折射现象

光从一种介质斜射入另一种介质时，传播方向发生改变，这种现象叫折射。

### 二、折射定律

在两种介质分界面上，入射光线、法线、折射光线在同一平面内，并满足：

```text
n1 sinθ1 = n2 sinθ2
```

### 三、折射率

折射率反映介质对光传播速度的影响。一般说来，折射率越大，光在该介质中传播速度越小。

### 四、偏折规律

- 从疏到密：折射角小于入射角，向法线偏折；
- 从密到疏：折射角大于入射角，背离法线偏折。

### 五、全反射初步

光从光密介质射向光疏介质时，当入射角大于某一临界角，可能发生全反射。

## 适用条件

- 折射定律用于分析光在不同介质界面的传播。
- 角度要相对法线来量，不是相对界面。
- 全反射只可能出现在光从光密介质射向光疏介质时。

## 易错点

1. **把角度相对界面量取**
2. **把“密到疏”和“疏到密”方向搞反**
3. **认为垂直入射没有折射现象**
4. **全反射条件记不全**

## 递进例题

### 例 1：偏折方向判断

光从空气斜射入水中，向法线偏还是背离法线偏？

**答案**：向法线偏折。

### 例 2：折射率理解

折射率较大的介质中，光速通常怎样？

**答案**：更小。

### 例 3：全反射条件

全反射一定发生在什么情况下？

**答案**：从光密介质射向光疏介质，且入射角大于临界角。

### 例 4：垂直入射

垂直射入界面时，光线方向变吗？

**答案**：方向不变，但仍可认为发生了折射过程。

## 练习

1. 折射角和入射角都是相对什么测量？
2. 光从玻璃进入空气，通常向哪里偏？
3. 折射率大的介质中光速是更大还是更小？
4. 全反射需要几个条件？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 法线 |
| 2 | 背离法线 |
| 3 | 更小 |
| 4 | 两个：密到疏、入射角大于临界角 |

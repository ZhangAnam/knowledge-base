# 高中物理：气体状态方程

> 高中物理 | 热学 | 知识体系：C1-4 ~ C1-6

---

## 本课学什么

本课学习气体状态参量、理想气体状态方程以及等温、等压、等容变化的基本规律。学完后，应能在压强、体积、温度三者之间建立联系，能根据过程条件选择合适公式。

## 核心概念 / 规律 / 模型

### 一、气体状态参量

描述气体状态的常用物理量有：

- 压强 `p`
- 体积 `V`
- 热力学温度 `T`

### 二、理想气体状态方程

一定质量的理想气体满足：

```text
pV/T = 常量
```

也可写成：

```text
p1V1/T1 = p2V2/T2
```

### 三、三种特殊过程

- 等温变化：温度不变，`pV = 常量`
- 等容变化：体积不变，`p/T = 常量`
- 等压变化：压强不变，`V/T = 常量`

### 四、绝对温度的重要性

状态方程中的温度必须用开尔文温度，不能直接用摄氏温度。

换算关系：

```text
T = t + 273
```

### 五、图像与过程分析

做题时先判断哪一个量保持不变，再选相应关系式。复杂题可分阶段处理。

## 适用条件

- 适用于一定质量的理想气体。
- 温度必须使用热力学温度。
- 多过程问题要分段列式。

## 易错点

1. **直接把摄氏温度代入公式**
2. **不区分等温、等压、等容过程**
3. **状态方程中气体质量发生改变却仍直接套公式**
4. **图像题只记结论，不看过程条件**

## 递进例题

### 例 1：等温变化

一定质量气体在温度不变时，体积变为原来的 `1/2`，压强如何变化？

**答案**：压强变为原来的 2 倍。

### 例 2：等压变化

压强不变时，温度从 `300 K` 升高到 `450 K`，体积如何变化？

**答案**：体积变为原来的 `1.5` 倍。

### 例 3：综合状态方程

已知 `p1、V1、T1` 和 `p2、V2`，求 `T2`，应使用什么关系？

**答案**：`p1V1/T1 = p2V2/T2`。

### 例 4：温标换算

`27℃` 等于多少开尔文？

**答案**：`300 K`。

## 练习

1. 气体状态参量有哪些？
2. 等温变化满足什么关系？
3. 状态方程中的温度用什么单位？
4. `127℃` 等于多少开尔文？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 压强、体积、热力学温度 |
| 2 | `pV = 常量` |
| 3 | 开尔文 |
| 4 | `400 K` |

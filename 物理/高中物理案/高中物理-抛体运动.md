# 高中物理：抛体运动

> 高中物理 | 力学

---

## 📌 核心概念

### 抛体运动
- **条件**：初速度不为零，只受重力
- **性质**：匀变速曲线运动
- **分解**：水平（匀速）+ 竖直（匀变速）

---

## 📌 平抛运动

### 水平方向
$$x = v_0 t$$
$$v_x = v_0$$

### 竖直方向
$$y = \frac{1}{2}gt^2$$
$$v_y = gt$$

### 速度
$$v = \sqrt{v_0^2 + (gt)^2}$$
$$\tan\theta = \frac{v_y}{v_x} = \frac{gt}{v_0}$$

### 位移
$$s = \sqrt{x^2 + y^2}$$

---

## 📌 斜抛运动

### 水平射程
$$X = v_0^2 \sin 2\alpha / g$$

### 最大高度
$$H = v_0^2 \sin^2 \alpha / 2g$$

### 射高与射程最大
- 射高最大：α = 90°
- 射程最大：α = 45°

---

## 📝 典型例题

### 例1：平抛计算
飞机在500m高空以100m/s水平飞行，投弹求：
(1) 飞行时间 (2) 水平距离

> 📋 答案
> (1) h = ½gt²，500 = 5t²，t = 10s
> (2) x = v₀t = 100×10 = 1000m

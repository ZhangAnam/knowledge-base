# 高中物理：核物理

> 高中物理 | 近代物理 | 知识体系：C3-3 ~ C3-5

---

## 本课学什么

本课学习原子核的组成、放射性、核反应和质能关系。学完后，应能区分 α、β、γ 三种射线的特点，理解半衰期的意义，也能知道核能来源于质量亏损对应的能量释放。

## 核心概念 / 规律 / 模型

### 一、原子核的组成

原子核由质子和中子组成。

核素常写成：

```text
^A_ZX
```

其中 `A` 是质量数，`Z` 是质子数。

### 二、放射现象

某些原子核会自发放出射线，这种现象叫放射性。

常见射线：

- α 射线：带正电，穿透能力较弱，电离能力强
- β 射线：带电粒子流，穿透能力较强
- γ 射线：电磁波，穿透能力最强

### 三、半衰期

大量放射性原子核中，有半数发生衰变所需时间叫半衰期。

半衰期反映衰变快慢，是核本身的属性。

### 四、核反应与核能

核裂变、核聚变都会伴随巨大能量变化。

### 五、质能关系

爱因斯坦质能关系：

```text
E = mc²
```

质量亏损对应能量释放。

## 适用条件

- 半衰期适用于大量原子核的统计规律。
- 核反应遵守质量数守恒和电荷数守恒。
- 不能把化学反应释放能量与核能量级混同。

## 易错点

1. **把半衰期理解成“全部衰变完的一半时间”**
2. **把 α、β、γ 本质混淆**
3. **核反应中忘记守恒质量数和电荷数**
4. **把质量亏损理解成质量真正消失**

## 递进例题

### 例 1：射线比较

三种射线中，哪种穿透能力最强？

**答案**：γ 射线。

### 例 2：半衰期理解

某放射性元素半衰期为 3 天，6 天后剩余多少？

**答案**：剩余原来的 `1/4`。

### 例 3：核反应守恒

书写核反应方程时要检查什么守恒？

**答案**：质量数守恒和电荷数守恒。

### 例 4：核能来源

核反应释放能量的根本原因是什么？

**答案**：质量亏损对应的能量释放。

## 练习

1. 原子核由什么组成？
2. 半衰期反映什么？
3. α、β、γ 中哪种电离能力强？
4. 质能关系公式是什么？

| 题号 | 答案 |
|---|---|
| 1 | 质子和中子 |
| 2 | 衰变快慢 |
| 3 | α 射线 |
| 4 | `E = mc²` |

# 高中物理5：恒定电流

> 高中物理 | 电磁学

---

## 📌 核心概念

### 电流 I
- **定义**：单位时间内通过导体横截面的电荷量
- **公式**：$I = \frac{q}{t}$
- **单位**：安培（A）
- **方向**：正电荷定向移动的方向

### 电压 U
- **定义**：使电荷定向移动形成电流的原因
- **单位**：伏特（V）

### 电阻 R
- **定义**：导体阻碍电流的性质
- **公式**：$R = \frac{U}{I}$
- **单位**：欧姆（Ω）

---

## 📌 重要定律

### 欧姆定律
$$I = \frac{U}{R}$$
- 适用于金属、电解液（纯电阻）

### 电阻定律
$$R = \rho\frac{l}{S}$$
- ρ：材料的电阻率
- l：导体长度
- S：横截面积

### 串联电路特点
- $I$ = $I_1$ = $I_2$ = ...
- $U$ = $U_1$ + $U_2$ + ...
- $R$ = $R_1$ + $R_2$ + ...
- $\frac{U_1}{U_2} = \frac{R_1}{R_2}$

### 并联电路特点
- $I$ = $I_1$ + $I_2$ + ...
- $U$ = $U_1$ = $U_2$ = ...
- $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}$
- $\frac{I_1}{I_2} = \frac{R_2}{R_1}$

---

## 📌 电功与电功率

### 电功 W
$$W = UIt$$

### 电功率 P
$$P = UI = I^2R = \frac{U^2}{R}$$

### 焦耳定律
$$Q = I^2Rt$$

---

## 📝 典型例题

### 例1：电阻计算
某导体两端电压为6V，通过电流为0.3A，求电阻。

> 📋 答案
> R = U/I = 6/0.3 = 20Ω

### 例2：串并联计算
两个电阻R₁=10Ω，R₂=20Ω。求串联和并联时的等效电阻。

> 📋 答案
> 串联：R = 10 + 20 = 30Ω
> 并联：1/R = 1/10 + 1/20 = 3/20，R = 20/3 ≈ 6.67Ω

---

## 🎯 高考技巧

1. **电路分析**：先识别串并联，再计算
2. **动态电路**：滑动变阻器引起的动态变化
3. **功率最值**：纯电阻电路中P=U²/R或P=I²R
4. **故障分析**：短路、断路的判断

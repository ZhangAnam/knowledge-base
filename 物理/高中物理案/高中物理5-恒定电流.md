# 高中物理5-恒定电流

> 高中物理｜电磁学主线 B2-1 ~ B2-4  
> 主题定位：建立电路基本量、欧姆定律和电功率分析框架

---

## 本课学什么

本课研究恒定电流电路中的基本物理量：电流、电阻、电压、电功、电功率。  
学完本课，应能区分电流形成条件与方向规定，理解欧姆定律和电阻定律的意义，会分析串并联电路，并能从能量角度处理电功、电热和功率问题。

---

## 核心概念 / 规律 / 模型

### 一、电流

电流表示电荷的定向移动强弱：

```text
I = q / t
```

方向规定为**正电荷定向移动方向**。  
在金属导体中，实际移动的是自由电子，所以电子移动方向与电流方向相反。

### 二、形成恒定电流的条件

要在导体中形成持续稳定的电流，需要：

1. 导体中有自由移动电荷；
2. 导体两端保持电势差。

### 三、电阻与电阻定律

电阻反映导体对电流的阻碍作用。

对同一导体，在一定条件下：

```text
R = U / I
```

导体电阻还与材料和几何形状有关：

```text
R = ρl / S
```

其中：

- `ρ`：电阻率
- `l`：长度
- `S`：横截面积

### 四、欧姆定律与伏安特性

对纯电阻导体：

```text
I = U / R
```

图像上，若 `U-I` 图像是一条过原点直线，则可认为该元件在此范围内满足欧姆定律。

### 五、串并联模型

#### 1. 串联

- 电流处处相等
- 总电压等于各部分电压之和
- 总电阻等于各电阻之和

#### 2. 并联

- 各支路电压相等
- 总电流等于各支路电流之和
- 总电阻满足倒数和关系

串并联分析的本质，是把复杂电路转化成等效电路。

### 六、电功、电热、电功率

```text
W = UIt
P = UI
Q = I²Rt
```

对纯电阻电路，还可写为：

```text
P = I²R = U² / R
```

这反映了电能转化为内能的过程。

---

## 适用条件

| 内容 | 适用条件 | 说明 |
|---|---|---|
| `I = q/t` | 平均电流定义 | 反映单位时间通过截面的电荷量 |
| `R = U/I` | 对导体某一工作状态成立 | 是电阻定义，不是决定式 |
| `R = ρl/S` | 均匀导体 | 温度变化明显时需谨慎 |
| `I = U/R` | 欧姆元件 | 非线性元件不能机械套用 |
| `P = U²/R`、`I²R` | 纯电阻电路 | 非纯电阻电路要优先用 `P = UI` |

---

## 易错点

1. **把电流方向和电子移动方向混为一谈**  
   规定方向按正电荷，不按电子。

2. **误把 `R = U/I` 当成“电压越大电阻越大”**  
   对同一欧姆导体，电阻本身由材料和几何条件决定。

3. **把所有用电器都当纯电阻**  
   电动机等元件不能直接用 `P = I²R` 表示总功率。

4. **串并联识别错误**  
   是否串联要看是否同一路径，是否并联要看两端是否分别接在同两点。

5. **电功与电热不区分**  
   纯电阻中两者相等，非纯电阻中一般不相等。

## 递进例题

### 例1：基础——欧姆定律求电阻

某导体两端电压 `6 V`，通过电流 `0.30 A`，求电阻。

**解答**

```text
R = U / I = 6 / 0.30 = 20 Ω
```

### 例2：关键——串并联等效

两个电阻 `R1 = 6 Ω`、`R2 = 3 Ω`。

（1）串联时总电阻多少？  
（2）并联时总电阻多少？

**解答**

```text
串联：R总 = R1 + R2 = 9 Ω
并联：1/R总 = 1/6 + 1/3 = 1/2
所以 R总 = 2 Ω
```

### 例3：易错——电子定向移动方向

一段金属导线中电流方向向右，判断电子定向移动方向。

**结论**

电子定向移动方向向左。

### 例4：综合——功率与发热

一只 `10 Ω` 电阻接在 `5 V` 电源两端，求：

（1）电流；  
（2）电功率；  
（3）10 s 内产生的热量。

**解答**

```text
I = U / R = 5 / 10 = 0.5 A
P = UI = 5 × 0.5 = 2.5 W
Q = I²Rt = 0.5² × 10 × 10 = 25 J
```

## 练习

1. 一段导体中 `20 s` 内通过截面的电荷量为 `10 C`，求电流。
2. 导体电阻率不变时，若长度变为原来的 2 倍、横截面积变为原来的 1/2，电阻变为原来的多少倍？
3. 两个电阻 `4 Ω` 与 `12 Ω` 并联，总电阻多少？
4. 一个纯电阻元件两端电压 `12 V`，电流 `2 A`，求功率。

---

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `I = q/t = 10/20 = 0.5 A` |
| 2 | `R' = ρ(2l)/(S/2) = 4ρl/S`，变为原来的 `4` 倍 |
| 3 | `1/R = 1/4 + 1/12 = 1/3`，所以 `R = 3 Ω` |
| 4 | `P = UI = 12 × 2 = 24 W` |

# 高中物理8-磁场

> 高中物理｜电磁学主线 B3-1 ~ B3-3  
> 主题定位：建立磁场、磁感应强度、安培力与洛伦兹力模型

---

## 本课学什么

本课研究磁场如何描述磁相互作用，以及磁场对电流和运动电荷分别如何作用。  
学完本课，应能理解磁感应强度的物理意义，会用左手定则判断安培力、洛伦兹力方向，能处理带电粒子在匀强磁场中的圆周运动模型。

---

## 核心概念 / 规律 / 模型

### 一、磁场与磁感应强度

磁体、电流周围存在磁场。  
磁场也是一种客观存在的物质，用来描述磁相互作用。

磁感应强度 `B` 是描述磁场强弱和方向的物理量。

在导线与磁场垂直时：

```text
B = F / IL
```

其中：

- `F` 为安培力
- `I` 为电流
- `L` 为导线在磁场中的有效长度

方向规定为磁场方向，可用小磁针北极受力方向理解。

### 二、磁感线与常见磁场

磁感线特点：

- 磁感线外部从 N 极指向 S 极；
- 疏密表示磁场强弱；
- 磁感线闭合，不存在起点和终点。

常见磁场模型：

1. 条形磁铁周围磁场
2. 直线电流周围磁场
3. 通电螺线管内部近似匀强磁场

### 三、安培力

通电导线在磁场中受到的力：

```text
F = BILsinθ
```

其中 `θ` 为电流方向与磁场方向夹角。

当导线与磁场垂直时：

```text
F = BIL
```

方向用**左手定则**判断。

### 四、洛伦兹力

运动电荷在磁场中受到的力：

```text
f = qvBsinθ
```

特点：

- 静止电荷不受洛伦兹力；
- 洛伦兹力始终垂直于速度方向；
- 洛伦兹力不做功，只改变速度方向，不改变速度大小。

### 五、带电粒子在匀强磁场中的圆周模型

当粒子速度方向与磁场方向垂直时，洛伦兹力提供向心力：

```text
qvB = mv² / r
```

得到：

```text
r = mv / qB
T = 2πm / qB
```

结论：

- 半径与 `m、v` 成正比，与 `q、B` 成反比；
- 周期与速度无关。

### 六、电场与磁场作用的比较

| 项目 | 电场力 | 洛伦兹力 |
|---|---|---|
| 是否对静止电荷起作用 | 是 | 否 |
| 力的方向 | 与场方向有关 | 与 `v` 和 `B` 都垂直 |
| 是否做功 | 可能做功 | 不做功 |

---

## 适用条件

| 内容 | 适用条件 | 说明 |
|---|---|---|
| `F = BILsinθ` | 通电导线处于磁场中 | `L` 为有效长度 |
| `f = qvBsinθ` | 运动电荷进入磁场 | 静止电荷不适用 |
| `qvB = mv²/r` | `v ⟂ B`，粒子只受洛伦兹力 | 常用于圆周运动模型 |
| 左手定则 | 判断安培力或洛伦兹力方向 | 对负电荷要注意方向反转 |

---

## 易错点

1. **磁场方向和受力方向混淆**  
   磁场方向不是受力方向，要用定则判断。

2. **把洛伦兹力和安培力当成同一个对象**  
   前者作用于运动电荷，后者作用于通电导线。

3. **忘记 `sinθ` 条件**  
   平行进入磁场时，洛伦兹力为零。

4. **认为洛伦兹力会使粒子变快或变慢**  
   洛伦兹力不做功，只改方向。

5. **负电荷方向直接照搬左手定则结果**  
   定则通常先按正电荷判断，再反向。

## 递进例题

### 例1：基础——安培力大小

一段长 `0.40 m` 的导线垂直放在 `0.50 T` 匀强磁场中，通电流 `2.0 A`，求安培力大小。

**解答**

```text
F = BIL = 0.50 × 2.0 × 0.40 = 0.40 N
```

### 例2：关键——洛伦兹力方向

磁场方向垂直纸面向里，正电荷向右运动，判断洛伦兹力方向。

**结论**

用左手定则判断，洛伦兹力方向向上。

### 例3：易错——平行进入磁场

带电粒子速度方向与磁场方向平行，是否受洛伦兹力？

**结论**

不受。因为 `sin0° = 0`。

### 例4：综合——圆周半径

质量 `m = 2.0 × 10^-27 kg`、电荷量 `q = 1.6 × 10^-19 C` 的粒子，以 `3.2 × 10^6 m/s` 垂直进入 `0.20 T` 匀强磁场，求轨道半径。

**解答**

```text
r = mv / qB
  = (2.0 × 10^-27 × 3.2 × 10^6) / (1.6 × 10^-19 × 0.20)
  = 0.20 m
```

## 练习

1. 一段导线长 `0.30 m`，电流 `5 A`，与 `0.40 T` 磁场垂直，求安培力大小。
2. 电子垂直进入匀强磁场后，速度大小是否变化？
3. 带电粒子若沿磁场方向射入磁场，轨迹是什么？
4. 判断：磁感线可以相交。

---

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `F = BIL = 0.40 × 5 × 0.30 = 0.60 N` |
| 2 | 不变 |
| 3 | 做匀速直线运动 |
| 4 | 错 |

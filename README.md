# Obsidian 个人知识库使用说明

## 📂 知识库位置

```
/root/.openclaw/workspace/knowledge-base/
```

## 🚀 如何在 Obsidian 中打开

1. 下载并安装 [Obsidian](https://obsidian.md/)
2. 打开 Obsidian，点击"打开文件夹作为仓库"
3. 选择 `/root/.openclaw/workspace/knowledge-base` 文件夹

## 📚 知识库结构

```
knowledge-base/
├── 00-首页.md              # 知识库入口
├── 技术/                   # 技术相关笔记
│   ├── 技术知识.md
│   └── Obsidian使用技巧.md
├── 生活/                   # 生活相关笔记
├── 工作/                   # 工作相关笔记
├── 归档/                   # 存档内容
└── 附件/                   # 图片、文件等附件
```

## ✨ 如何使用

### 添加新笔记
1. 直接在对应分类文件夹下创建 `.md` 文件
2. 使用 `[[笔记名称]]` 链接到其他笔记
3. 使用 `#标签` 组织内容

### 整理文档/链接
当你需要我帮你整理文档或链接时，只需告诉我：
- 文档内容/链接
- 想要的分类
- 笔记的标题（可选）

我会自动：
1. 提取关键信息
2. 转换为 Markdown 格式
3. 创建/更新笔记文件
4. 建立相关链接

### 搜索笔记
- 在 Obsidian 中使用 `Ctrl/Cmd + P` 快速搜索笔记
- 使用标签过滤特定主题的笔记
- 查看图谱视图了解笔记关联

## 💡 建议的使用方式

1. **定期整理**: 每周或每月整理一次笔记
2. **标签系统**: 建立统一的标签体系
3. **建立连接**: 使用双链将相关笔记连接起来
4. **保持简洁**: 每个笔记专注一个主题

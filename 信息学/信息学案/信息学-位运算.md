# 信息学专题：位运算

> 信息学 | 基础技巧 | 二进制思维

---

## 本课学什么

位运算的价值不在于“写起来很酷”，而在于它能直接操作二进制位，常用于状态压缩、快速判断、集合表示与低层优化。

本课重点掌握：

- 常见位运算符的含义；
- 常用技巧为什么成立；
- 什么时候该用位运算，什么时候不必硬用。

---

## 核心知识 / 核心方法

### 1. 六个常见操作

| 操作 | 写法 | 含义 |
| --- | --- | --- |
| 与 | `a & b` | 两位都为 1 才为 1 |
| 或 | `a | b` | 有一位为 1 就为 1 |
| 异或 | `a ^ b` | 不同为 1 |
| 取反 | `~a` | 每一位取反 |
| 左移 | `a << k` | 整体左移 `k` 位 |
| 右移 | `a >> k` | 整体右移 `k` 位 |

### 2. 常见技巧为什么成立

#### 判断奇偶

```cpp
if (x & 1) {
    // 奇数
}
```

因为最低位是 `1` 表示奇数，最低位是 `0` 表示偶数。

#### 取出第 k 位

```cpp
int bit = (x >> k) & 1;
```

#### 设置第 k 位为 1

```cpp
x |= (1 << k);
```

#### 清除第 k 位

```cpp
x &= ~(1 << k);
```

#### 统计最低位 1

```cpp
int lowbit = x & (-x);
```

它能取出二进制表示中最右边那个 `1` 对应的值。

#### 判断是否为 2 的幂

```cpp
x > 0 && (x & (x - 1)) == 0
```

因为 2 的幂二进制中只有一个 `1`。

---

## 易错点 / 关键提醒

1. **忘记加括号**。如 `x & 1 == 0` 实际优先级不是你想的那样。
2. **把左移机械理解成乘法**。对负数和溢出场景要谨慎。
3. **对符号位理解不清**，尤其是 `~x` 和右移负数。
4. **为了炫技滥用位运算**，代码反而更难读。

## 例题

### 例1：基础例题——统计二进制中 1 的个数

```cpp
int cnt = 0;
while (x) {
    x &= (x - 1);
    ++cnt;
}
```

每执行一次，会删掉最低位的一个 `1`。

### 例2：应用例题——用二进制表示集合

若 `mask` 的第 `k` 位为 `1`，表示元素 `k` 在集合中。

这样可以高效表示子集、状态压缩 DP 等问题。

## 练习题

1. 写出判断一个数是否为奇数的位运算写法。
2. 为什么 `x & (x - 1)` 能去掉最低位的 `1`？
3. 用位运算表示集合时，第 `k` 位为 `1` 代表什么？

## 答案

1. `x & 1`。
2. 因为 `x - 1` 会把最低位的 `1` 变成 `0`，其右侧全变成 `1`，再与原数按位与就删掉了这个 `1`。
3. 表示编号为 `k` 的元素被选入集合。

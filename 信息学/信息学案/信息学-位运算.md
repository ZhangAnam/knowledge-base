## 📌 位运算基础

### 按位操作符
| 操作 | 符号 | 说明 |
|------|------|------|
| 与 | a & b | 都是1才为1 |
| 或 | a \| b | 有1则为1 |
| 异或 | a ^ b | 不同为1 |
| 取反 | ~a | 0变1,1变0 |
| 左移 | a << b | a × 2^b |
| 右移 | a >> b | a ÷ 2^b |

---

## 📌 常用技巧

### 判断奇偶
```cpp
x & 1  // 1为奇数，0为偶数
```

### 交换两数
```cpp
a ^= b; b ^= a; a ^= b;
```

### 取最低位1
```cpp
x & (-x)
```

### 判断是否为2的幂
```cpp
x & (x-1) == 0
```

---

## 📌 位运算优先级
- ~ > << >> > & > ^ > |
- 加括号最安全

---

## 📝 典型例题

### 例1：二进制中1的个数
求x的二进制表示中1的个数。

> 📋 答案
```cpp
int cnt = 0;
while (x) {
    x &= (x-1);
    cnt++;
}
```

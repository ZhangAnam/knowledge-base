## 📌 栈的概念

### 特性
后进先出（LIFO）

### 操作
- push: 入栈
- pop: 出栈
- top: 查看栈顶
- empty: 判空

---

## 📌 C++实现

### 使用STL
```cpp
#include <stack>
stack<int> s;
s.push(x);
s.pop();
s.top();
s.empty();
```

## 📝 典型例题

### 例1：括号匹配
判断括号序列是否合法。

> 📋 答案
```cpp
stack<char> s;
for (char c : str) {
    if (c == '(' || c == '[' || c == '{') s.push(c);
    else {
        if (s.empty()) return false;
        if (!match(s.top(), c)) return false;
        s.pop();
    }
}
return s.empty();
```

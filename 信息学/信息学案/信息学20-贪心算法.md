## 📌 经典例题

### 活动选择
```cpp
// 按结束时间排序
sort(activities, key=end);
int cnt = 0, last_end = 0;
for (auto [s, e] : activities) {
    if (s >= last_end) {
        cnt++;
        last_end = e;
    }
}
```

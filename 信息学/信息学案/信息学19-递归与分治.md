## 📌 递归

### 要素
1. 终止条件
2. 递归公式

### 示例：斐波那契
```cpp
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
```

### 思想

## 📌 优化

### 记忆化
```cpp
int dp[100];
int fib(int n) {
    if (dp[n] != -1) return dp[n];
    return dp[n] = fib(n-1) + fib(n-2);
}
```

## 📝 典型例题

### 例1：归并排序

> 📋 答案
```cpp
void merge_sort(int l, int r) {
    if (l >= r) return;
    int mid = (l + r) / 2;
    merge_sort(l, mid);
    merge_sort(mid+1, r);
    merge(l, mid, r);
}
```

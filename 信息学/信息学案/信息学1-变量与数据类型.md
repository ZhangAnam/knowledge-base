## 📌 核心概念

### 变量
- **定义**：存储数据的容器
- **命名规则**：字母、数字、下划线，不能以数字开头
- **使用**：先声明后使用

### 数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 整型 int | 整数 | 1, 100, -5 |
| 浮点型 float/double | 小数 | 3.14, -0.5 |
| 字符型 char | 单字符 | 'a', 'A' |
| 字符串 string | 字符序列 | "Hello" |
| 布尔型 bool | 真/假 | true, false |

## 📝 典型例题

### 例1：交换两个数
输入两个整数，交换后输出。

> 📋 答案
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    swap(a, b);
    cout << a << " " << b << endl;
    return 0;
}
```

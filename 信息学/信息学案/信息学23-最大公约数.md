## 📌 欧几里得算法

### 辗转相除
```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

### 复杂度
O(log min(a,b))

---

## 📌 扩展欧几里得

### 用途
求ax + by = gcd(a,b)的解

```cpp
int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    int x1, y1;
    int g = exgcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return g;
}
```

# 信息学23：最大公约数

> 信息学｜数论基础

## 本课学什么

最大公约数常出现在约分、整除关系和“尽量大且都能整除”的题里。本课重点是理解欧几里得算法为什么成立，以及 GCD 和 LCM 如何配合使用。

## 学习目标

- 理解 GCD 和 LCM 的意义。
- 掌握欧几里得算法及其递归边界。
- 能识别约分、互质、公共因子等常见应用。

## 概念

### 1. 最大公约数
两个整数的最大公约数是同时整除它们的最大正整数。

### 2. 欧几里得算法
gcd(a,b)=gcd(b,a mod b) 是这类问题的核心转化，它能让数字迅速变小。

### 3. 最小公倍数
LCM 可由 GCD 推出：lcm(a,b)=a/gcd(a,b)*b。写法顺序要注意减少溢出风险。

## 算法思想 / 分析思路

- 题目出现“尽量大且都能整除”时，要想到 GCD。
- 若要求多个数的公共因子，可把 GCD 逐个递推下去。
- 求 LCM 时要先除后乘，避免中间结果过大。

## 实现步骤

1. 明确题目是求 GCD、LCM，还是借它解决约分和互质判断。
2. 使用欧几里得算法不断把问题化成更小的数。
3. 若有多个数，就把答案与下一个数继续求 gcd。
4. 若要求 lcm，再通过 GCD 公式计算。

## 常见错误

1. **把 GCD 和 LCM 混淆**：一个求公共因子最大，一个求公共倍数最小。
2. **忘记 gcd(a,0)=a**：这是递归的重要边界。
3. **求 LCM 时直接先乘再除**：大数据时容易溢出。
4. **多个数的 GCD 只看最小值**：必须逐个迭代计算。

## 例题

### 例1：约分分数 24/36
**层次**：基础例题

**题目**：把 24/36 约成最简分数。

**分析**：先求 gcd(24,36)=12，再同时除以 12，得到最简形式。

**答案/结论**：结果为 2/3。

### 例2：若干木棒剪成等长且尽量长
**层次**：关键例题

**题目**：长度分别为 12、18、24 的木棒，要剪成长度相同且尽量长的小段。

**分析**：“尽量长且都能整除”正是在找这几个数的最大公约数。

**答案/结论**：公共长度应为 6。

### 例3：求最小公倍数时的溢出问题
**层次**：易错例题

**题目**：为什么 lcm(a,b) 更推荐写成 a/gcd(a,b)*b？

**分析**：若先算 a*b，数据较大时可能先溢出，再去除以 gcd 就已经来不及。

**答案/结论**：先除后乘更安全。

## 练习

1. gcd(48,18) 的结果是多少？
2. 为什么 gcd(a,b) 可以转化成 gcd(b,a mod b)？
3. 三个数 30、45、75 的最大公约数是多少？
4. 若两个数互质，意味着什么？

## 答案

1. 结果是 6。
2. 因为两组数的公约数集合相同。
3. 先算 gcd(30,45)=15，再和 75 求 gcd，结果仍是 15。
4. 意味着它们的最大公约数是 1。

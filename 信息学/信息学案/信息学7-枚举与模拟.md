## 📝 典型例题

### 例1：百钱买百鸡
公鸡5元，母鸡3元，小鸡1元3只，100元买100只鸡。

> 📋 答案
```cpp
for (int g = 0; g <= 100; g++) {
    for (int m = 0; m <= 100; m++) {
        int x = 100 - g - m;
        if (x >= 0 && 5*g + 3*m + x/3 == 100 && x % 3 == 0) {
            cout << g << " " << m << " " << x << endl;
        }
    }
}
```

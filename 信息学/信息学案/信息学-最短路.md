## 📌 Dijkstra算法

### 适用场景
单源最短路，边权非负

### 代码实现
```cpp
void dijkstra(int s) {
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> pq;
    dist[s] = 0;
    pq.push({0, s});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;
        for (auto [v, w] : adj[u]) {
            if (dist[v] > dist[u] + w) {
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
}
```

### 时间复杂度
- 朴素：O(V²)
- 堆优化：O((V+E)logV)

## 📌 Floyd算法

### 适用场景
多源最短路，任意两点间最短路

### 代码实现
```cpp
for (int k = 1; k <= n; k++)
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            dis[i][j] = min(dis[i][j], dis[i][k] + dis[k][j]);
```

### 时间复杂度
O(V³)

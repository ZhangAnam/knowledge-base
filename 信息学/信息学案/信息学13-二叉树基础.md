## 📌 二叉树概念

### 定义
每个节点最多有两个子树的树

### 特殊类型
- 满二叉树
- 完全二叉树
- 二叉搜索树

---

## 📌 节点结构

```cpp
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};
```

---

## 📌 性质

1. 第i层最多2^(i-1)个节点
2. 深度为k最多2^k-1个节点
3. n0 = n2 + 1

---

## 📝 典型例题

### 例1：计算二叉树深度

> 📋 答案
```cpp
int depth(TreeNode* root) {
    if (!root) return 0;
    return max(depth(root->left), depth(root->right)) + 1;
}
```

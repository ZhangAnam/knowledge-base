## 📌 BFS概念

### 思想
按层次遍历，先访问距离近的点

### 队列实现
```cpp
queue<int> q;
q.push(start);
while (!q.empty()) {
    int u = q.front(); q.pop();
    for (int v : adj[u]) {
        if (!vis[v]) {
            vis[v] = true;
            q.push(v);
        }
    }
}
```

---

## 📌 与DFS对比

| DFS | BFS |
|-----|-----|
| 递归/栈 | 队列 |
| 深度优先 | 层次优先 |
| 适合路径 | 最短路径 |

---

## 📌 应用

1. 最短路（无权图）
2. 层次遍历
3. 扩散问题

## 📝 典型例题

### 例1：迷宫最短路径

> 📋 答案
```cpp
struct Node {int x, y, d;};
queue<Node> q;
q.push({sx, sy, 0});
while (!q.empty()) {
    Node cur = q.front(); q.pop();
    for (int dir = 0; dir < 4; dir++) {
        int nx = cur.x + dx[dir];
        int ny = cur.y + dy[dir];
        if (valid(nx, ny)) {
            q.push({nx, ny, cur.d+1});
        }
    }
}
```

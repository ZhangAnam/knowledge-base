## 本课学什么

代码片段不是背下来就能直接套的“万能答案”，而是帮助你快速搭出正确框架的工具。  
本页重点学习几类高频模板：**基础算法、数据结构、图论、动态规划与常见工具函数**。

学完后，你应该能够：

- 知道某段模板适合解决什么问题；
- 明白模板中哪些变量需要按题目改写；
- 避开“只会复制，不会检查边界和复杂度”的常见问题。

---

## 核心方法：怎样正确使用代码片段

使用模板前先问自己三件事：

1. **这段代码解决的核心问题是什么？**
2. **它依赖哪些全局变量、下标习惯、边界条件？**
3. **当前题目规模是否允许这段代码的复杂度？**

建议顺序：

```text
先判断模型 → 再选模板 → 再按题目改变量和边界 → 最后自测极端情况
```

## 核心代码片段

### 1. 快速幂

适用场景：求 `a^b`，尤其是取模幂运算。

```cpp
long long qpow(long long a, long long b, long long mod) {
    long long res = 1 % mod;
    a %= mod;
    while (b > 0) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return res;
}
```

关键理解：

- 本质是把指数按二进制拆开；
- 时间复杂度从 `O(b)` 降到 `O(log b)`。

### 2. 归并排序

适用场景：排序、统计逆序对、分治类问题。

```cpp
void merge_sort(vector<int>& a, int l, int r, vector<int>& tmp) {
    if (l >= r) return;
    int mid = (l + r) >> 1;
    merge_sort(a, l, mid, tmp);
    merge_sort(a, mid + 1, r, tmp);

    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) tmp[k++] = a[i++];
        else tmp[k++] = a[j++];
    }
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (int p = l; p <= r; ++p) a[p] = tmp[p];
}
```

关键提醒：

- `tmp` 数组通常要提前开好；
- 区间边界是最容易写错的地方。

### 3. 快速排序

适用场景：一般排序；但在竞赛中很多时候直接 `sort` 更稳妥。

```cpp
int partition(vector<int>& a, int l, int r) {
    int pivot = a[r], i = l - 1;
    for (int j = l; j < r; ++j) {
        if (a[j] <= pivot) swap(a[++i], a[j]);
    }
    swap(a[++i], a[r]);
    return i;
}

void quick_sort(vector<int>& a, int l, int r) {
    if (l >= r) return;
    int p = partition(a, l, r);
    quick_sort(a, l, p - 1);
    quick_sort(a, p + 1, r);
}
```

### 4. 单调栈

适用场景：找左边/右边第一个更大（更小）元素、区间贡献问题。

```cpp
stack<int> st;
for (int i = 0; i < n; ++i) {
    while (!st.empty() && a[st.top()] < a[i]) st.pop();
    st.push(i);
}
```

关键理解：

- 栈里维护的是“仍然可能对后面元素有用”的下标；
- 比较符号不同，含义就不同。

### 5. 离散化

适用场景：数值很大，但只关心相对大小关系。

```cpp
vector<int> b = a;
sort(b.begin(), b.end());
b.erase(unique(b.begin(), b.end()), b.end());
for (int& x : a) {
    x = lower_bound(b.begin(), b.end(), x) - b.begin();
}
```

关键提醒：

- 离散化后得到的是“排名”或“下标”，不是原值；
- 如果题目要恢复原值，必须保留原数组或映射表。

### 6. 差分数组

适用场景：多次区间加减、最后统一还原。

```cpp
diff[l] += val;
diff[r + 1] -= val;

for (int i = 1; i <= n; ++i) {
    diff[i] += diff[i - 1];
}
```

关键提醒：

- 题目若是 1-based 下标，这段写法很自然；
- 要确认 `r + 1` 没有越界。

### 7. DFS遍历

适用场景：树、图的搜索、连通块、回溯。

```cpp
void dfs(int u, int fa) {
    for (int v : g[u]) {
        if (v == fa) continue;
        dfs(v, u);
    }
}
```

关键提醒：

- 在一般图中往往还需要 `vis` 数组；
- 树和图的DFS模板不能机械混用。

### 8. BFS

适用场景：无权图最短路、最少步数问题、分层遍历。

```cpp
queue<int> q;
dist[start] = 0;
q.push(start);
while (!q.empty()) {
    int u = q.front();
    q.pop();
    for (int v : g[u]) {
        if (dist[v] == -1) {
            dist[v] = dist[u] + 1;
            q.push(v);
        }
    }
}
```

关键提醒：

- **无权图最短路** 用 BFS；
- 有边权时不能直接套这个模板。

### 9. 01背包

适用场景：每件物品最多选一次。

```cpp
for (int i = 1; i <= n; ++i) {
    for (int j = W; j >= w[i]; --j) {
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```

关键提醒：

- `j` 必须倒序；
- 若改成正序，就变成了“完全背包”的转移风格。

### 10. 最长公共子序列（LCS）

```cpp
for (int i = 1; i <= n; ++i) {
    for (int j = 1; j <= m; ++j) {
        if (a[i] == b[j]) dp[i][j] = dp[i - 1][j - 1] + 1;
        else dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
    }
}
```

### 11. 常用工具函数

#### 最大公约数

```cpp
int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a;
}
```

#### 判断素数

```cpp
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; ++i) {
        if (n % i == 0) return false;
    }
    return true;
}
```

#### 全排列

```cpp
sort(a.begin(), a.end());
do {
    // 处理当前排列
} while (next_permutation(a.begin(), a.end()));
```

#### 二分查找模板

```cpp
int l = 0, r = n - 1;
while (l < r) {
    int mid = (l + r) >> 1;
    if (check(mid)) r = mid;
    else l = mid + 1;
}
return l;
```

## 易错点 / 关键提醒

1. **模板里的数组范围、下标起点不一定适合当前题目**。  
2. **BFS模板不能直接处理带权最短路**。  
3. **01背包的内层循环方向常考**。  
4. **快速幂若涉及取模，要从一开始就全程取模**。  
5. **离散化后得到的是新编号，不是原始数值**。

## 例题

### 例1：区间加法应该优先想到什么模板

如果有很多次“把区间 `[l, r]` 全部加上 `val`”的操作，最后再统一输出结果，优先想到什么？

#### 解析

这种题不是要求每次都真的把整段逐个修改，而是把影响先记下来，最后一次还原。  
因此优先使用差分数组。

#### 答案

**差分数组模板**

### 例2：无权图最短路

如果边权都相同或都视作 1，应该优先选择 DFS 还是 BFS？

#### 解析

BFS按层扩展，第一次到达某点时路径最短，更适合无权最短路。

#### 答案

**BFS**

### 例3：二分模板为什么要配合 `check(mid)`

#### 解析

二分不是“凭感觉猜答案”，而是利用“答案区间具有单调性”。  
`check(mid)` 的作用就是判断当前值是否满足条件，从而缩小区间。

#### 答案

**因为二分本质上是在单调性条件下不断排除不可能区间。**

4. 无权图最短路为什么通常使用 BFS？  
5. 离散化后如果需要恢复原值，应该提前保留什么信息？

---

## 答案

1. **因为它把指数按二进制拆分，复杂度降为 `O(log b)`**。  
2. **差分数组**。  
3. **为了保证每件物品只被使用一次**。  
4. **因为 BFS 按层扩展，第一次到达即为最短步数**。  
5. **原数组或离散值映射表**。

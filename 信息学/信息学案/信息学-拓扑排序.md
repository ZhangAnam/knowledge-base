# 考点7：拓扑排序

> 信息学 | 图论 | NOI大纲

## 📌 核心概念

### 拓扑序列
有向无环图(DAG)中，所有顶点排成的线性序列

### 判定是否有环
拓扑排序无法完成则存在环

---

## 📌 Kahn算法（入度法）

### 思想
每次选择入度为0的顶点加入序列

### 代码实现
```cpp
queue<int> q;
vector<int> indeg(n+1);
// 计算入度
for (int u = 1; u <= n; u++)
    for (int v : adj[u])
        indeg[v]++;

for (int i = 1; i <= n; i++)
    if (indeg[i] == 0) q.push(i);

while (!q.empty()) {
    int u = q.front(); q.pop();
    ans.push_back(u);
    for (int v : adj[u]) {
        indeg[v]--;
        if (indeg[v] == 0) q.push(v);
    }
}
```

### 时间复杂度
O(V+E)

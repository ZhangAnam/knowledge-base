---

## 📌 寻址方式

### 立即寻址
```asm
MOV AX, 5      # 数据在指令中
```

### 寄存器寻址
```asm
MOV AX, BX     # 数据在寄存器中
```

### 直接寻址
```asm
MOV AX, [1000H] # 地址在指令中
```

### 寄存器间接
```asm
MOV AX, [BX]    # 地址在寄存器中
```

### 相对寻址
```asm
JMP LABEL     # PC+偏移量
```

## 📌 CISC vs RISC

| 特点 | CISC | RISC |
|------|------|------|
| 指令数 | 多 | 少 |
| 指令长度 | 变长 | 定长 |
| 寻址方式 | 复杂 | 简单 |
| 寄存器 | 少 | 多 |
| 例子 | x86 | ARM |

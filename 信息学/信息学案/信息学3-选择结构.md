## 📌 if 语句

### 单分支
```cpp
if (条件) {
    // 语句
}
```

### 双分支
```cpp
if (条件) {
    // 成立执行
} else {
    // 不成立执行
}
```

### 多分支
```cpp
if (条件1) {
    
} else if (条件2) {
    
} else {
    
}
```

## 📌 switch 语句

### 语法
```cpp
switch (表达式) {
    case 常量1: 
        break;
    case 常量2: 
        break;
    default: 
        break;
}
```

## 📝 典型例题

### 例1：成绩等级
输入成绩，输出等级（90+ A, 80+ B, 70+ C, 60+ D, <60 E）。

> 📋 答案
```cpp
int score;
cin >> score;
if (score >= 90) cout << "A";
else if (score >= 80) cout << "B";
else if (score >= 70) cout << "C";
else if (score >= 60) cout << "D";
else cout << "E";
```

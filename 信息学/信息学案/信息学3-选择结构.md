# 信息学3：选择结构

> 信息学｜程序控制基础

## 本课学什么

当题目出现“如果……就……”时，就进入了选择结构。重点不是会写 if，而是会把分类讨论写得完整、不重叠、不错边界。

## 学习目标

- 理解 if、if...else、else if 的基本作用。
- 能把题目中的分类讨论翻译成程序分支。
- 会检查条件范围是否完整、是否有重叠。

## 概念

### 1. 选择结构
选择结构的本质是“根据条件决定执行哪一段代码”。它对应题目中的情况判断和规则分段。

### 2. 单分支与双分支
只有满足条件才执行的是单分支；两种情况都要处理时通常使用 if...else。

### 3. 多分支
当情况超过两种时，常用 else if 串联，让判断从上到下逐步筛选。

## 算法思想 / 分析思路

- 先把题目中的所有情况列出来，再写代码。
- 范围题一定要先想边界属于哪一类。
- 多分支判断按从大到小或从小到大写，通常更清楚。

## 实现步骤

1. 列出所有可能情况。
2. 为每种情况写出明确条件。
3. 按顺序组织 if、else if、else。
4. 用边界值测试结果是否正确。

## 常见错误

1. **条件重叠**：前面的分支已经覆盖了后面的情况。
2. **边界遗漏**：只写 x > 60 和 x < 60，却忘了 x == 60。
3. **把 = 写成 ==**：赋值和比较混淆会直接改变程序逻辑。
4. **分支顺序混乱**：顺序不合理时，容易先进入错误分支。

## 例题

### 例1：判断正数、负数或零
**层次**：基础例题

**题目**：输入整数 x，判断它是正数、负数还是 0。

**分析**：这是典型三分类题，条件要互不重叠并覆盖全部情况。

**答案/结论**：可按 x>0、x<0、其他情况依次判断。

### 例2：成绩等级划分
**层次**：关键例题

**题目**：输入分数 score，按分段输出等级。

**分析**：多分支范围题建议从高到低判断，更容易控制边界。

**关键步骤**：
- 先判断 score >= 90。
- 再判断 score >= 80、score >= 60。
- 最后用 else 处理剩余情况。

**答案/结论**：范围题的关键是边界清楚、顺序清楚。

### 例3：闰年判断
**层次**：易错例题

**题目**：输入年份 y，判断是否为闰年。

**分析**：闰年规则涉及复合条件，需要逻辑运算配合。

**答案/结论**：复杂条件要拆开理解，再组合成 if 条件。

## 练习

1. 输入一个整数，判断奇偶。
2. 输入三条边长，判断能否构成三角形。
3. 输入两个数，输出较大值。
4. 输入月份，判断该月属于哪个季度。

## 答案

1. 偶数条件可写成 n % 2 == 0。
2. 三角形需满足任意两边之和大于第三边。
3. 可用 if 比较 a 和 b。
4. 按 1-3、4-6、7-9、10-12 分类即可。

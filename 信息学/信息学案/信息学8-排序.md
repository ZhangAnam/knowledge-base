## 📌 排序算法

### 冒泡排序
```cpp
for (int i = 0; i < n-1; i++) {
    for (int j = 0; j < n-i-1; j++) {
        if (a[j] > a[j+1]) swap(a[j], a[j+1]);
    }
}
```
时间复杂度：O(n²)

### 选择排序
```cpp
for (int i = 0; i < n; i++) {
    int k = i;
    for (int j = i+1; j < n; j++) {
        if (a[j] < a[k]) k = j;
    }
    swap(a[i], a[k]);
}
```

## 📌 C++排序

### sort
```cpp
#include <algorithm>
sort(a, a+n);        // 升序
sort(a, a+n, greater<int>());  // 降序
```

## 📝 典型例题

### 例1：排序并去重
输入n个数，排序后去重输出。

> 📋 答案
```cpp
sort(a, a+n);
cout << a[0];
for (int i = 1; i < n; i++) {
    if (a[i] != a[i-1]) cout << " " << a[i];
}
```

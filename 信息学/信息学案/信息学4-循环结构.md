# 信息学4：循环结构

> 信息学｜程序控制基础

## 本课学什么

当同一类操作需要重复很多次时，循环结构就比手写重复代码高效得多。本课重点是搞清楚“从哪里开始、何时结束、每轮做什么”。

## 学习目标

- 理解 for 和 while 的使用场景。
- 能把“重复执行”问题写成循环。
- 会检查循环次数、边界和是否会死循环。

## 概念

### 1. 循环
循环是让程序按照某种规律反复执行，适合处理求和、统计、枚举和批量输入。

### 2. for 与 while
已知循环次数时，for 更直观；循环次数不固定但由条件控制时，while 更常见。

### 3. 循环变量
循环变量像计数器，决定当前在第几轮。初值、终止条件和更新方式缺一不可。

## 算法思想 / 分析思路

- 先问自己：这件事要重复多少次，结束信号是什么。
- 很多循环题都能拆成“初始化—重复处理—更新答案”。
- 写完循环后，用很小的数据手动模拟一遍。

## 实现步骤

1. 确定循环目标，如求和、统计或找最大值。
2. 设计循环变量和结束条件。
3. 在循环体中完成每一轮处理。
4. 及时更新统计量或状态。
5. 循环结束后输出结果。

## 常见错误

1. **死循环**：循环变量没有更新，或终止条件永远不成立。
2. **少一轮或多一轮**：初值或 <=、< 的选择错误。
3. **初始化位置错误**：求和、最大值等变量初始化不当会造成结果偏差。
4. **读入语句放错位置**：需要每轮读一个数时，读入必须在循环内部。

## 例题

### 例1：求 1 到 n 的和
**层次**：基础例题

**题目**：输入 n，输出 1+2+...+n。

**分析**：这是最典型的累加模型：答案变量从 0 开始，每轮加上当前 i。

**答案/结论**：学会“初始化 + 循环累加”的写法。

### 例2：统计 n 个数中有多少个偶数
**层次**：关键例题

**题目**：先输入 n，再输入 n 个整数，输出其中偶数个数。

**分析**：这是“重复读入 + 条件统计”的组合题。

**答案/结论**：每读到一个数，就判断并更新计数器。

### 例3：倒序拆位
**层次**：易错例题

**题目**：输入一个正整数，依次输出它的每一位（从低位到高位）。

**分析**：可以反复使用 n % 10 取末位，再用 n /= 10 删除末位。

**答案/结论**：循环常与取模、整除配合。

## 练习

1. 计算 1 到 100 的偶数和。
2. 输入 n 个数，求最大值。
3. 输入一个整数，统计它的十进制位数。
4. 输入若干个数，以 0 结束，求它们的总和。

## 答案

1. 可以只累加偶数，或让循环变量每次加 2。
2. 先把第一个数作为初始最大值，再逐个比较。
3. 可用 while(n>0) 不断 n/=10 计数。
4. 这是按结束标志控制的 while 模型。

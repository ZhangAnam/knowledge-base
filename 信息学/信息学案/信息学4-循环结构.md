# 信息学4：循环结构

> 信息学 | 基础编程

---

## 📌 for 循环

### 语法
```cpp
for (初始化; 条件; 更新) {
    // 循环体
}
```

### 常见形式
```cpp
for (int i = 0; i < n; i++) {
    // 执行n次
}
```

---

## 📌 while 循环

### 语法
```cpp
while (条件) {
    // 循环体
}
```

### do-while
```cpp
do {
    // 至少执行一次
} while (条件);
```

---

## 📌 循环控制

### break
跳出整个循环

### continue
跳过本次循环

---

## 📝 典型例题

### 例1：求和
计算1+2+...+100。

> 📋 答案
```cpp
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i;
}
cout << sum;
```

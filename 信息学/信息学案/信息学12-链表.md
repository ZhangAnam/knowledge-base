## 📌 链表概念

### 节点结构
```cpp
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};
```

### 特点
- 动态大小
- 插入删除O(1)
- 查找O(n)

## 📝 典型例题

### 例1：链表反转
反转单链表。

> 📋 答案
```cpp
ListNode* reverse(ListNode* head) {
    ListNode* pre = nullptr;
    while (head) {
        ListNode* nxt = head->next;
        head->next = pre;
        pre = head;
        head = nxt;
    }
    return pre;
}
```

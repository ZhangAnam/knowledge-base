# 信息学11：队列

> 信息学 | 数据结构

---

## 📌 队列概念

### 特性
先进先出（FIFO）

### 操作
- push: 入队
- pop: 出队
- front: 查看队首
- back: 查看队尾
- empty: 判空

---

## 📌 C++实现

### 使用STL
```cpp
#include <queue>
queue<int> q;
q.push(x);
q.pop();
q.front();
q.back();
q.empty();
```

### 双端队列 deque
```cpp
#include <deque>
deque<int> dq;
dq.push_back();
dq.push_front();
```

## 📝 典型例题

### 例1：BFS模板
图的广度优先搜索。

> 📋 答案
```cpp
queue<int> q;
q.push(start);
while (!q.empty()) {
    int u = q.front(); q.pop();
    for (int v : adj[u]) {
        if (!vis[v]) {
            vis[v] = true;
            q.push(v);
        }
    }
}
```

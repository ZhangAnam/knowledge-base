## 📌 DFS概念

### 思想
一条路走到黑，回溯尝试其他路

### 递归实现
```cpp
void dfs(int u) {
    vis[u] = true;
    for (int v : adj[u]) {
        if (!vis[v]) dfs(v);
    }
}
```

## 📝 典型例题

### 例1：求连通块数量

> 📋 答案
```cpp
void dfs(int u) {
    vis[u] = true;
    for (int v : adj[u]) {
        if (!vis[v]) dfs(v);
    }
}
int main() {
    for (int i = 1; i <= n; i++) {
        if (!vis[i]) {
            dfs(i);
            ans++;
        }
    }
}
```

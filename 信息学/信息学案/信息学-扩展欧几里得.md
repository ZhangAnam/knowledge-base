# 考点8：扩展欧几里得

> 信息学 | 数论 | NOI大纲

---

## 📌 核心概念

### 扩展欧几里得
求ax + by = gcd(a,b) 的一组解

### 乘法逆元
ax ≡ 1 (mod p) ⇒ x是a模p的逆元

---

## 📌 算法实现

### 求x,y
```cpp
int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1; y = 0;
        return a;
    }
    int x1, y1;
    int g = exgcd(b, a%b, x1, y1);
    x = y1;
    y = x1 - (a/b) * y1;
    return g;
}
```

### 求逆元
```cpp
int modInverse(int a, int p) {
    int x, y;
    int g = exgcd(a, p, x, y);
    if (g != 1) return -1;  // 无逆元
    return (x % p + p) % p;
}
```

---

## 📌 应用场景

1. 求解同余方程
2. 乘法逆元
3. 线性同余方程组

# 信息学专题：扩展欧几里得

> 信息学 | 数论 | 线性不定方程与逆元

---

## 本课学什么

普通欧几里得算法只告诉我们 `gcd(a, b)` 是多少；扩展欧几里得则进一步求出：

```text
ax + by = gcd(a, b)
```

的一组整数解。

它是处理逆元、线性同余方程、裴蜀定理相关问题的重要工具。

---

## 核心知识 / 核心方法

### 1. 裴蜀定理背景

对任意整数 `a, b`，一定存在整数 `x, y`，使得：

```text
ax + by = gcd(a, b)
```

exgcd 做的就是把这一组 `x, y` 算出来。

### 2. 递归写法

```cpp
long long exgcd(long long a, long long b, long long &x, long long &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    long long x1, y1;
    long long g = exgcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return g;
}
```

### 3. 为什么能递推回来

因为：

```text
a % b = a - (a / b) * b
```

所以下层方程中的系数，可以换算回上层的 `a, b` 形式。

### 4. 求逆元

若 `gcd(a, p) = 1`，则 `a` 在模 `p` 下有逆元。

```cpp
long long inv(long long a, long long p) {
    long long x, y;
    long long g = exgcd(a, p, x, y);
    if (g != 1) return -1;
    return (x % p + p) % p;
}
```

---

## 易错点 / 关键提醒

1. **把 gcd 算对了，但 `x, y` 递推错了**。
2. **逆元存在条件没检查**，只有 `gcd(a, p) = 1` 才有逆元。
3. **答案是负数时不做模调整**。
4. **不会区分 exgcd 与快速幂求逆元的适用前提**。

---

## 例题

### 例1：基础例题——求 `5x + 13y = gcd(5,13)` 的一组解

由于 `gcd(5,13)=1`，exgcd 能求出一组满足 `5x + 13y = 1` 的整数解。

### 例2：应用例题——求 5 在模 13 下的逆元

因为 `5x ≡ 1 (mod 13)`，等价于求 `5x + 13y = 1`。

可得逆元为 `8`，因为 `5 × 8 = 40 ≡ 1 (mod 13)`。

---

## 练习题

1. exgcd 比普通 gcd 多求出了什么？
2. 为什么求逆元可以转化成 `ax + py = 1`？
3. 若 `gcd(a, p) != 1`，为什么逆元不存在？

## 答案

1. 多求出一组满足 `ax + by = gcd(a, b)` 的整数解。
2. 因为 `ax ≡ 1 (mod p)` 等价于 `ax + py = 1`。
3. 因为这时左边所有值都被 `gcd(a, p)` 整除，不可能等于 1。

## 📌 顺序查找

### 代码
```cpp
int find(int a[], int n, int x) {
    for (int i = 0; i < n; i++) {
        if (a[i] == x) return i;
    }
    return -1;
}
```
时间复杂度：O(n)

## 📌 二分查找

### 前提
有序数组

### 代码
```cpp
int binarySearch(int a[], int n, int x) {
    int l = 0, r = n-1;
    while (l <= r) {
        int mid = (l + r) / 2;
        if (a[mid] == x) return mid;
        if (a[mid] < x) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}
```
时间复杂度：O(log n)

## 📌 核心概念

### 字典树
一种树形结构，用于存储和查找字符串

### 节点结构
```cpp
struct Node {
    int next[26];
    int cnt;  // 以此节点为结尾的单词数
    int num;   // 经过此节点的前缀数
};
```

---

## 📌 基本操作

### 插入
```cpp
void insert(string s) {
    int p = 0;
    for (char c : s) {
        int idx = c - 'a';
        if (!trie[p].next[idx]) 
            trie[p].next[idx] = ++tot;
        p = trie[p].next[idx];
        trie[p].num++;
    }
    trie[p].cnt++;
}
```

### 查找
```cpp
int find(string s) {
    int p = 0;
    for (char c : s) {
        int idx = c - 'a';
        if (!trie[p].next[idx]) return 0;
        p = trie[p].next[idx];
    }
    return trie[p].cnt;
}
```

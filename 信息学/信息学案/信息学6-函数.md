## 📌 函数定义

### 语法
```cpp
返回类型 函数名(参数列表) {
    // 函数体
    return 值;
}
```

### 示例
```cpp
int max(int a, int b) {
    return a > b ? a : b;
}
```

## 📝 典型例题

### 例1：阶乘
用递归求n!。

> 📋 答案
```cpp
int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n-1);
}
```

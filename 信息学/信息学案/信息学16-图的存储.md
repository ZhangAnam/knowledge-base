# 信息学16：图的存储

> 信息学 | 数据结构

---

## 📌 邻接矩阵

### 定义
```cpp
int g[100][100];  // g[i][j] = 1 表示i到j有边
```
- 优点：O(1)查询
- 缺点：O(n²)空间

---

## 📌 邻接表

### vector实现
```cpp
vector<int> adj[100];
// 添加边
adj[u].push_back(v);
```
- 优点：O(n+m)空间
- 缺点：查询需遍历

---

## 📌 链式前向星

### 实现
```cpp
struct Edge {int to, next;} e[1000];
```
适用于稠密图优化

---

## 📝 典型例题

### 例1：图的遍历
用邻接表存储并遍历图。

> 📋 答案
```cpp
vector<vector<int>> adj(n);
for (auto [u,v] : edges) {
    adj[u].push_back(v);
    adj[v].push_back(u);
}
```

## 📌 进制转换

### 常见进制
- 二进制(Binary)：0,1
- 八进制(Octal)：0-7
- 十进制(Decimal)：0-9
- 十六进制(Hex)：0-9,A-F

### 进制转换
```cpp
// 十进制转其他进制
string toBase(int n, int base) {
    string s;
    while (n > 0) {
        s = to_string(n % base) + s;
        n /= base;
    }
    return s;
}
```

## 📌 有符号数表示

### 原码
最高位为符号位，0正1负

### 反码
- 正数：同原码
- 负数：符号位外各位取反

### 补码
- 正数：同原码
- 负数：取反加1

### 补码优势
1. 0唯一表示
2. 便于加减运算统一
3. 便于溢出判断

# 考点4：并查集

> 信息学 | 数据结构 | NOI大纲

---

## 📌 核心概念

### 定义
一种树形数据结构，支持合并和查找操作

### 基本思想
每个元素属于一个集合，通过父指针指向代表元素

---

## 📌 基本操作

### 初始化
```cpp
int fa[MAXN];
void init(int n) {
    for (int i = 1; i <= n; i++) fa[i] = i;
}
```

### 查找（路径压缩）
```cpp
int find(int x) {
    if (fa[x] == x) return x;
    return fa[x] = find(fa[x]);
}
```

### 合并
```cpp
void union(int x, int y) {
    x = find(x);
    y = find(y);
    if (x != y) fa[x] = y;
}
```

## 📌 应用场景

1. 朋友关系（连通分量）
2. 电网连通性
3. 食物链种类关系

## 📝 典型例题

### 例1：朋友圈数量
n个人，m对朋友关系，求朋友圈数量。

> 📋 答案
> 初始化n个集合，合并m对关系，统计fa[i]==i的数量

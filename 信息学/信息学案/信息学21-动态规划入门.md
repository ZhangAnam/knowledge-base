## 📌 DP概念

### 思想
分阶段决策，求最优解

### 三要素
1. 状态
2. 转移
3. 边界

---

## 📌 经典例题

### 斐波那契
```cpp
dp[0] = 0, dp[1] = 1;
for (int i = 2; i <= n; i++) {
    dp[i] = dp[i-1] + dp[i-2];
}
```

### 爬楼梯
```cpp
dp[i] = dp[i-1] + dp[i-2];
```

---

## 📌 解题步骤

1. 确定状态
2. 找转移方程
3. 确定边界
4. 计算顺序

---

## 📝 典型例题

### 例1：01背包

> 📋 答案
```cpp
for (int i = 1; i <= n; i++) {
    for (int j = W; j >= w[i]; j--) {
        dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
    }
}
```

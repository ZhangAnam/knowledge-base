## 📌 一维数组

```cpp
int a[100];      // 大小固定
vector<int> a;  // 动态数组
```

## 📌 二维数组

### 定义
```cpp
int a[100][100];

```

## 📌 字符串

### C风格
```cpp
char s[100];
scanf("%s", s);
```

### C++风格
```cpp
string s;
cin >> s;
```

### 常用操作
- s.length() 长度
- s.substr() 子串
- s.find() 查找

## 📝 典型例题

### 例1：逆序输出
输入n个数，逆序输出。

> 📋 答案
```cpp
int n;
cin >> n;
vector<int> a(n);
for (int i = 0; i < n; i++) cin >> a[i];
for (int i = n-1; i >= 0; i--) cout << a[i] << " ";
```

# 信息学15：堆与优先队列

> 信息学 | 数据结构

---

## 📌 堆概念

### 大根堆
父节点大于等于子节点

### 小根堆
父节点小于等于子节点

### 性质
完全二叉树，堆顶是最大/最小值

---

## 📌 优先队列

### C++实现
```cpp
#include <queue>
priority_queue<int> pq;        // 大根堆
priority_queue<int, vector<int>, greater<int>> pq;  // 小根堆
```

### 操作
- push: 入队 O(log n)
- pop: 出队 O(log n)
- top: 查看堆顶 O(1)

---

## 📌 应用

1. TOP-K问题
2. 哈夫曼编码
3. 合并有序数组

---

## 📝 典型例题

### 例1：TOP-K
找出前k大的数。

> 📋 答案
```cpp
priority_queue<int, vector<int>, greater<int>> pq;
for (int x : a) {
    pq.push(x);
    if (pq.size() > k) pq.pop();
}
```

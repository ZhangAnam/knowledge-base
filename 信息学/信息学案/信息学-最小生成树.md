# 考点6：最小生成树

> 信息学 | 图论 | NOI大纲

## 📌 核心概念

### 最小生成树(MST)
连接所有顶点，边权之和最小的生成树

---

## 📌 Kruskal算法

### 思想
按边权从小到大贪心选择，利用并查集判断连通性

### 代码实现
```cpp
struct Edge {int u, v, w;};
sort(edges.begin(), edges.end(), cmp);
for (auto e : edges) {
    if (find(e.u) != find(e.v)) {
        union(e.u, e.v);
        ans += e.w;
        cnt++;
    }
}
```

### 时间复杂度
O(E log E)

---

## 📌 Prim算法

### 思想
从任意点出发，每次选择权值最小的边扩展

### 代码实现
```cpp
vector<int> dist(n+1, INF);
dist[1] = 0;
priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
pq.push({0, 1});
while (!pq.empty()) {
    auto [d, u] = pq.top(); pq.pop();
    if (d > dist[u]) continue;
    for (auto [v, w] : adj[u]) {
        if (w < dist[v]) {
            dist[v] = w;
            pq.push({dist[v], v});
        }
    }
}
```

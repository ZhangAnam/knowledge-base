## 📌 算术运算符

### 基本运算
| 运算符 | 说明 | 示例 |
|--------|------|------|
| + | 加 | a + b |
| - | 减 | a - b |
| * | 乘 | a * b |
| / | 除 | a / b |

### 优先级

## 📝 典型例题

### 例1：判断奇偶
输入一个整数，判断奇偶。

> 📋 答案
```cpp
int n;
cin >> n;
if (n % 2 == 0) cout << "偶数";
else cout << "奇数";
```

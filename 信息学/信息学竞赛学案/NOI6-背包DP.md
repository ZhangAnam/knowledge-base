# NOI学案 6：背包DP

> 信息学竞赛 | 动态规划

---

## 📌 核心知识点

### 0/1背包
- 每件物品选或不选
- f[i][j] = max(f[i-1][j], f[i-1][j-w]+v)

### 完全背包
- 每件无限件
- 循环顺序优化

### 多重背包
- 件数限制
- 二进制拆分优化

### 组合背包
- 分组选一个

---

## 📝 典型例题

### 01背包模板
```cpp
// 一维优化
vector<int> knapsack01(int W, vector<int>& w, vector<int>& v) {
    vector<int> f(W+1, 0);
    for (int i = 0; i < n; i++) {
        for (int j = W; j >= w[i]; j--) {
            f[j] = max(f[j], f[j-w[i]] + v[i]);
        }
    }
    return f;
}
```

### 完全背包
```cpp
// 正序循环
for (int j = w[i]; j <= W; j++)
    f[j] = max(f[j], f[j-w[i]] + v[i]);
```

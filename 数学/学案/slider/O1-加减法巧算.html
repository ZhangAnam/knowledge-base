🎯 凑整法 — 补整再还差

接近整百整千的数 → 先当整数算 → 再补回差值
例1：加法凑整
99 + 78
= (100 − 1) + 78
= 100 + 78 − 1 = 177
例2：减法凑整
156 − 98
= 156 − 100 + 2
= 58

🎯 凑整法 — 多数中找搭档

例3
997 + 3 + 45
= (997 + 3) + 45
= 1000 + 45 = 1045
例4
364 + 287 + 36 + 13
= (364+36) + (287+13)
= 400 + 300 = 700
先扫一遍，找哪些数字能凑成整百整千

🔗 配对法 — 两两凑整

找出相加恰好得整数的两项，优先配对
1/9 + 2/9 + 3/9 + 4/9 + 5/9 + 6/9 + 7/9 + 8/9
= 1 + 1 + 1 + 1 = 4
拓展：189 − 45 + 55
= 189 + (55 − 45) = 189 + 10 = 199

🚌 带着符号搬家

加减混合时，符号和数字必须一起移动，单独搬数字会算错！
例：500 − 198 − 102
两减数合并：
= 500 − (198 + 102)
= 500 − 300 = 200
例：532 − 97
97 接近 100：
= 532 − 100 + 3
= 432 + 3 = 435

🚌 带符号搬家 — 奥数规律题

999 − 998 + 997 − 996 + ··· + 3 − 2 + 1
= (999−998) + (997−996) + ··· + (3−2) + 1
每对差为 1，从 999 到 3 共 499 对，再加最后 +1
= 499 × 1 + 1 = 500
① 确认每对差值   ② 数共几对   ③ 查有无多余项

📦 去括号规则

➕ 括号前是加号
括号内符号不变
365 − (65 + 48)
= 365 − 65 − 48
= 300 − 48 = 252
➖ 括号前是减号
括号内符号全部取反
456 − (156 − 44)
= 456 − 156 + 44
= 300 + 44 = 344
1000 − (123 + 456) = 1000 − 123 − 456 = 421

⚠️ 三大易错点

① 减法不能随意换位
100 − 36 − 14
写成 100 − (36+14) 才对
不是因为减法有交换律
而是"连减等于减去总和"
② 搬家必须带符号
189 −45 +55
搬 +55 时整体搬：
✅ 189 + 55 − 45
❌ 189 + 55 + 45（丢了负号）
③ 减号括号必须全变号
80 − (25 + 7)
✅ 80 − 25 − 7 = 48
❌ 80 − 25 + 7 = 62
括号前是减，里面全变！

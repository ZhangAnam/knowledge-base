# 学案 62-69：比和比例

> 六年级 | 看懂“两个量之间的关系”，再谈计算和应用

---

## 本课学什么

- 理解比、比值、最简整数比的意义。
- 会判断两个比能否组成比例，并会解比例。
- 认识正比例、反比例的基本特点，会解决按比分配问题。

## 核心理解

### 1. 什么是比
比表示两个量之间的倍比关系。

```text
a : b
```

- 前项：`a`
- 后项：`b`
- 比号：`:`
- 比值：`a ÷ b`

### 2. 比和除法、分数的联系
```text
a : b = a ÷ b = a/b
```

所以化简比，实质上和约分很像；但**比是关系，分数是数，除法是运算**，意义不完全相同。

### 3. 什么是比例
表示两个比相等的式子，叫作比例。

例如：

```text
2 : 3 = 4 : 6
```

### 4. 比例的基本性质
在比例里，两外项之积等于两内项之积。

```text
a : b = c : d
=> a × d = b × c
```

### 5. 正比例和反比例
- **正比例**：一个量增大，另一个量也随着增大，而且比值一定。
- **反比例**：一个量增大，另一个量反而减小，而且乘积一定。

### 6. 按比分配
总量不变时，按几个部分的比来分配，先求总份数，再求每份对应的量。

## 方法与提醒

### 化简比
1. 如果有小数或分数，先化成整数比。
2. 再同时除以前项和后项的最大公因数。

### 解比例
1. 根据比例的基本性质列出乘法等式。
2. 再解方程。

### 判断正比例或反比例
- 看变化方向：同向还是反向。
- 再看“比值是否一定”或“乘积是否一定”。

## 易错点

- **比的后项不能为 0。**
- 化简比后结果一般写成最简整数比，不写成分数。
- 判断比例时，不是只看数字像不像，而要看两个比值是否相等。
- 正比例和反比例不能只看“一个增一个减”，还要看有没有“不变量”。

## 例题精讲

### 例 1：化简比
化简 `18 : 24`

**解答：**
`18 : 24 = 3 : 4`

### 例 2：小数比化简
化简 `0.6 : 1.5`

**解答：**
先同时乘 10，得 `6 : 15`，再化简成 `2 : 5`。

### 例 3：判断比例
判断 `3 : 5` 和 `12 : 20` 能否组成比例。

**解答：**
`3 ÷ 5 = 0.6`，`12 ÷ 20 = 0.6`，所以能组成比例。

### 例 4：解比例
解比例：`x : 18 = 2 : 3`

**解答：**
`x ÷ 18 = 2 ÷ 3`

`3x = 36`

`x = 12`

### 例 5：按比分配
把 60 本练习本按 `2 : 3` 分给甲、乙两个小组，各分多少本？

**解答：**
总份数：`2 + 3 = 5`

每份：`60 ÷ 5 = 12`（本）

甲：`12 × 2 = 24`（本）

乙：`12 × 3 = 36`（本）

### 例 6：正比例与反比例
购买单价一定时，总价和数量是什么关系？

**解答：** 正比例。因为 `总价 ÷ 数量 = 单价`（比值一定）。

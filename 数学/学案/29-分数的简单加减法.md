# 学案 29：分数的简单加减法（同分母）

> 三年级 | 分数的简单加减法

---

## 📌 知识点

### 同分母分数加法
分母不变，分子相加。
```
a/b + c/b = (a+c)/b
```

### 同分母分数减法
分母不变，分子相减。
```
a/b - c/b = (a-c)/b
```

### 计算注意
- 结果要化成最简分数
- 分子为0时，分数为0
- 分子等于分母时，分数为1

---

## 📝 例题

### 例题 1：加法计算
计算：1/5 + 2/5

**解法**：
```
1/5 + 2/5 = (1+2)/5 = 3/5
```

**答案**：3/5

---

### 例题 2：减法计算
计算：5/7 - 2/7

**解法**：
```
5/7 - 2/7 = (5-2)/7 = 3/7
```

**答案**：3/7

---

### 例题 3：结果化简
计算：2/6 + 2/6

**解法**：
```
2/6 + 2/6 = 4/6 = 2/3
```

**答案**：2/3

---

### 例题 4：应用题
一块布做了窗帘用去 2/8，做床单用去 3/8，一共用了多少？

**解法**：
```
2/8 + 3/8 = 5/8
```

**答**：一共用了5/8。

---

### 例题 5：应用题（减法）
一桶水有 7/8 桶，用去了 3/8 桶，还剩多少？

**解法**：
```
7/8 - 3/8 = 4/8 = 1/2
```

**答**：还剩1/2桶。

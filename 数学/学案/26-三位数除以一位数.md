# 学案 26：三位数除以一位数

> 三年级 | 三位数除以一位数

---

## 📌 知识点

### 核心计算方法
三位数除以一位数的计算步骤：
1. **判断首位**：看被除数的前一位能否除以除数
2. **逐位除**：从高位到低位依次除
3. **余数**：余数必须小于除数

### 竖式计算方法
```
   1 2 3    ← 商
 3 ) 3 6 9  ← 被除数
   3        ← 3×1=3
   -
   0 6      ← 余数6
     6      ← 3×2=6
   -
     9      ← 余数9
     9      ← 3×3=9
   -
     0      ← 整除
```

### 特殊情况的商
- **首位不够除**：看前两位
- **中间有0**：0要写出来
- **末尾有0**：0不省略

## 📝 例题

### 例题 1：基础计算
计算：369 ÷ 3

**解法**：
```
   1 2 3
 3 ) 3 6 9
   3
   -
   0 6
     6
   -
     9
     9
   -
     0
```

**答案**：123

---

### 例题 2：首位不够除
计算：124 ÷ 4

**解法**：
```
    3 1
 4 ) 1 2 4
    1 2    ← 4×3=12
    -
      4    ← 余数4
      4    ← 4×1=4
    -
      0
```

**答案**：31

---

### 例题 3：商中间有0
计算：525 ÷ 5

**解法**：
```
    1 0 5
 5 ) 5 2 5
    5
    -
    0 2
      0
    -
      2 5
      2 5
    -
      0
```

**答案**：105

---

### 例题 4：应用题
408个苹果装进4个箱子，平均每个箱子装多少个？

**解法**：
```
408 ÷ 4 = 102（个）
```

**答**：平均每个箱子装102个。

# 学案 33：亿以内数的认识

> 四年级 | 大数先分级，再读写

---

## 本课学什么

- 认识亿以内的计数单位和数位顺序。
- 会把大数按“四位一级”分级，正确读数、写数。
- 会处理读数和写数时的“0”。

## 核心理解

### 1. 计数单位和数位
常见计数单位有：个、十、百、千、万、十万、百万、千万。

数位是这些单位所在的位置，例如：个位、十位、百位……万位、十万位等。

### 2. 为什么要分级
读大数时，如果一位一位地读，很容易乱。我国读数习惯是**四位一级**：

```text
个级：个、十、百、千
万级：万、十万、百万、千万
```

所以读数时，常常先从左到右分成一组一组。

### 3. 读数时怎么处理 0
- 每一级末尾的 0 不读。
- 中间有一个 0 或连续几个 0，都只读一个“零”。

例如：`3050 0400` 读作“三千零五十万零四百”。

### 4. 写数时为什么有时要补 0
写数时，如果某一级或某一位上没有数字，也要用 0 占位，不然数位就会错位。

## 方法与提醒

### 读数步骤
1. 先分级。
2. 从高位级读起。
3. 每读完一级，再加上“万”等级名。
4. 按规则处理 0。

### 写数步骤
1. 先确定有几个级。
2. 按顺序写出每一级上的数。
3. 缺哪一位就补 0。

## 易错点

- **把“万级”和“个级”混在一起读。**
- 连续几个 0 不能读成好几个“零”。
- 写数时漏写 0，会导致数位全部错位。
- 读数和写数都要从高位开始检查。

## 例题精讲

### 例 1：读数
读出 `40750020`。

**分级：** `4075 | 0020`

**解答：** 四千零七十五万零二十。

### 例 2：写数
写出“三千零六万零九十”。

**分析：**
- 万级：3006
- 个级：0090

**解答：** `30060090`

### 例 3：说出数位
在数字 `58420317` 中：
- 8 在什么数位上？
- 2 表示多少？

**解答：**
- 8 在百万位上。
- 2 在万位上，表示 2 个万。

### 例 4：易错辨析
`5008006` 读作“五百万八千零六”对吗？

**解答：** 对。

**说明：** 分级后是 `500 | 8006`，中间的 0 不单独多读。

### 例 5：生活应用
某市图书馆藏书 `1265000` 册，这个数读作什么？它是几位数？

**解答：** 一百二十六万五千；7 位数。

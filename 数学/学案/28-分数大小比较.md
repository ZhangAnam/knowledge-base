# 学案 28：分数大小比较

> 三年级 | 分数大小比较

---

## 📌 知识点

### 比较方法

#### 1. 同分母比较
分母相同，分子大的分数大。
```
2/5 < 4/5 < 5/5
```

#### 2. 同分子比较
分子相同，分母大的分数小。
```
1/2 > 1/3 > 1/4 > 1/5
```

#### 3. 分子分母都不同
需要通分或画图比较。

#### 4. 与1/2比较
- 分子 > 分母的一半：大于1/2
- 分子 = 分母的一半：等于1/2
- 分子 < 分母的一半：小于1/2

---

## 📝 例题

### 例题 1：同分母比较
比较 3/7 和 5/7 的大小

**解法**：分母相同，看分子
- 3 < 5，所以 3/7 < 5/7

**答案**：3/7 < 5/7

---

### 例题 2：同分子比较
比较 1/4 和 1/3 的大小

**解法**：分子相同，看分母
- 4 > 3，所以 1/4 < 1/3

**答案**：1/4 < 1/3

---

### 例题 3：与1/2比较
判断 3/5 与 1/2 的大小

**解法**：
- 5 ÷ 2 = 2.5
- 3 > 2.5，所以 3/5 > 1/2

**答案**：3/5 > 1/2

---

### 例题 4：通分比较
比较 2/3 和 3/4 的大小

**解法**：通分到同分母
- 3和4的最小公倍数是12
- 2/3 = 8/12，3/4 = 9/12
- 8/12 < 9/12

**答案**：2/3 < 3/4

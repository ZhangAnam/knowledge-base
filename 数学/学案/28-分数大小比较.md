# 学案 28：分数大小比较

> 三年级 | 会比大小，先看“分成多少份”和“取了多少份”

---

## 本课学什么

- 会比较**同分母分数**和**同分子分数**的大小。
- 会借助 `1`、`1/2` 等熟悉的分数帮助判断大小。
- 知道比较分数时，必须先确认比较的是**同一个整体**。

## 核心理解

### 1. 同分母分数怎么比
把同一个整体平均分成同样多份时，**谁取的份数多，谁就大**。

例如：`3/8` 和 `5/8`，每份都是 `1/8`，5 份比 3 份多，所以 `5/8 > 3/8`。

### 2. 同分子分数怎么比
取的份数一样多时，**每份分得越大，整个分数就越大**。

例如：`1/3` 和 `1/5`，都是取 1 份，但把整体分成 3 份时，每份比分成 5 份时更大，所以 `1/3 > 1/5`。

### 3. 可以借助“1”来比较
- 分子等于分母，分数等于 `1`。
- 分子小于分母，分数小于 `1`。
- 分子大于分母，分数大于 `1`。

### 4. 比大小前先看整体是否相同
两个分数如果对应的整体不同，就不能只看分数符号来判断实际大小。

## 比较方法

### 方法一：同分母比较
分母相同，比较分子；分子大的分数大。

### 方法二：同分子比较
分子相同，比较分母；分母小的分数大。

### 方法三：和 1 或 1/2 比较
遇到不方便直接比的分数，可以想办法改写或借助熟悉的分数判断。

## 易错点

- **同分子比较时，规律和同分母比较正好相反。**
- 不能见到“分母大”就说“分数大”。
- 比较分数大小时，默认整体相同；整体不同，实际大小可能不同。
- 书写比较结果时要用 `>`、`<` 或 `=`，不要只写“大”“小”。

## 例题精讲

### 例 1：同分母比较
比较 `2/7` 和 `5/7` 的大小。

**解答：** 分母都为 7，比较分子。因为 `2 < 5`，所以 `2/7 < 5/7`。

### 例 2：同分子比较
比较 `3/4` 和 `3/8` 的大小。

**分析：** 都取 3 份，但 `1/4` 比 `1/8` 大，所以 3 个 `1/4` 比 3 个 `1/8` 大。

**解答：** `3/4 > 3/8`

### 例 3：借助 1 比较
比较 `7/9` 和 `9/9` 的大小。

**解答：** `9/9 = 1`，而 `7/9` 比 1 小，所以 `7/9 < 9/9`。

### 例 4：借助“相同份数”比较
比较 `3/8` 和 `1/2` 的大小。

**分析：** `1/2 = 4/8`，所以只要比较 `3/8` 和 `4/8`。

**解答：** `3/8 < 1/2`

### 例 5：应用题
同样大的一块月饼，小强吃了 `3/6`，小华吃了 `2/6`，谁吃得多？

**解答：** 因为 `3/6 > 2/6`，所以小强吃得多。

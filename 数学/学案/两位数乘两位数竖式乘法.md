# 两位数乘两位数竖式乘法

## 一、竖式乘法基本格式

```
     23
  ×  45
  -----
```

## 二、计算步骤

### 第一步：乘以个位
先用下面的数的**个位**（5）乘以上面的数（23）

```
     23
  ×  45
  -----
    115   ← 23 × 5 = 115
```

### 第二步：乘以十位
再用下面的数的**十位**（4）乘以上面的数（23）
**注意：十位相乘后，结果要左移一位（空一列）**

```
     23
  ×  45
  -----
    115   ← 23 × 5 = 115
    920   ← 23 × 40 = 920（4是十位，所以92后面要加0）
  -----
```

### 第三步：相加
把两次乘积相加

```
     23
  ×  45
  -----
    115   ← 23 × 5 = 115
    920   ← 23 × 40 = 920
  -----
   1035   ← 115 + 920 = 1035
```

## 三、完整示例

### 例1：23 × 45 = 1035

```
     23      ← 被乘数（23）
  ×  45      ← 乘数（45）
  -----
    115      ← 23 × 5 = 115
    920      ← 23 × 40 = 920
  -----
   1035      ← 115 + 920 = 1035
```

### 例2：37 × 28 = 1036

```
     37
  ×  28
  -----
    296      ← 37 × 8 = 296
    740      ← 37 × 20 = 740
  -----
   1036
```

### 例3：56 × 47 = 2632

```
     56
  ×  47
  -----
    392      ← 56 × 7 = 392
   2240      ← 56 × 40 = 2240
  -----
   2632
```

## 四、核心要点

| 要点 | 说明 |
|------|------|
| 1. 乘个位 | 用乘数的个位乘被乘数 |
| 2. 乘十位 | 用乘数的十位乘被乘数，结果**左移一位** |
| 3. 相加 | 两次乘积相加 |

## 五、记忆口诀

> **竖式乘法要记牢，**
> **先乘个位后乘十位，**
> **十位结果往左挪，**
> **最后相加错不了！**

## 六、易错点提醒

❌ **错误**：忘记把十位乘积左移
```
     23
  ×  45
  -----
    115
    92     ← 错！没有左移
  -----
    207    ← 错误答案
```

✅ **正确**：十位乘积左移一位
```
     23
  ×  45
  -----
    115
    920    ← 对！左移了一位（4×23=92，后面加0变成920）
  -----
   1035    ← 正确答案
```

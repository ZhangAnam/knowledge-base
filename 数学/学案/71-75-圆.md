# 学案 71-75：圆

> 六年级 | 圆的认识、周长和面积

---

## 📌 知识点

### 圆的认识
- 圆心：O
- 半径：r（连接圆心和圆上任意一点）
- 直径：d（通过圆心的弦）
- 关系：d = 2r

### 圆的周长
C = 2πr = πd（π≈3.14）

### 圆的面积
S = πr²

### 圆环面积
S = π(R² - r²)

---

## 📝 例题

### 例题 1
r=5cm，求周长和面积

**解法**：
C = 2×3.14×5 = 31.4cm
S = 3.14×5² = 78.5cm²

**答案**：周长31.4cm，面积78.5cm²

# 学案 32：长方形、正方形面积计算

> 三年级 | 长方形、正方形面积

---

## 📌 知识点

### 长方形面积
```
面积 = 长 × 宽
S = a × b
```

### 正方形面积
```
面积 = 边长 × 边长
S = a × a = a²
```

### 公式逆用
```
长 = 面积 ÷ 宽
宽 = 面积 ÷ 长
边长 = √面积
```

---

## 📝 例题

### 例题 1：基础计算
一个长方形，长6厘米，宽4厘米，求面积。

**解法**：
```
S = 6 × 4 = 24 (cm²)
```

**答案**：24 平方厘米

---

### 例题 2：正方形计算
一个正方形，边长9厘米，求面积。

**解法**：
```
S = 9 × 9 = 81 (cm²)
```

**答案**：81 平方厘米

---

### 例题 3：逆用公式
一个长方形，面积30平方厘米，宽5厘米，长是多少？

**解法**：
```
长 = 30 ÷ 5 = 6 (cm)
```

**答案**：6 厘米

---

### 例题 4：知道周长求面积
一个长方形，周长20厘米，宽3厘米，求面积。

**解法**：
```
(长 + 3) × 2 = 20
长 + 3 = 10
长 = 7 cm

面积 = 7 × 3 = 21 cm²
```

**答案**：21 平方厘米

---

### 例题 5：应用题
学校操场是一个长方形，长100米，宽60米。这个操场的面积是多少平方米？合多少公顷？

**解法**：
```
面积 = 100 × 60 = 6000 (m²)
6000 m² = 0.6 公顷
```

**答案**：6000平方米，合0.6公顷。

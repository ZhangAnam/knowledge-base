# 考点10：函数的极值

> 高中数学 | 导数

---

## 📌 考点详解

### 极值定义
设函数f(x)在点x₀附近有定义：
- 若对x₀附近的所有点，都有f(x) < f(x₀)，则f(x₀)是**极大值**，x₀是**极大值点**
- 若对x₀附近的所有点，都有f(x) > f(x₀)，则f(x₀)是**极小值**，x₀是**极小值点**

### 极值判定定理
1. f'(x₀) = 0
2. x₀两侧导数符号**异号**

### 第一充分条件
f'(x)在x₀两侧异号 ⇒ x₀是极值点

---

## 📌 求极值的步骤

1. 求定义域
2. 求f'(x)
3. 求f'(x)=0的根（候选极值点）
4. 判断根两侧符号
5. 求极值

---

## 📝 典型例题

### 例1：求极值
求f(x) = x³ - 3x的极值。

> 📋 答案
> f'(x) = 3x² - 3 = 3(x+1)(x-1)
> f'(x)=0 ⇒ x = ±1
> x < -1：f'(x) > 0
> -1 < x < 1：f'(x) < 0
> x > 1：f'(x) > 0
> 极大值：f(-1) = 2
> 极小值：f(1) = -2

# 考点9：利用导数研究函数的单调性

> 高中数学 | 导数

---

## 📌 考点详解

### 单调性判定
- f'(x) > 0 ⇒ f(x)单调递增
- f'(x) < 0 ⇒ f(x)单调递减
- f'(x) = 0 ⇒ 可能为极值点

### 解题步骤
1. 求定义域
2. 求导数f'(x)
3. 解不等式f'(x)>0或f'(x)<0
4. 写出单调区间

---

## 📌 导数与极值

### 极值点
f'(x)=0的点，且两侧符号不同

### 极值
极值点处的函数值

---

## 📝 典型例题

### 例1：求单调区间
求f(x) = x³ - 3x²的单调区间。

> 📋 答案
> 定义域：R
> f'(x) = 3x² - 6x = 3x(x-2)
> f'(x) > 0：x < 0 或 x > 2
> f'(x) < 0：0 < x < 2
> 增区间：(-∞,0)，(2,+∞)
> 减区间：(0,2)

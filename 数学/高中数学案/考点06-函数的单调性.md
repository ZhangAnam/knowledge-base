# 考点6：函数的单调性

> 高中数学 | 函数

---

## 📌 考点详解

### 定义
对于定义域I内某个区间D上的任意x₁,x₂：
- 若x₁ < x₂时，f(x₁) < f(x₂)，则f(x)在D上**增函数**
- 若x₁ < x₂时，f(x₁) > f(x₂)，则f(x)在D上**减函数**

### 证明步骤
1. 取值：设x₁ < x₂
2. 作差：f(x₁) - f(x₂)
3. 变形
4. 判断符号
5. 结论

---

## 📌 单调性常用结论

### 复合函数
同增异减

### 四则运算
- 增+增=增
- 减+减=减
- 增-减=增
- 减-增=减

---

## 📝 典型例题

### 例1：证明单调性
证明f(x) = x³在R上是增函数。

> 📋 答案
> 设x₁ < x₂
> f(x₁) - f(x₂) = x₁³ - x₂³
> = (x₁-x₂)(x₁²+x₁x₂+x₂²)
> 因为x₁-x₂ < 0，且x₁²+x₁x₂+x₂² > 0
> 所以f(x₁) - f(x₂) < 0
> f(x)在R上是增函数

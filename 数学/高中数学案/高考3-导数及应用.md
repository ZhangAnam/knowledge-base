# 高考专题3：导数及应用

> 类型：高考专题学案 | 对应知识体系：B3-4 至 B3-7 | 目标：导数大题与压轴基础

---

## 本课学什么
- 本专题突出“求导—列表—结论—回到题意”的完整流程。
- 围绕“导数的概念与计算”建立可直接讲授的知识链与方法链。
- 围绕“利用导数研究函数的单调性”建立可直接讲授的知识链与方法链。
- 围绕“函数的极值”建立可直接讲授的知识链与方法链。

## 核心概念 / 性质 / 方法
1. 高考导数题不能只会求导，还要会解释符号、比较端点、处理参数。
2. 【导数的概念与计算】导数定义来源于平均变化率的极限：f'(x0)=lim_{Δx→0}[f(x0+Δx)-f(x0)]/Δx。
3. 【利用导数研究函数的单调性】研究单调性的标准流程：求定义域→求导→解 f'(x)>0 或 f'(x)<0→写增减区间。
4. 【函数的极值】求极值的一般流程：求导→找驻点或不可导点→看左右导数符号变化。
5. 【函数的最值】闭区间连续函数一定有最值，求法通常是比较端点值与区间内极值点函数值。

## 易错点
- 【导数的概念与计算】把函数值与导数值混淆，f(1) 与 f'(1) 含义完全不同。
- 【利用导数研究函数的单调性】只求出驻点，不继续判断导数符号。
- 【函数的极值】只解出 f'(x)=0 就直接写极值，没看符号变化。
- 【函数的最值】在闭区间题中漏算端点值。

## 递进例题
### 例1：用公式求导

**题目**：求 y=x^3-2x 的导数。

**讲解**：
1. 利用幂函数求导公式：(x^n)'=nx^{n-1}。
2. 分别求导得 (x^3)'=3x^2，(-2x)'=-2。
3. 和差法则下直接相加。

**答案**：y'=3x^2-2

### 例2：二次函数单调区间

**题目**：研究函数 f(x)=x^2-2x 的单调性。

**讲解**：
1. 求导得 f'(x)=2x-2。
2. 令 f'(x)=0，得 x=1。
3. 当 x<1 时 f'(x)<0，函数递减；当 x>1 时 f'(x)>0，函数递增。

**答案**：在 (-∞,1] 上递减，在 [1,+∞) 上递增

### 例3：求三次函数极值

**题目**：求 f(x)=x^3-3x 的极值。

**讲解**：
1. 求导得 f'(x)=3x^2-3=3(x-1)(x+1)。
2. 关键点为 x=-1,1。列表可得：导数在 (-∞,-1) 为正，在 (-1,1) 为负，在 (1,+∞) 为正。
3. 所以 x=-1 处极大，x=1 处极小，再代回原函数求函数值。

**答案**：极大值为 f(-1)=2，极小值为 f(1)=-2

## 练习

2. 研究 f(x)=x^3 的单调性。
3. 求 f(x)=x^2-4x 的极值。
4. 求 f(x)=x^2+1 在 [-1,2] 上的最值。

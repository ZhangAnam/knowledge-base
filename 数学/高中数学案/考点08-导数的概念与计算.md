# 考点8：导数的概念与计算

> 高中数学 | 导数

---

## 📌 考点详解

### 导数的定义
f'(x₀) = lim(Δx→0) [f(x₀+Δx) - f(x₀)] / Δx

### 几何意义
导数f'(x₀)表示曲线y=f(x)在点(x₀,f(x₀))处的切线斜率

### 基本导数公式
| 函数 | 导数 |
|------|------|
| C | 0 |
| xⁿ | nxⁿ⁻¹ |
| sin x | cos x |
| cos x | -sin x |
| eˣ | eˣ |
| aˣ | aˣlna |
| ln x | 1/x |
| logₐx | 1/(x·lna) |

---

## 📌 导数的运算法则
- (u±v)' = u' ± v'
- (uv)' = u'v + uv'
- (u/v)' = (u'v - uv')/v²

---

## 📝 典型例题

### 例1：求导数
求y = x³ + 2x² - 5x + 3的导数。

> 📋 答案
> y' = 3x² + 4x - 5

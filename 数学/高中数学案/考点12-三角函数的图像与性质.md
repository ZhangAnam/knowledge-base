# 考点12：三角函数的图像与性质

> 类型：考点学案 | 对应知识体系：C1-C5 | 聚焦：周期、奇偶、最值、平移与伸缩

---

## 本课学什么
- 理解正弦、余弦、正切函数的图像特征与性质来源。
- 会求周期、振幅、初相与对称轴等基础信息。
- 能处理图像变换后的函数性质。

## 核心概念 / 性质 / 方法
1. y=sin x 与 y=cos x 都以 2π 为最小正周期，值域为 [-1,1]；y=tan x 的最小正周期为 π。
2. 图像变换 y=A sin(ωx+φ)+b 中，A 控制振幅，ω 控制周期，φ 控制左右平移，b 控制上下平移。
3. 判断三角函数性质时，先看基本函数，再看变换，不要一次性混在一起。
4. 诱导公式与同角关系常用于把陌生角化成熟悉角。

## 易错点
- 把周期公式 T=2π/ω 用到 tan 上，正切的周期应是 π/ω。
- 平移方向判断反，特别是 ωx+φ 中的相位处理。
- 忽略定义域导致正切函数性质写错。
- 最值与振幅混淆，振幅是离中线的距离，不是最大值本身。

## 递进例题
### 例1：求周期与值域

**题目**：求函数 y=2sin(3x)-1 的周期和值域。

**讲解**：
1. 由 y=A sin(ωx)+b 知 A=2，ω=3，b=-1。
2. 周期 T=2π/3。
3. sin(3x) 的值域为 [-1,1]，乘 2 后为 [-2,2]，再整体下移 1。

**答案**：周期为 2π/3，值域为 [-3,1]

### 例2：判断奇偶性

**题目**：函数 y=sin x 是奇函数还是偶函数？

**讲解**：
1. 利用公式 sin(-x)=-sin x。
2. 满足奇函数定义，因此图像关于原点对称。

**答案**：奇函数

3. 正切函数的最小正周期是多少？

# 考点18：一元二次不等式

> 高中数学 | 不等式

---

## 📌 考点详解

### 解法步骤
1. 化标准：ax² + bx + c > 0 (a>0)
2. 求方程：ax² + bx + c = 0的根
3. 写解集：看二次函数图像

### 解集口诀
- 大于0：两根之外
- 小于0：两根之间

---

## 📌 典型情况

| Δ | 方程根 | >0解集 | <0解集 |
|---|--------|--------|--------|
| >0 | x₁<x₂ | (-∞,x₁)∪(x₂,+∞) | (x₁,x₂) |
| =0 | x₀ | (-∞,x₀)∪(x₀,+∞) | ∅ |
| <0 | 无实根 | R | ∅ |

---

## 📝 典型例题

### 例1：解不等式
x² - 5x + 6 > 0

> 📋 答案
> x² - 5x + 6 = 0 ⇒ x = 2 或 x = 3
> 解集：(-∞,2) ∪ (3,+∞)

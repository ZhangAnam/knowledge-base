# 考点1：集合的概念与表示

> 高中数学 | 集合与逻辑

---

## 📌 考点详解

### 集合的定义
某些确定的对象组成的整体

### 集合的表示
- 列举法：{1,2,3}
- 描述法：{x|x>0}
- 图示法：Venn图

### 元素与集合的关系
- ∈：属于
- ∉：不属于

---

## 📌 常见数集符号

| 符号 | 名称 |
|------|------|
| N | 自然数集 |
| N*或N+ | 正整数集 |
| Z | 整数集 |
| Q | 有理数集 |
| R | 实数集 |
| ∅ | 空集 |

---

## 📝 典型例题

### 例1：判断元素关系
判断下列关系是否正确：
(1) 0 ∈ N (2) √2 ∈ Q (3) -1 ∈ Z

> 📋 答案
> (1) 错误，0 ∉ N*，但0 ∈ N（视教材而定）
> (2) 错误，√2 ∉ Q
> (3) 正确

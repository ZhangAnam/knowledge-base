# 考点38：平面向量的概念与运算

> 类型：考点学案 | 对应知识体系：C7 | 聚焦：向量、模、共线与线性运算

---

## 本课学什么
- 理解向量既有大小又有方向，是研究几何关系的重要工具。
- 掌握向量的加减法、数乘及其几何意义。
- 会判断向量共线与相等。

## 核心概念 / 性质 / 方法
1. 向量可以用有向线段表示，但向量本身与起点位置无关，只与大小和方向有关。
2. 零向量长度为 0，方向不确定；单位向量长度为 1。
3. 向量加法满足三角形法则与平行四边形法则，减法可看成加上相反向量。
4. 若向量 b=λa，则 a 与 b 共线，这是证明平行的重要代数条件。

## 易错点
- 把“有向线段”与“向量”完全等同，忽略自由向量本质。
- 零向量与单位向量概念混淆。
- 向量相等只看长度，不看方向。
- 共线条件中漏掉数乘关系。

## 递进例题
### 例1：用三角形法则求和

**题目**：已知向量 a、b 首尾相接，如何表示它们的和？

**讲解**：
1. 把向量 a 的终点作为向量 b 的起点。
2. 从 a 的起点指向 b 的终点的向量，就是 a+b。

**答案**：a+b 由起点指向终点

### 例2：判断共线

**题目**：若 b=2a，说明 a 与 b 有什么关系？

**讲解**：
1. 一个向量是另一个向量的实数倍。
2. 这正是向量共线的判定条件。

**答案**：a 与 b 共线，且方向相同

## 练习
1. 长度为 0 的向量叫什么？
2. 若 b=-3a，则 a 与 b 的方向关系如何？
3. 向量加法常用哪两个几何法则？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 零向量 |
| 2 | 相反 |
| 3 | 三角形法则、平行四边形法则 |

# 学案 30-31：平面直角坐标系

> 七年级下册 | 对应知识体系 C1-1 ~ C1-2：平面直角坐标系、点的位置表示、坐标中的平移与对称

---

## 本课学什么

这节课学习平面直角坐标系：怎样用有序数对表示点的位置，怎样判断点所在象限，以及图形或点经过平移、对称后坐标怎样变化。学完后，你要能准确读写坐标，会根据符号判断位置，还能用坐标描述简单的图形变化。

## 核心理解 / 方法

### 1. 为什么需要平面直角坐标系

在数轴上只能表示“一维位置”，而平面上的点需要两个方向来确定：

- 横向位置
- 纵向位置

所以引入两条互相垂直的数轴：

- 横轴：`x` 轴
- 纵轴：`y` 轴
- 交点：原点 `O(0,0)`

### 2. 点的坐标是有顺序的

平面上的点通常记作：

```text
A(x, y)
```

- 前一个数表示横坐标
- 后一个数表示纵坐标

例如：

- `A(3,2)`：先向右 3，再向上 2
- `B(-2,4)`：先向左 2，再向上 4

> 坐标是“有序数对”，`(3,2)` 和 `(2,3)` 表示的不是同一个点。

### 3. 四个象限怎样判断

| 象限 | 横坐标 | 纵坐标 |
|---|---|---|
| 第一象限 | `+` | `+` |
| 第二象限 | `-` | `+` |
| 第三象限 | `-` | `-` |
| 第四象限 | `+` | `-` |

特别注意：

- 在 `x` 轴上的点，纵坐标是 0
- 在 `y` 轴上的点，横坐标是 0
- 坐标轴上的点不属于任何象限

### 4. 关于坐标轴或原点对称时，坐标怎么变

若点 `P(x,y)`：

| 变化 | 对应坐标 |
|---|---|
| 关于 x 轴对称 | `(x,-y)` |
| 关于 y 轴对称 | `(-x,y)` |
| 关于原点对称 | `(-x,-y)` |

理解方法：

- 关于 `x` 轴对称：上下翻，左右不变
- 关于 `y` 轴对称：左右翻，上下不变
- 关于原点对称：横纵坐标都变号

### 5. 平移时看“加减变化”

如果点 `(x,y)`：

- 向右平移 3 个单位：`(x+3,y)`
- 向左平移 3 个单位：`(x-3,y)`
- 向上平移 2 个单位：`(x,y+2)`
- 向下平移 2 个单位：`(x,y-2)`

## 易错点

1. **把横坐标和纵坐标顺序写反**
   - `(2,5)` 与 `(5,2)` 不同。

2. **把坐标轴上的点当作某个象限内的点**
   - 只要有一个坐标为 0，就不在任何象限。

3. **关于 x 轴、y 轴对称变号弄反**
   - 关于 x 轴：只变 `y`
   - 关于 y 轴：只变 `x`

4. **平移时方向与加减搞错**
   - 向右是横坐标增加，不是减少。

## 递进例题

### 例 1：读出点的坐标

点 `A` 在平面直角坐标系中，向右 4 个单位，向上 3 个单位，写出点 `A` 的坐标。

**解答**：

```text
A(4,3)
```

### 例 2：判断象限

点 `B(-2,5)` 在第几象限？

**分析**：横坐标负，纵坐标正。

**解答**：

在第二象限。

### 例 3：判断坐标轴上的点

点 `C(0,-4)` 在哪里？

**解答**：

因为横坐标为 0，所以点 `C` 在 `y` 轴上，不属于任何象限。

### 例 4：关于坐标轴对称

点 `P(3,-2)`：

1. 关于 `x` 轴对称后的点是什么？
2. 关于 `y` 轴对称后的点是什么？
3. 关于原点对称后的点是什么？

**解答**：

1. `(3,2)`
2. `(-3,-2)`
3. `(-3,2)`

### 例 5：平移变化

点 `Q(-1,4)` 向右平移 5 个单位，再向下平移 2 个单位，得到点 `Q'`，求 `Q'` 的坐标。

**解答**：

```text
(-1+5, 4-2) = (4,2)
```

### 例 6：综合判断

已知点 `M(a,b)` 在第四象限，那么 `a` 与 `b` 分别满足什么条件？

**解答**：

第四象限横正纵负，所以：

```text
a > 0，b < 0
```

## 练习

1. 写出点 `A` 的坐标：向左 3 个单位，向上 2 个单位。
2. 点 `(-5,-1)` 在第几象限？
3. 点 `(6,0)` 在哪里？
4. 点 `(2,3)` 关于 `x` 轴对称后的坐标是多少？
5. 点 `(2,3)` 关于 `y` 轴对称后的坐标是多少？
6. 点 `(2,3)` 关于原点对称后的坐标是多少？
7. 点 `(-4,1)` 向右平移 6 个单位后坐标是多少？
8. 判断：点 `(0,5)` 在第一象限。对还是错？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `(-3,2)` |
| 2 | 第三象限 |
| 3 | 在 `x` 轴上 |
| 4 | `(2,-3)` |
| 5 | `(-2,3)` |
| 6 | `(-2,-3)` |
| 7 | `(2,1)` |
| 8 | 错，在 `y` 轴上 |

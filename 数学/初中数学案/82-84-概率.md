# 学案 82-84：概率

> 九年级上册 | 概率

---

## 📌 知识点

### 概率
P(A) = 事件A可能结果数 / 所有可能结果数

### 计算方法
- 列举法
- 树状图
- 频率估计概率

### 概率性质
- 0 ≤ P(A) ≤ 1
- P(必然事件) = 1
- P(不可能事件) = 0

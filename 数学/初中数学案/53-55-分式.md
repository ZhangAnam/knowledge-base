# 学案 53-55：分式

> 八年级下册 | 对应知识体系 A5-1 ~ A5-4：分式概念、基本性质、约分通分与运算

---

## 本课学什么

这节课学习分式的意义、分式有无意义的条件、分式的基本性质，以及约分、通分、加减乘除运算。学完后，你要会规范处理分式，也要能先判断字母取值限制，再进行化简和计算。

## 核心理解 / 方法

### 1. 什么是分式

形如 `A/B`，且分母中含有字母，叫分式。

### 2. 分式有意义的条件

分母不能等于 0。

所以做题时常先求使分母不为 0 的条件。

### 3. 分式的基本性质

分式的分子、分母同时乘或除以同一个不为 0 的整式，分式值不变。

### 4. 约分与通分

- 约分：把分子分母中的公因式约去
- 通分：把几个分式化成同分母分式

### 5. 分式运算

- 乘法：分子乘分子，分母乘分母
- 除法：除以一个分式等于乘它的倒数
- 加减法：先通分，再合并

## 易错点

1. **忽略分母不能为 0**
2. **分子分母只约去一部分，不是公因式也乱约**
3. **加减分式时分母直接相加**
4. **除法忘记化成乘倒数**

## 递进例题

### 例 1：判断有意义条件

分式 `1/(x-2)` 在什么条件下有意义？

**答案**：`x ≠ 2`。

### 例 2：约分

化简：`6x/9x²`

**解答**：`2/(3x)`，同时要求 `x ≠ 0`。

### 例 3：加法

计算：

```text
1/x + 1/2x
```

**解答**：

```text
= 2/2x + 1/2x
= 3/2x
```

### 例 4：除法

计算：

```text
x/3 ÷ x²/6
```

**解答**：

```text
x/3 × 6/x² = 2/x
```

## 练习

1. 分式 `2/(x+1)` 有意义时，`x` 不能等于多少？
2. 化简：`8a/12a²`
3. 计算：`1/a + 2/a`
4. 计算：`x/5 ÷ x/10`

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `-1` |
| 2 | `2/(3a)`，且 `a ≠ 0` |
| 3 | `3/a` |
| 4 | `2`，且 `x ≠ 0` |

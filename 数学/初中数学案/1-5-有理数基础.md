# 学案 1-5：有理数基础

> 七年级上册 | 对应知识体系 A1-1 ~ A1-5：正负数、有理数分类、数轴、相反数与绝对值、大小比较

---

## 本课学什么

这节课要建立“有理数”的整体认识：为什么需要负数，怎样在数轴上表示数，怎样理解相反数与绝对值，以及怎样比较有理数大小。学完后，你不仅要会做题，更要能说清“符号表示什么”“绝对值表示什么”“数轴为什么能帮助比较大小”。

## 核心理解 / 方法

### 1. 为什么要有负数

在小学里，我们常用正数表示数量；到了初中，很多量带有**方向或相反意义**，这时只靠正数不够。

- 向东走 3 米，记作 `+3 米`
- 向西走 3 米，记作 `-3 米`
- 盈利 50 元，记作 `+50 元`
- 亏损 50 元，记作 `-50 元`

所以，**正负号不仅表示大小的一部分，还表示意义或方向**。

### 2. 有理数怎样分类

有理数包括：

- **整数**：`…，-3，-2，-1，0，1，2，3，…`
- **分数**：如 `1/2，-3/4，2.5，-0.8`

也可以按符号分成：

- 正有理数
- 0
- 负有理数

> 关键提醒：`0` 是整数，也是有理数，但**既不是正数，也不是负数**。

### 3. 数轴是“数”和“位置”的桥梁

数轴的三要素：

1. **原点**
2. **正方向**
3. **单位长度**

在数轴上：

- 原点右边的数是正数
- 原点左边的数是负数
- 越靠右的点表示的数越大

因此，数轴不仅能表示数，还能帮助我们理解大小关系。

### 4. 相反数与绝对值要分清

| 概念 | 含义 | 例子 |
|---|---|---|
| 相反数 | 只有符号不同、绝对值相等的两个数 | `3` 与 `-3` |
| 绝对值 | 一个数到原点的距离 | `|-3|=3`，`|3|=3` |

- `a` 的相反数是 `-a`
- 互为相反数的两个数在数轴上位于原点两侧，到原点距离相等
- 绝对值一定**不小于 0**

绝对值分段理解：

```text
|a| = a   (a ≥ 0)
|a| = -a  (a < 0)
```

### 5. 比较有理数大小的三种常用方法

#### 方法一：用数轴比较
越靠右越大。

#### 方法二：先看正负
- 正数大于 0，0 大于负数
- 所以：正数 > 0 > 负数

#### 方法三：同为负数时看绝对值
两个负数比较大小，**绝对值大的反而小**。

例如：

- `-2 > -5`
- 因为 `-2` 在数轴上更靠右
- 也因为 `|-2| < |-5|`

## 易错点

1. **把 0 误认为正数**
   - 错：0 是最小的自然数，所以是正数。
   - 对：0 既不是正数，也不是负数。

2. **把相反数和绝对值混为一谈**
   - `-5` 的相反数是 `5`
   - `-5` 的绝对值也是 `5`
   - 这道题恰好结果相同，但概念完全不同。

3. **负数比较大小时只看数字大小，不看符号**
   - 错：`-8 > -3`
   - 对：`-8 < -3`

4. **忘记绝对值表示距离，一定非负**
   - 不可能出现 `|a| = -3`

## 递进例题

### 例 1：用正负数表示相反意义的量

某天早晨气温比 0℃ 低 4℃，中午气温比 0℃ 高 6℃，分别记作什么？

**分析**：低于 0℃ 用负数表示，高于 0℃ 用正数表示。

**解答**：

- 早晨记作 `-4℃`
- 中午记作 `+6℃`

**小结**：先判断“相对基准是增加还是减少、上升还是下降”。

### 例 2：在数轴上判断大小

比较 `-3`、`2`、`0` 的大小。

**分析**：正数在右边，负数在左边，0 在中间。

**解答**：

```text
2 > 0 > -3
```

**小结**：正数一定比负数大，0 介于两者之间。

### 例 3：相反数与绝对值辨析

写出 `-7` 的相反数和绝对值。

**解答**：

- 相反数：`7`
- 绝对值：`7`

**说明**：这题结果一样，但“相反数”是换符号，“绝对值”是看距离。

### 例 4：负数大小比较

比较 `-4` 和 `-9` 的大小。

**分析**：两个都是负数，绝对值小的反而大。

**解答**：

因为 `|-4| = 4`，`|-9| = 9`，且 `4 < 9`，所以：

```text
-4 > -9
```

### 例 5：综合判断

把下面各数填入对应横线上：`-3，0，2.5，-1/2，7`

- 正有理数：______
- 负有理数：______
- 整数：______

**解答**：

- 正有理数：`2.5，7`
- 负有理数：`-3，-1/2`
- 整数：`-3，0，7`

## 练习

1. 用正负数表示：存入 200 元记作 `+200`，那么取出 80 元记作多少？
2. 判断：`0` 是不是有理数？是不是正数？
3. 在数轴上，离原点 5 个单位长度的整数有哪两个？
4. 写出下列各数的相反数：`6，-2.5，0`
5. 求下列各数的绝对值：`|-8|，|3/4|，|0|`
6. 比较大小：`-6` 与 `1`
7. 比较大小：`-2` 与 `-7`
8. 把 `-4，3，0，-1.2，5` 按从小到大的顺序排列。

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `-80` |
| 2 | `0` 是有理数，但不是正数 |
| 3 | `-5` 和 `5` |
| 4 | `-6，2.5，0` |
| 5 | `8，3/4，0` |
| 6 | `-6 < 1` |
| 7 | `-2 > -7` |
| 8 | `-4 < -1.2 < 0 < 3 < 5` |

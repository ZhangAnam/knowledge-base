# 学案 16-18：一元一次方程

> 七年级上册 | 对应知识体系 B1-1 ~ B1-3：等式与方程、一元一次方程解法、实际问题建模

---

## 本课学什么

这节课学习一元一次方程：什么情况下需要设未知数，为什么解方程时可以去括号、移项、去分母，以及怎样把实际问题转化成方程。学完后，你要会解标准方程，也要能从题目中的数量关系列出方程。

## 核心理解 / 方法

### 1. 等式和方程有什么区别

- **等式**：表示左右两边相等的式子，如 `3+2=5`
- **方程**：含有未知数的等式，如 `x+2=5`

所以，方程首先是等式，只不过它里面有未知数，需要求解。

### 2. 什么是一元一次方程

满足两个条件：

1. 只含有一个未知数
2. 未知数的次数是 1

例如：

- `3x+2=11` 是一元一次方程
- `x^2=9` 不是，因为次数是 2
- `x+y=5` 不是，因为有两个未知数

### 3. 解方程为什么可以这样做

解方程的本质是：**在保持等式两边相等的前提下，把未知数单独留在一边**。

常用依据：

- 等式两边同时加上或减去同一个数，仍相等
- 等式两边同时乘或除以同一个不为 0 的数，仍相等

### 4. 一元一次方程的基本步骤

常见顺序：

1. 去分母
2. 去括号
3. 移项
4. 合并同类项
5. 系数化为 1
6. 检验

> 不是每道题都要六步都写出来，但思路通常按这个顺序更稳。

### 5. 列方程解决应用题的核心

一般分四步：

1. **审题**：找等量关系
2. **设未知数**：用字母表示所求量
3. **列方程**：把等量关系写成方程
4. **解方程并作答**

常见等量关系：

- 总量 = 部分量之和
- 原有量 ± 变化量 = 现在量
- 单价 × 数量 = 总价
- 路程 = 速度 × 时间

## 易错点

1. **移项只换位置，不变符号**
   - `3x+5=11` 中，`+5` 移到右边应变为 `-5`。

2. **去分母时漏乘某一项**
   - 方程每一项都要乘公分母。

3. **去括号时负号分配错误**
   - `-(x-2)=-x+2`

4. **系数化为 1 时，忘记左右同时除以系数**

5. **应用题只写出 x，不写单位和结论**

## 递进例题

### 例 1：最基础的一步方程

解方程：`x+7=19`

**解答**：

```text
x = 19 - 7 = 12
```

### 例 2：需要移项和合并同类项

解方程：`3x-5=2x+7`

**解答**：

```text
3x - 5 = 2x + 7
3x - 2x = 7 + 5
x = 12
```

### 例 3：含括号方程

解方程：`2(x+3)=14`

**解答**：

```text
2x + 6 = 14
2x = 8
x = 4
```

### 例 4：含分母方程

解方程：`x/3 + 2 = 5`

**解答**：

```text
x/3 = 3
x = 9
```

也可以写成：方程两边同时乘 3，得 `x+6=15`，所以 `x=9`。

### 例 5：应用题建模

某班买练习册，每本 8 元，一共买了若干本，又付了 4 元运费，共花 60 元。买了多少本？

**分析**：总价 = 书本价 + 运费。

设买了 `x` 本，则：

```text
8x + 4 = 60
```

解得：

```text
8x = 56
x = 7
```

**答**：买了 7 本。

### 例 6：易错型应用题

一个数的 3 倍比这个数大 10，求这个数。

**分析**：
“3 倍比这个数大 10”表示：

```text
3x = x + 10
```

不是 `3x+10=x`。

**解答**：

```text
3x = x + 10
2x = 10
x = 5
```

## 练习

1. 解方程：`x-6=13`
2. 解方程：`4x=28`
3. 解方程：`2x+3=11`
4. 解方程：`5x-7=3x+9`
5. 解方程：`3(x-2)=12`
6. 解方程：`x/4 - 1 = 2`
7. 某商品单价 12 元，买若干件共花 84 元，买了多少件？
8. 某数加上 9 等于它的 3 倍，求这个数。

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `x=19` |
| 2 | `x=7` |
| 3 | `x=4` |
| 4 | `x=8` |
| 5 | `x=6` |
| 6 | `x=12` |
| 7 | 设买了 `x` 件，`12x=84`，得 `x=7`，买了 7 件 |
| 8 | 设这个数为 `x`，`x+9=3x`，得 `x=4.5` |

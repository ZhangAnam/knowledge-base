# 学案 44-52：因式分解

> 八年级上册 | 对应知识体系 A4-1 ~ A4-4：公因式、公式法与整式结构化处理

---

## 本课学什么

这节课学习因式分解的意义、提公因式法和公式法，理解因式分解与整式乘法互为逆过程。学完后，你要会把一个整式写成几个整式乘积的形式，也要能根据结构特征快速识别适合的方法。

## 核心理解 / 方法

### 1. 什么是因式分解

把一个整式化成几个整式的乘积形式，叫做因式分解。

例如：

```text
ax + ay = a(x + y)
```

### 2. 因式分解和整式乘法的关系

因式分解是整式乘法的逆过程。

所以，做因式分解时要反过来观察：原式像不像某个乘法结果。

### 3. 提公因式法

若各项都有共同因数，可以先提公因式。

步骤：

1. 找各项共同因数；
2. 提到括号外；
3. 检查括号内各项是否正确。

### 4. 公式法

常见公式：

```text
a² - b² = (a+b)(a-b)
a² ± 2ab + b² = (a ± b)²
```

### 5. 一般思路

- 先看能否提公因式；
- 再看是否符合平方差或完全平方公式；
- 必要时多次分解，直到不能继续。

## 易错点

1. **不是乘积形式也叫因式分解**
2. **漏提公因式中的符号**
3. **平方差公式写反**
4. **把 `a²+b²` 误分成 `(a+b)²`**

## 递进例题

### 例 1：提公因式

分解：`3x + 6`

**解答**：`3(x+2)`。

### 例 2：平方差公式

分解：`x² - 9`

**解答**：`(x+3)(x-3)`。

### 例 3：完全平方公式

分解：`x² + 6x + 9`

**解答**：`(x+3)²`。

### 例 4：先提后套公式

分解：`2x² - 18`

**解答**：

```text
2(x² - 9) = 2(x+3)(x-3)
```

## 练习

1. 分解：`4a + 8`
2. 分解：`y² - 16`
3. 分解：`x² - 10x + 25`
4. 分解：`3x² - 27`

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `4(a+2)` |
| 2 | `(y+4)(y-4)` |
| 3 | `(x-5)²` |
| 4 | `3(x+3)(x-3)` |

# 学案 19-22：几何图形初步

> 七年级上册 | 对应知识体系 D1-1 ~ D1-2：直线、射线、线段与角，角的关系与基本表达

---

## 本课学什么

这节课学习几何中的基础语言：点、线、线段、射线、角，以及余角、补角、邻补角等关系。重点是把图形“看懂、说准、写规范”。学完后，你要能准确识别几何对象，会用符号表示图形关系，并能处理基础角度计算。

## 核心理解 / 方法

### 1. 几何从“形”开始，但要会用语言表达

常见几何对象：

| 对象 | 特点 |
|---|---|
| 点 | 只表示位置，没有大小 |
| 直线 | 向两端无限延伸，没有端点 |
| 射线 | 有一个端点，向一端无限延伸 |
| 线段 | 有两个端点，可以度量长度 |
| 角 | 由有公共端点的两条射线组成 |

### 2. 图形表示要规范

- 点用大写字母表示，如 `A`、`B`
- 直线可记作直线 `AB`
- 射线可记作射线 `OA`
- 线段可记作线段 `AB`
- 角可记作 `∠AOB` 或 `∠1`

> 角的顶点必须写在中间，例如 `∠AOB` 的顶点是 `O`。

### 3. 线段的基本事实

- 两点确定一条直线
- 两点之间，线段最短
- 如果点 `C` 在线段 `AB` 上，那么：

```text
AC + CB = AB
```

这叫线段的和差关系。

### 4. 角的度量与分类

按大小分类：

- 锐角：小于 `90°`
- 直角：等于 `90°`
- 钝角：大于 `90°` 小于 `180°`
- 平角：等于 `180°`
- 周角：等于 `360°`

### 5. 余角、补角、邻补角

| 名称 | 条件 |
|---|---|
| 余角 | 两个角和为 `90°` |
| 补角 | 两个角和为 `180°` |
| 邻补角 | 有公共顶点和一条公共边，另一边互为反向延长线，且和为 `180°` |

> 余角、补角强调的是“数量关系”；邻补角既有数量关系，又有位置关系。

## 易错点

1. **把射线和线段混淆**
   - 射线只有一个端点，线段有两个端点。

2. **写角时顶点不放中间**
   - `∠AOB` 才表示顶点是 `O`。

3. **把“互补”误写成“相等”**
   - 两个补角不一定相等，它们只是和为 `180°`。

4. **看见相邻两个角就当邻补角**
   - 还要看另一边是否互为反向延长线。

## 递进例题

### 例 1：识别几何对象

指出下列说法对应的几何对象：

1. 有两个端点，可以测量长度
2. 有一个端点，向一端无限延伸
3. 向两端无限延伸

**解答**：

1. 线段
2. 射线
3. 直线

### 例 2：线段和差关系

点 `C` 在线段 `AB` 上，已知 `AC=3 cm`，`CB=5 cm`，求 `AB`。

**解答**：

```text
AB = AC + CB = 3 + 5 = 8(cm)
```

### 例 3：余角和补角计算

若一个角是 `35°`，求它的余角和补角。

**解答**：

- 余角：`90°-35°=55°`
- 补角：`180°-35°=145°`

### 例 4：邻补角求未知角

如图（文字理解）：`∠1` 与 `∠2` 是邻补角，且 `∠1=68°`，求 `∠2`。

**解答**：

因为邻补角和为 `180°`，所以：

```text
∠2 = 180° - 68° = 112°
```

### 例 5：综合判断

下列说法正确的是（  ）

A. 两个角相等，它们一定是余角  
B. 两个角互补，它们一定相邻  
C. 90° 的角没有余角  
D. 线段可以向两端延长成两条射线

**解答**：

选 `D`。

- A 错，相等不代表和为 90°
- B 错，补角不一定相邻
- C 错，90° 的余角是 0°，通常在本课范围内不作为正角讨论，但“没有余角”表达不严谨
- D 对

## 练习

1. 直线有几个端点？射线有几个端点？
2. 点 `C` 在线段 `AB` 上，`AC=7 cm`，`CB=4 cm`，求 `AB`。
3. 一个角是 `24°`，它的余角是多少？
4. 一个角是 `132°`，它的补角是多少？
5. 若 `∠AOB=90°`，`∠BOC=35°`，并且 `∠AOC=∠AOB+∠BOC`，求 `∠AOC`。
6. 判断：两个角和为 `180°`，这两个角一定是邻补角。对还是错？
7. 判断：两点之间线段最短。对还是错？
8. 写出角 `∠XYZ` 的顶点是哪一点。

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 直线 0 个端点，射线 1 个端点 |
| 2 | `AB=11 cm` |
| 3 | `66°` |
| 4 | `48°` |
| 5 | `125°` |
| 6 | 错，可能不相邻 |
| 7 | 对 |
| 8 | 顶点是 `Y` |

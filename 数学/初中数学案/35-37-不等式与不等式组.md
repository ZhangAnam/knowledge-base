# 学案 35-37：不等式与不等式组

> 七年级下册 | 对应知识体系 B3-1 ~ B3-4：不等式意义、解法、数轴表示与应用

---

## 本课学什么

这节课学习不等式与一元一次不等式组：为什么不等式两边同乘负数时方向要改变，怎样在数轴上表示解集，以及怎样把“至少、至多、不超过、不少于”这类实际条件翻译成不等式。学完后，你要能解不等式，也能判断不等式组的公共解。

## 核心理解 / 方法

### 1. 不等式表示什么

不等式描述的是“大小关系”，而不是一个确定值。

常见符号：

- `>` 大于
- `<` 小于
- `≥` 大于等于
- `≤` 小于等于

### 2. 解不等式的依据

- 两边同时加上或减去同一个数，不等号方向不变。
- 两边同时乘或除以同一个正数，不等号方向不变。
- 两边同时乘或除以同一个负数，**不等号方向改变**。

这是解不等式最关键的一条规则。

### 3. 一元一次不等式的解法步骤

1. 去分母
2. 去括号
3. 移项
4. 合并同类项
5. 系数化为 1
6. 注意负数除法时变号

### 4. 数轴怎样表示解集

- 空心圆点：不包含该点
- 实心圆点：包含该点
- 向右：表示大于某数
- 向左：表示小于某数

### 5. 不等式组的公共解

两个不等式同时成立的解，叫做不等式组的解集。

口诀可理解为：

- “同大取大”
- “同小取小”
- “大小小大取中间”
- “大大小小无解”

## 易错点

1. **乘或除以负数时忘记变号**
2. **解集与某个特殊值混淆**
3. **数轴端点画错实心或空心**
4. **不等式组只解其中一个，不求公共部分**

## 递进例题

### 例 1：基础不等式

解不等式：`x + 3 > 7`

**解答**：`x > 4`

### 例 2：需要变号

解不等式：`-2x < 6`

**解答**：两边同除以 `-2`，不等号变向，得 `x > -3`。

### 例 3：不等式组

解不等式组：

```text
x > 1
x ≤ 4
```

**解答**：公共解为 `1 < x ≤ 4`。

### 例 4：应用题

某商品每件成本 15 元，售价 20 元。要使利润不少于 200 元，至少卖出多少件？

**分析**：每件利润 `5` 元。

**列不等式**：`5x ≥ 200`

**解答**：`x ≥ 40`，所以至少卖出 40 件。

## 练习

1. 解不等式：`3x - 2 ≥ 7`
2. 解不等式：`-4x > 12`
3. 解不等式组：

```text
x ≥ -1
x < 3
```

4. 某班订阅报纸，每份 3 元，预算不超过 90 元，最多可订多少份？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `x ≥ 3` |
| 2 | `x < -3` |
| 3 | `-1 ≤ x < 3` |
| 4 | `3x ≤ 90`，得 `x ≤ 30`，所以最多 30 份 |

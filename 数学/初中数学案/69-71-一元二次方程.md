## 📌 知识点

### 一元二次方程
ax² + bx + c = 0 (a≠0)

### 解法
1. 直接开平方法
2. 配方法
3. 公式法：x = (-b ± √(b²-4ac)) / 2a
4. 因式分解法

### 判别式
Δ = b² - 4ac
- Δ > 0：两个不相等实根
- Δ = 0：两个相等实根
- Δ < 0：无实根

---

## 📝 例题

### 例题 1
x² - 5x + 6 = 0

**解法**：因式分解：(x-2)(x-3)=0

**答案**：x₁=2, x₂=3

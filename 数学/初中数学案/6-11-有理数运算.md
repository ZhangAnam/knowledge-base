# 学案 6-11：有理数运算

> 七年级上册 | 对应知识体系 A2-1 ~ A2-6：加法、减法、乘法、除法、乘方、混合运算

---

## 本课学什么

这节课学习有理数的基本运算及混合运算。重点不是死记符号法则，而是理解：加减法怎样统一，乘除法为什么有“同号得正、异号得负”，乘方表示什么，混合运算为什么必须讲顺序。学完后，你要能把运算写对、顺序理清，还能对结果的正负做出判断。

## 核心理解 / 方法

### 1. 有理数加法：先看符号，再看绝对值

#### 同号两数相加
- 取相同的符号
- 绝对值相加

例如：

```text
(-3) + (-5) = -(3+5) = -8
```

#### 异号两数相加
- 取绝对值较大的那个数的符号
- 用较大的绝对值减去较小的绝对值

例如：

```text
(-7) + 4 = -(7-4) = -3
```

### 2. 减法统一成加法

有理数减法最核心的一句话：

```text
a - b = a + (-b)
```

也就是：**减去一个数，等于加上这个数的相反数**。

这样做的好处是：减法问题统一转化为加法问题。

### 3. 乘法和除法先判符号，再算绝对值

#### 乘法法则
- 同号得正
- 异号得负
- 任何数与 0 相乘都得 0

#### 除法法则
- 同号得正，异号得负
- 除以一个不等于 0 的数，等于乘这个数的倒数

```text
a ÷ b = a × 1/b  (b ≠ 0)
```

### 4. 乘方表示“几个相同因数相乘”

```text
a^n = a × a × ... × a（n 个 a）
```

- `a` 叫底数
- `n` 叫指数

特别注意：

- `(-2)^4 = 16`
- `(-2)^3 = -8`
- `-2^4 = -(2^4) = -16`

> 说明：如果负号没有被括号一起括住，先算乘方，再添负号。

### 5. 混合运算顺序

有理数混合运算与小学一致，但更要注意符号。

运算顺序：

1. 先算乘方
2. 再算乘除
3. 最后算加减
4. 有括号先算括号里面

### 6. 运算前先做“结果预判”

做题前可以先想：

- 结果大概是正还是负？
- 大概比哪个数大、比哪个数小？
- 有没有可以简便计算的地方？

这能帮助你减少低级错误。

## 易错点

1. **异号相加时直接把绝对值相加**
   - 错：`(-8)+3=-11`
   - 对：`(-8)+3=-5`

2. **减法不先变加法**
   - `5-(-2)` 不是 `5-2`，而是 `5+2`

3. **负号和乘方括号看错**
   - `(-3)^2 = 9`
   - `-3^2 = -9`

4. **除法中忘记除数不能为 0**
   - `a÷0` 没有意义

5. **混合运算不按顺序，见一步算一步**
   - 复杂式子一定先整体观察，再动笔。

## 递进例题

### 例 1：异号加法

计算：`(-11) + 6`

**分析**：异号相加，看绝对值大小。

**解答**：

- `|-11| > |6|`
- 结果取负号
- `11 - 6 = 5`

所以：

```text
(-11) + 6 = -5
```

### 例 2：减法转化

计算：`7 - (-4)`

**解答**：

```text
7 - (-4) = 7 + 4 = 11
```

**小结**：看到减法，先想“加相反数”。

### 例 3：乘除法符号判断

计算：`(-3) × 5 × (-2)`

**分析**：两个负因数，积为正。

**解答**：

```text
(-3) × 5 × (-2)
= -15 × (-2)
= 30
```

### 例 4：乘方辨析

计算：`(-2)^4` 与 `-2^4`

**解答**：

```text
(-2)^4 = 16
-2^4 = -(2^4) = -16
```

**说明**：括号决定负号是不是底数的一部分。

### 例 5：混合运算

计算：`18 ÷ (-3) + 2 × (-4)^2`

**解答**：

```text
18 ÷ (-3) + 2 × (-4)^2
= -6 + 2 × 16
= -6 + 32
= 26
```

### 例 6：利用运算律简算

计算：`(-2.5) + 7 + (-7) + 2.5`

**分析**：先找互为相反数或能凑整的数。

**解答**：

```text
[(-2.5) + 2.5] + [7 + (-7)] = 0 + 0 = 0
```

# 学案 61-63：平行四边形

> 八年级下册 | 四边形专题 | 对应知识体系：D3-1 平行四边形判定与性质、D3-2 特殊平行四边形、D3-3 梯形与中位线

---

## 本课学什么

本课学习平行四边形的**性质、判定和常见转化思路**，并认识矩形、菱形、正方形这些特殊平行四边形。  
学完后，你要能判断一个四边形是否为平行四边形，能根据题目条件选择合适的性质或判定，还要会用“平行四边形 → 特殊平行四边形”的层层深入思路解决问题。

---

## 核心理解 / 方法

### 一、先分清“性质”和“判定”

| 类型 | 作用 | 常见表述 |
|---|---|---|
| 性质 | 已知是平行四边形，推出别的结论 | 对边平行、对边相等、对角相等、对角线互相平分 |
| 判定 | 已知一些条件，证明它是平行四边形 | 两组对边分别平行；两组对边分别相等；一组对边平行且相等；对角线互相平分 |

> **关键提醒：**  
> “性质”是**已有身份再用特点**，  
> “判定”是**先凑条件证明身份**。

### 二、平行四边形的核心性质

设四边形 \(ABCD\) 是平行四边形，则：

1. \(AB \parallel CD\)，\(AD \parallel BC\)
2. \(AB = CD\)，\(AD = BC\)
3. \(\angle A = \angle C\)，\(\angle B = \angle D\)
4. 邻角互补
5. 对角线互相平分：若对角线交于点 \(O\)，则 \(AO = CO\)，\(BO = DO\)

### 三、四种高频判定方法

1. **两组对边分别平行**
2. **两组对边分别相等**
3. **一组对边平行且相等**
4. **对角线互相平分**

做题时要先看已知条件更接近哪一种，不要一上来就乱证角、证边。

### 四、特殊平行四边形的关系

| 图形 | 在平行四边形基础上增加的条件 | 典型性质 |
|---|---|---|
| 矩形 | 有一个角是直角 | 四个角都是直角；对角线相等 |
| 菱形 | 有一组邻边相等 | 四条边都相等；对角线互相垂直；对角线平分一组对角 |
| 正方形 | 既是矩形又是菱形 | 角、边、对角线的性质都最完整 |

### 五、梯形中位线的理解

梯形的中位线是连接两腰中点的线段，它有两个关键结论：

1. **平行于两底**
2. **长度等于两底和的一半**

它常用于把“复杂长度关系”转化成“平均量”关系。

---

## 易错点

1. **把“对角线相等”误当成平行四边形判定条件。**  
   对角线相等只能说明有可能是矩形，不能单独判定是平行四边形。

2. **把“一组对边相等”误当成充分条件。**  
   只有一组对边相等，不一定是平行四边形；必须达到“两组”或“一组平行且相等”。

3. **证明矩形、菱形时忘记先证明它是平行四边形。**  
   题目若没有直接给出“平行四边形”，要先建立这个身份。

4. **菱形对角线性质用错。**  
   菱形的对角线互相垂直、互相平分，并且平分对角；矩形则是对角线相等、互相平分。

5. **梯形中位线公式写反。**  
   是“和的一半”，不是“差的一半”。

## 递进例题

### 例题 1：用性质求边和角

在平行四边形 \(ABCD\) 中，\(AB = 8\)，\(BC = 5\)，\(\angle A = 70^\circ\)。求 \(CD\)、\(AD\)、\(\angle C\)、\(\angle B\)。

**思路：** 已知图形身份，直接用平行四边形性质。

**解：**

- 因为 \(ABCD\) 是平行四边形，
- 所以 \(AB = CD\)，\(BC = AD\)，\(\angle A = \angle C\)，邻角互补。

于是：

- \(CD = 8\)
- \(AD = 5\)
- \(\angle C = 70^\circ\)
- \(\angle B = 180^\circ - 70^\circ = 110^\circ\)

**答案：** \(CD=8\)，\(AD=5\)，\(\angle C=70^\circ\)，\(\angle B=110^\circ\)

### 例题 2：用对角线判定平行四边形

四边形 \(ABCD\) 的对角线 \(AC\) 与 \(BD\) 交于点 \(O\)，已知 \(AO = CO\)，\(BO = DO\)。求证：四边形 \(ABCD\) 是平行四边形。

**思路：** 已知两条对角线互相平分，直接选对应判定。

**证明：**

因为 \(AO = CO\)，\(BO = DO\)，  
所以四边形 \(ABCD\) 的两条对角线互相平分。  
根据“对角线互相平分的四边形是平行四边形”，可得：

\[
ABCD \text{ 是平行四边形。}
\]

**方法总结：**  
看到“交点把两条对角线都分成相等的两段”，优先想到平行四边形判定。

### 例题 3：由平行四边形进一步判定矩形

在平行四边形 \(ABCD\) 中，已知 \(\angle A = 90^\circ\)。求证：\(ABCD\) 是矩形。

**思路：** 矩形本质上是“有一个角是直角的平行四边形”。

**证明：**

因为 \(ABCD\) 是平行四边形，且 \(\angle A = 90^\circ\)，  
平行四边形邻角互补，所以 \(\angle B = 90^\circ\)；  
对角相等，所以 \(\angle C = 90^\circ\)，\(\angle D = 90^\circ\)。  
四个角都是直角，因此 \(ABCD\) 是矩形。

**答案：** \(ABCD\) 是矩形

### 例题 4：梯形中位线求长

在梯形 \(ABCD\) 中，\(AB \parallel CD\)，上底 \(AB = 6\)，下底 \(CD = 10\)，中位线长是多少？

**思路：** 直接用梯形中位线公式。

**解：**

\[
\text{中位线长}=\frac{AB+CD}{2}=\frac{6+10}{2}=8
\]

**答案：** 8

## 练习

### 基础理解

1. 在平行四边形 \(ABCD\) 中，若 \(AB = 12\)，则 \(CD =\) ______。
2. 在平行四边形 \(ABCD\) 中，若 \(\angle A = 48^\circ\)，则 \(\angle B =\) ______。
3. 平行四边形的对角线有什么位置与数量关系？

### 方法应用

4. 四边形 \(ABCD\) 中，已知 \(AB \parallel CD\)，且 \(AB = CD\)。判断 \(ABCD\) 是否一定是平行四边形，并说明理由。
5. 在平行四边形 \(ABCD\) 中，对角线交于点 \(O\)。若 \(AO = 4\)，\(BO = 6\)，求 \(AC\) 与 \(BD\)。
6. 一个平行四边形有一个角是直角，这个平行四边形是什么特殊四边形？

### 易错辨析

7. “对角线相等的四边形一定是平行四边形。”这句话对吗？请简要说明。
8. “有一组对边平行且相等的四边形是平行四边形。”这句话对吗？

### 综合提升

9. 在菱形 \(ABCD\) 中，已知两条对角线长分别为 6 和 8，求菱形的边长。
10. 梯形两底长分别为 5 和 13，求其中位线长。

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 12 |
| 2 | \(132^\circ\) |
| 3 | 对角线互相平分 |
| 4 | 一定是。因为一组对边平行且相等，可判定为平行四边形 |
| 5 | \(AC=8\)，\(BD=12\) |
| 6 | 矩形 |
| 7 | 不对。对角线相等不一定能判定平行四边形，例如等腰梯形的对角线也相等 |
| 8 | 对 |
| 9 | 边长为 5。因为对角线互相垂直平分，边长 \(=\sqrt{3^2+4^2}=5\) |
| 10 | 9 |

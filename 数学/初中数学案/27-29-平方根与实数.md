# 学案 27-29：平方根与实数

> 七年级下册 | 对应知识体系 A4-1 ~ A4-2：平方根、立方根、实数与数轴扩展

---

## 本课学什么

这节课学习平方根、算术平方根、立方根，以及由此扩展出的实数概念。重点是弄清“哪个数的平方等于它”“根号表示什么”“为什么负数没有平方根但有立方根”。学完后，你要能正确求常见数的平方根、算术平方根、立方根，并理解无理数也属于实数，能在数轴上找到位置。

## 核心理解 / 方法

### 1. 平方根：谁平方后等于这个数

如果 `x^2=a`，那么 `x` 叫做 `a` 的平方根。

例如：

- 因为 `3^2=9`，`(-3)^2=9`
- 所以 `9` 的平方根是 `3` 和 `-3`

结论：

- 正数有两个平方根，互为相反数
- 0 的平方根是 0
- 负数没有平方根

### 2. 算术平方根只取“非负的那个”

正数 `a` 的正平方根叫做 `a` 的算术平方根，记作 `√a`。

例如：

- `√9=3`
- `√16=4`

> 关键区别：
> - `9` 的平方根是 `±3`
> - `√9` 的值是 `3`

### 3. 立方根与平方根不同

如果 `x^3=a`，那么 `x` 叫做 `a` 的立方根，记作 `∛a`。

例如：

- `∛8=2`
- `∛(-8)=-2`

因为：

- 正数的立方根是正数
- 负数的立方根是负数
- 0 的立方根是 0

### 4. 实数：有理数与无理数合在一起

以前学的整数、分数都是有理数。现在出现像 `√2`、`π` 这样不能写成有限小数或无限循环小数的数，它们叫**无理数**。

- 有理数 + 无理数 = 实数

所以，实数包括：

- 有理数
- 无理数

### 5. 实数都可以在数轴上表示

虽然 `√2` 不能写成有限小数，但它仍然是一个确定的数，因此也能在数轴上找到对应点。这样，数轴就从“有理数的数轴”扩展成“实数的数轴”。

## 易错点

1. **把平方根和算术平方根混淆**
   - `16` 的平方根是 `±4`
   - `√16=4`

2. **认为负数也有平方根**
   - 在实数范围内，`√(-4)` 没有意义。

3. **把 `√a` 写成 `±√a`**
   - 根号 `√a` 本身表示算术平方根，只取非负值。

4. **误以为无理数不能在数轴上表示**
   - 无理数也有确定位置。

## 递进例题

### 例 1：求平方根

求 `25` 的平方根。

**解答**：

因为 `5^2=25`，`(-5)^2=25`，所以 `25` 的平方根是：

```text
±5
```

### 例 2：求算术平方根

求 `√49`。

**解答**：

```text
√49 = 7
```

### 例 3：辨析题

判断 `√9` 和 `9` 的平方根是否相同。

**分析**：

- `√9=3`
- `9` 的平方根是 `±3`
- `3` 的平方根是 `±√3`

所以它们不是同一回事。

### 例 4：求立方根

求 `∛(-27)`。

**解答**：

因为 `(-3)^3=-27`，所以：

```text
∛(-27) = -3
```

### 例 5：判断数的类别

在 `-2，0，3/5，√2，π` 中，哪些是有理数？哪些是无理数？

**解答**：

- 有理数：`-2，0，3/5`
- 无理数：`√2，π`

### 例 6：数轴上的实数

比较 `√3` 和 `1.8` 的大小。

**分析**：

因为 `√3≈1.732`，所以：

```text
√3 < 1.8
```

## 练习

1. 求 `36` 的平方根。
2. 求 `√64`。
3. 判断：`√(-9)` 在实数范围内有没有意义？
4. 求 `∛125`。
5. 求 `∛(-1)`。
6. 在 `-7，2/3，√5，0，π` 中，写出所有有理数。
7. 判断：`2` 和 `-2` 都是 `4` 的平方根。对还是错？
8. 判断：所有实数都能在数轴上表示。对还是错？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `±6` |
| 2 | `8` |
| 3 | 没有意义 |
| 4 | `5` |
| 5 | `-1` |
| 6 | `-7，2/3，0` |
| 7 | 对 |
| 8 | 对 |

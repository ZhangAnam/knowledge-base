# 学案 23-26：相交线与平行线

> 七年级下册 | 对应知识体系 D1-3：相交线、对顶角、邻补角、平行线的性质与判定

---

## 本课学什么

这节课学习相交线与平行线中的核心关系：对顶角、邻补角，以及同位角、内错角、同旁内角。重点是区分“性质”和“判定”：已知平行可以推出角的关系，已知角的关系也可以反过来判断平行。学完后，你要能看图找角、会用关系推角度，也会用角度关系判断两直线是否平行。

## 核心理解 / 方法

### 1. 相交线中的基本角关系

两条直线相交，会形成四个角。

#### 对顶角
- 位置：相对的两个角
- 性质：**对顶角相等**

#### 邻补角
- 位置：相邻
- 数量关系：和为 `180°`

所以，相交线中常用两条规律：

```text
对顶角相等
邻补角互补
```

### 2. 平行线与截线形成的三类重要角

一条截线截两条直线时，会出现：

| 角的名称 | 平行时的关系 |
|---|---|
| 同位角 | 相等 |
| 内错角 | 相等 |
| 同旁内角 | 互补 |

这三类角是本课的核心。

### 3. 性质与判定要分开

#### 平行线的性质
如果两条直线平行，那么：

- 同位角相等
- 内错角相等
- 同旁内角互补

#### 平行线的判定
如果：

- 同位角相等
- 或内错角相等
- 或同旁内角互补

那么这两条直线平行。

> 关键词区分：
> - “已知平行，推出角关系”是**性质**
> - “已知角关系，推出平行”是**判定**

### 4. 做题顺序建议

1. 先看图中哪两条线平行、哪两条线相交
2. 找到角的位置关系
3. 再决定用性质还是判定
4. 如果有未知角，先找能列式的等量关系

## 易错点

1. **把同位角和内错角位置看错**
   - 先找“截线”，再看角位。

2. **性质和判定混用**
   - 题目已知平行，用性质；题目要证明平行，用判定。

3. **看到两个角相等就直接说平行**
   - 必须先确认它们是同位角或内错角等对应关系。

4. **对顶角与邻补角混淆**
   - 对顶角相等；邻补角和为 `180°`。

## 递进例题

### 例 1：相交线中的基础关系

两条直线相交，`∠1=58°`，求它的对顶角和邻补角。

**解答**：

- 对顶角：`58°`
- 邻补角：`180°-58°=122°`

### 例 2：已知平行，用性质求角

已知 `a∥b`，一条截线与 `a` 所成的同位角是 `65°`，求 `b` 上对应的同位角。

**解答**：

因为两直线平行，同位角相等，所以该角也是 `65°`。

### 例 3：已知平行，用同旁内角求未知角

已知 `a∥b`，同旁内角一个是 `110°`，另一个是多少？

**解答**：

```text
180° - 110° = 70°
```

所以另一个角是 `70°`。

### 例 4：利用判定判断平行

已知两条直线被一条截线所截，内错角相等，能得出什么结论？

**解答**：

由“内错角相等，两直线平行”，可得这两条直线平行。

### 例 5：综合推理题

如图（文字理解）：`a∥b`，`∠1=50°`，`∠2` 与 `∠1` 是同旁内角，求 `∠2`。

**解答**：

因为平行线的同旁内角互补，所以：

```text
∠2 = 180° - 50° = 130°
```

### 例 6：易错辨析

判断：

“两个角相等，所以两条直线平行。”

**分析**：不一定。只有当这两个角是同位角或内错角等满足判定条件的位置关系时，才能推出平行。

## 练习

1. 两条直线相交，一个角是 `47°`，它的对顶角是多少？
2. 两条直线相交，一个角是 `47°`，与它相邻的角是多少？
3. 已知 `a∥b`，一个同位角是 `83°`，另一个同位角是多少？
4. 已知 `a∥b`，一个同旁内角是 `102°`，另一个同旁内角是多少？
5. 判断：内错角相等，两直线平行。对还是错？
6. 判断：同旁内角互补，两直线平行。对还是错？
7. 判断：对顶角互补。对还是错？
8. 若两条直线被第三条直线所截，同位角相等，可以得到什么结论？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `47°` |
| 2 | `133°` |
| 3 | `83°` |
| 4 | `78°` |
| 5 | 对 |
| 6 | 对 |
| 7 | 错，对顶角相等，不一定互补 |
| 8 | 两直线平行 |

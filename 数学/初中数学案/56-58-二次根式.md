# 学案 56-58：二次根式

> 八年级下册 | 对应知识体系 A6-1 ~ A6-4：二次根式概念、性质、化简与运算

---

## 本课学什么

这节课学习二次根式的意义、最简二次根式、同类二次根式，以及二次根式的加减乘除。学完后，你要会根据被开方数的条件判断式子是否有意义，也要会把根式化简成最简形式后再运算。

## 核心理解 / 方法

### 1. 什么是二次根式

形如 `√a` 的式子叫二次根式，其中要求 `a ≥ 0`。

### 2. 基本性质

- `(√a)² = a`（`a ≥ 0`）
- `√(a²) = |a|`

### 3. 最简二次根式

被开方数中不含能开得尽方的因数，分母中不含根号。

### 4. 化简思路

把被开方数分解成“完全平方数 × 其余部分”，再把完全平方数开出来。

例如：

```text
√12 = √(4×3) = 2√3
```

### 5. 根式运算

- 加减：先化成同类二次根式
- 乘法：`√a × √b = √ab`
- 除法：`√a / √b = √(a/b)`（`b>0`）

## 易错点

1. **忽略被开方数非负条件**
2. **把 `√(a+b)` 写成 `√a+√b`**
3. **`√(a²)` 忘记写绝对值**
4. **加减前不先化简**

## 递进例题

### 例 1：判断是否有意义

`√(x-1)` 有意义时，`x` 满足什么条件？

**答案**：`x ≥ 1`。

### 例 2：化简

化简：`√18`

**解答**：`√18 = √(9×2) = 3√2`。

### 例 3：同类二次根式加法

计算：`2√3 + 5√3`

**答案**：`7√3`。

### 例 4：综合化简

计算：`√8 + √18`

**解答**：

```text
√8 = 2√2
√18 = 3√2
所以结果是 5√2
```

## 练习

1. `√(2x+4)` 有意义时，`x` 满足什么条件？
2. 化简：`√27`
3. 计算：`3√5 - √5`
4. 计算：`√12 + √3`

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `x ≥ -2` |
| 2 | `3√3` |
| 3 | `2√5` |
| 4 | `3√3` |

# 学案 59-60：勾股定理

> 八年级下册 | 对应知识体系 D4-1 ~ D4-3：直角三角形边关系、逆定理与应用

---

## 本课学什么

这节课学习勾股定理、勾股定理逆定理，以及它在求边长、判断直角三角形中的应用。学完后，你要会在直角三角形中建立平方关系，也要能根据三边数量关系反向判断一个三角形是不是直角三角形。

## 核心理解 / 方法

### 1. 勾股定理内容

在直角三角形中，两条直角边长分别为 `a、b`，斜边长为 `c`，则：

```text
a² + b² = c²
```

### 2. 适用前提

只有在**直角三角形**中才能直接使用勾股定理。

### 3. 勾股定理逆定理

如果三角形三边长满足：

```text
a² + b² = c²
```

且 `c` 是最长边，那么这个三角形是直角三角形。

### 4. 常见用途

- 已知两边求第三边
- 判断三角形形状
- 解决实际测量问题
- 与面积、周长、坐标等结合

## 易错点

1. **不是直角三角形也乱用勾股定理**
2. **斜边不是最长边**
3. **逆定理中没先找最长边**
4. **求边长时忘记开方**

## 递进例题

### 例 1：求斜边

直角三角形两直角边长为 `3` 和 `4`，求斜边长。

**解答**：

```text
c² = 3² + 4² = 25
c = 5
```

### 例 2：求直角边

直角三角形斜边长为 `13`，一条直角边长为 `5`，求另一条直角边长。

**解答**：

```text
a² = 13² - 5² = 169 - 25 = 144
a = 12
```

### 例 3：逆定理判断

三边长分别为 `6、8、10`，这个三角形是不是直角三角形？

**解答**：

```text
6² + 8² = 36 + 64 = 100 = 10²
```

所以它是直角三角形。

### 例 4：实际应用

一个长方形长 `8`，宽 `6`，对角线长多少？

**答案**：`10`。

## 练习

1. 直角三角形两直角边是 `5、12`，斜边是多少？
2. 三边长为 `7、24、25` 的三角形是什么三角形？
3. 直角三角形斜边为 `10`，一条直角边为 `6`，另一条是多少？
4. 正方形边长为 `4`，对角线长多少？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | `13` |
| 2 | 直角三角形 |
| 3 | `8` |
| 4 | `4√2` |

# 学案 12-15：整式

> 七年级上册 | 对应知识体系 A3-1 ~ A3-4：用字母表示数、单项式、多项式、同类项与整式加减

---

## 本课学什么

这节课从“数字计算”走向“代数表达”。你要理解字母为什么能表示数，单项式和多项式怎样识别，同类项为什么才能合并，以及整式加减的本质是什么。学完后，你要能把生活中的数量关系写成式子，也能规范地整理整式。

## 核心理解 / 方法

### 1. 为什么要用字母表示数

字母可以表示：

- **未知数**：暂时不知道的量，如 `x`
- **变化的量**：会变化的数量，如速度 `v`
- **一般规律**：如长方形周长 `2(a+b)`

字母并不神秘，它只是把“具体数”推广成“普遍关系”。

### 2. 单项式：数与字母的积

像 `3a^2b`、`-xy`、`5` 这样的式子，都是单项式。

识别单项式时要会看两个关键词：

| 内容 | 含义 | 例：`-4x^2y` |
|---|---|---|
| 系数 | 数字因数 | `-4` |
| 次数 | 所有字母指数的和 | `2+1=3` |

> 常数也是单项式，非零常数的次数是 0。

### 3. 多项式：几个单项式的和

例如：

```text
2x^2 - 3x + 1
```

它由三个项组成：`2x^2`、`-3x`、`1`。

- 多项式有几项，就叫几项式
- 多项式的次数，是**次数最高的项的次数**
- 不含字母的项叫**常数项**

### 4. 同类项为什么才能合并

同类项必须满足：

- 所含字母相同
- 相同字母的指数也相同

例如：

- `3x` 和 `-5x` 是同类项
- `2ab^2` 和 `-7ab^2` 是同类项
- `3x` 和 `3x^2` 不是同类项
- `ab` 和 `ba` 是同类项（字母相同，指数相同）

**本质理解**：

`3x + 5x = 8x`，就像 3 个苹果加 5 个苹果等于 8 个苹果；
但 `3x + 5y` 不能合并，因为“东西不是同一种”。

### 5. 整式加减的步骤

1. 如果有括号，先去括号
2. 找同类项
3. 合并同类项
4. 按规范顺序整理

去括号要特别注意符号：

- `+(...)` 去括号后，括号内各项符号不变
- `-(...)` 去括号后，括号内各项符号都改变

## 易错点

1. **把字母相同但指数不同的项当成同类项**
   - `2x` 和 `2x^2` 不能合并。

2. **漏看系数前的负号**
   - `-a` 的系数是 `-1`，不是 `1`。

3. **去括号时只去掉括号，不变符号**
   - `5-(2x-1)=5-2x+1`

4. **把多项式次数看成所有项次数之和**
   - `x^2+x+1` 的次数是 2，不是 3。

## 递进例题

### 例 1：判断单项式的系数和次数

写出单项式 `-3a^2b` 的系数和次数。

**解答**：

- 系数：`-3`
- 次数：`2+1=3`

### 例 2：判断多项式的项和次数

多项式 `2x^2-5x+7` 有几项？次数是多少？常数项是多少？

**解答**：

- 三项：`2x^2`、`-5x`、`7`
- 次数：2
- 常数项：7

### 例 3：合并同类项

化简：`4x - 7x + 2`

**解答**：

```text
4x - 7x + 2 = -3x + 2
```

### 例 4：去括号再合并

化简：`3a + 2(a-4)`

**解答**：

```text
3a + 2(a-4)
= 3a + 2a - 8
= 5a - 8
```

### 例 5：负号括号的易错题

化简：`(2x-3) - (x+5)`

**分析**：第二个括号前是减号，去括号时每一项都要变号。

**解答**：

```text
(2x-3) - (x+5)
= 2x - 3 - x - 5
= x - 8
```

### 例 6：列式表达数量关系

长方形长为 `x`，宽比长少 3，写出它的周长。

**解答**：

宽是 `x-3`，周长为：

```text
2[x+(x-3)] = 4x-6
```

## 练习

1. 单项式 `5m^2n` 的系数是______，次数是______。
2. 多项式 `x^3-2x+4` 的次数是______，常数项是______。
3. 下列哪一组是同类项：`3a` 与 `5a^2`；`-2xy` 与 `7yx`；`4m^2n` 与 `m^2n^2`。
4. 化简：`7x-3x+5`
5. 化简：`2a+3a-4`
6. 化简：`5y-(2y-1)`
7. 化简：`3(2x-1)+x`
8. 某种练习本单价为 `a` 元，买 6 本应付多少元？若再加 2 元运费，一共多少元？

## 答案

| 题号 | 答案 |
|---|---|
| 1 | 系数 `5`，次数 `3` |
| 2 | 次数 `3`，常数项 `4` |
| 3 | 只有 `-2xy` 与 `7yx` 是同类项 |
| 4 | `4x+5` |
| 5 | `5a-4` |
| 6 | `3y+1` |
| 7 | `7x-3` |
| 8 | 6 本 `6a` 元，一共 `6a+2` 元 |

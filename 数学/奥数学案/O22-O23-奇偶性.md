# 奥数学案 O22-O23：奇偶性

> 三至四年级 | 奥数数论 | 主题：奇数偶数的运算规律、配对思想与结论判断

---

## 本课学什么

这一课学习奇偶性的基本规律，并用它解决“不用算出具体值，也能判断结果”的问题。  
核心能力是：**只看一个数是奇还是偶，就能快速判断和、差、积的性质，甚至判断某件事是否可能。**

---

## 核心知识与核心方法

### 1. 奇数与偶数的本质

- 偶数可以写成 `2n`
- 奇数可以写成 `2n+1`

因此，奇偶性的判断本质上是在看“能不能被 2 整除”。

### 2. 基本规律

| 运算 | 结果 |
| --- | --- |
| 偶 + 偶 | 偶 |
| 奇 + 奇 | 偶 |
| 奇 + 偶 | 奇 |
| 偶 - 偶 | 偶 |
| 奇 - 奇 | 偶 |
| 奇 - 偶 | 奇 |
| 偶 - 奇 | 奇 |
| 偶 × 任意整数 | 偶 |
| 奇 × 奇 | 奇 |

### 3. 为什么这些规律成立

例如：

```text
(2a+1)+(2b+1)=2(a+b+1)
```

结果是 2 的倍数，所以“奇 + 奇 = 偶”。

再如：

```text
(2a+1)×(2b+1)=2(2ab+a+b)+1
```

结果是 `2n+1` 的形式，所以“奇 × 奇 = 奇”。

### 4. 两种常见策略

| 策略 | 适用题型 |
| --- | --- |
| 直接套规律 | 判断和、差、积的奇偶性 |
| 配对思想 | 连续整数求和、很多数相加时的奇偶判断 |

---

## 易错点与关键提醒

1. **把“奇数个奇数相加”和“偶数个奇数相加”混淆。**  
   - 偶数个奇数相加得偶数  
   - 奇数个奇数相加得奇数

2. **把奇数乘奇数错判成偶数。**  
   这是高频错误，务必记牢：奇×奇=奇。

3. **只会背，不会用“2n、2n+1”解释。**  
   一旦题目变形，就容易乱。

4. **题目要的是奇偶，不必真的把大数算出来。**

## 例题

### 例题 1：最基础判断

判断：任意两个奇数的和是偶数。

**解答**

设两个奇数是 `2a+1` 和 `2b+1`：

```text
(2a+1)+(2b+1)=2(a+b+1)
```

是 2 的倍数，所以是偶数。

**答案**：正确

### 例题 2：连续乘积的奇偶性

奇数 × 奇数 × 奇数 × …… × 奇数（10 个）的积是奇数还是偶数？

**思路**：奇数乘奇数仍是奇数，连续乘下去结果不变。

**答案**：奇数

### 例题 3：连续整数和

`1+2+3+…+100` 是奇数还是偶数？

**解答**

把首尾配对：

```text
1+100=101
2+99=101
……
50+51=101
```

一共有 50 对，每对都是奇数。

```text
奇数×50（偶数个 101 相加）
```

偶数个奇数相加得偶数。

**答案**：偶数

### 例题 4：不用计算判断

判断 `(23×45)+(18×7)` 的和是奇数还是偶数。

**解答**

- `23×45`：奇×奇=奇
- `18×7`：偶×奇=偶
- 奇+偶=奇

**答案**：奇数

### 例题 5：不可能判断

5 个奇数的和可能是偶数吗？

**解答**

5 个奇数，是奇数个奇数相加，结果一定是奇数，不可能是偶数。

**答案**：不可能

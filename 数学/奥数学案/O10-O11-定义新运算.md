# 奥数学案 O10-O11：定义新运算

> 三至四年级 | 奥数计算模块 A3 | 主题：O10 简单定义新运算；O11 复杂定义新运算

---

## 本课学什么

这一课学习定义新运算题。  
它的本质不是普通四则运算，而是**先读懂规则，再按规则代入**。

- **O10** 侧重简单新运算：直接代入、按顺序计算。
- **O11** 侧重复杂新运算：嵌套、多个符号、逆向求值与比较大小。

## 对应知识体系与前序知识

- 对应知识体系：
  - **O10 简单定义新运算**
  - **O11 复杂定义新运算**
- 前序知识：
  - 课内 `15`：表内乘法
  - 课内 `16`：表内除法
  - 课内 `35`：三位数乘两位数
- 后续衔接：
  - 更复杂的符号规定题
  - 逻辑推理、代数思维与函数式阅读

## 核心模型与策略

### 一、O10：简单定义新运算

题目会先规定一个新符号的意思，例如：

```text
a★b=a×b+a-b
```

看到 `a★b`，就不能把它当成平常的 `+` 或 `×`，而要按照题目给出的规则翻译。

### 二、O11：复杂定义新运算

复杂题常见三种变化：

1. **嵌套运算**：如 `(2★3)★4`
2. **多个符号并存**：如 `★` 和 `◆` 同时出现
3. **逆向求值**：已知 `x★5=12`，反过来求 `x`

### 三、总策略：把新符号翻译成熟悉算式

定义新运算题并不神秘，本质上就是“新符号翻译题”。  
先翻译，再按普通运算顺序计算。

## 解题方法

### 方法一：先看清定义

一定先读完整个定义，确认：

- 谁对应 `a`
- 谁对应 `b`
- 运算顺序是什么

### 方法二：代入时尽量带括号

如果 `a` 或 `b` 本身是一个式子，带括号更安全。

### 方法三：嵌套题一层一层算

像 `(4★3)★2` 这种题，必须先算里面的 `4★3`，再把结果整体代入下一层。

### 方法四：逆向题先把新运算翻译成熟悉等式

例如 `x◆5=12`，先翻译成普通式子，再解。

## 易错点/关键提醒

1. **把位置放反。**  
   `a★b` 和 `b★a` 一般不一定相等。

2. **代入后漏括号。**  
   尤其当输入的是一个式子时，漏括号很容易算错。

3. **把新运算当成熟悉运算。**  
   新符号没有天然意义，只有题目规定的意义。

4. **嵌套题跳步。**  
   一层没算清，就直接往下代，会越来越乱。

## 递进例题

### 例 1：O10 直接代入

已知：

```text
a★b=a×b+a-b
```

求：`3★5`

```text
3★5
=3×5+3-5
=15+3-5
=13
```

答案：`13`

### 例 2：O10 看清位置

已知：

```text
a◎b=a-b×2
```

求：`8◎3`

```text
8◎3
=8-3×2
=8-6
=2
```

答案：`2`

### 例 3：O11 嵌套新运算

已知：

```text
a★b=a+b×2
```

求：`(4★3)★2`

```text
先算 4★3
4★3=4+3×2=10

再算 10★2
10★2=10+2×2=14
```

答案：`14`

### 例 4：O11 两个符号一起用

已知：

```text
a◆b=a+b
a★b=a×b-1
```

求：`(2◆3)★4`

```text
2◆3=2+3=5
5★4=5×4-1=19
```

答案：`19`

### 例 5：O11 逆向求值

已知：

```text
a⊙b=2a-b
```

若 `x⊙5=9`，求 `x`。

```text
x⊙5=2x-5
2x-5=9
2x=14
x=7
```

答案：`7`

## 分层练习

### A 组：O10 基础代入

1. 已知 `a★b=a×b+a-b`，求 `3★5`
2. 已知 `a◎b=2a+b`，求 `6◎4`
3. 已知 `a◆b=a²-b`，求 `5◆2`

### B 组：O11 嵌套与比较

4. 已知 `a★b=a+b+1`，求 `(2★3)★4`
5. 已知 `a◎b=a×b-b`，求 `(3◎4)◎2`
6. 已知 `a◆b=2a-b`，分别求 `3◆5` 与 `5◆3`

### C 组：O11 综合提升

7. 已知 `a★b=a+b-1`，若 `x★5=12`，求 `x`
8. 已知 `a◎b=a×b+2`，若 `4◎x=18`，求 `x`
9. 已知 `a★b=a+b×2`，`a◆b=a×b`，求 `(2★3)◆4`

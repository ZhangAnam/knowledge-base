# 奥数学案 O12：整除的概念

> 三年级 | 奥数数论

---

## 📌 知识点

### 整除的定义
整数 a 除以整数 b（b≠0），如果商是整数且没有余数，叫做 a 能被 b 整除。

### 写法
```
a ÷ b = c（c为整数）
记作：b | a
读作：b 整除 a
```

### 常见整除判定
| 除数 | 判定方法 |
|:---:|---------|
| 2 | 末位是偶数（0,2,4,6,8） |
| 3 | 各位数字之和能被3整除 |
| 4 | 末两位能被4整除 |
| 5 | 末位是0或5 |
| 6 | 同时能被2和3整除 |
| 9 | 各位数字之和能被9整除 |
| 11 | 奇数位之和与偶数位之和的差能被11整除 |

## 📝 例题

### 例题 1：判断整除
判断下列数能否被 2、3、5 整除：

1. 246
- 2：246÷2=123 ✓
- 3：2+4+6=12，能被3整除 ✓
- 5：末位不是0或5 ✗

2. 105
- 2：奇数 ✗
- 3：1+0+5=6，能被3整除 ✓
- 5：末位是5 ✓

---

### 例题 2：找规律
在□里填什么数字，使 3□7 能被 3 整除？

**解法**：
```
3 + □ + 7 = 10 + □
要能被3整除，10+□必须是3的倍数
□可以是：2（12）、5（15）、8（18）
```

**答案**：2、5或8

---

### 例题 3：综合判断
判断 2016 是否能被 4、6、9 整除？

**解法**：
- 4：16÷4=4 ✓
- 6：2016是偶数且2+0+1+6=9能被3整除 ✓
- 9：2+0+1+6=9能被9整除 ✓

**答案**：都能整除

---

### 例题 4：实际应用
学校组织春游，共有 368 人参加。需要分成若干队，每队人数相等，且不能是1人。可能的每队人数有哪些？

**解法**：
```
找368的因数（除了1和368）
368÷2=184 ✓
368÷4=92 ✓
368÷8=46 ✓
368÷16=23 ✓
```

**答案**：2、4、8、16、23、46、92、184

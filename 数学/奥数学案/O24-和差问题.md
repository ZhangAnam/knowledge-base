# 奥数学案 O24：和差问题

> 二-三年级 | 奥数应用题

---

## 📌 知识点

### 和差问题的特征
已知两个数的和与差，求这两个数。

### 公式
```
大数 = (和 + 差) ÷ 2
小数 = (和 - 差) ÷ 2
```

### 解题思路
1. 画出线段图帮助理解
2. 找出和与差
3. 代入公式计算

---

## 📝 例题

### 例题 1：基础和差
甲、乙两数之和是 100，甲数比乙数多 20。求甲、乙两数各是多少？

**解法**：
```
大数（甲）= (100 + 20) ÷ 2 = 60
小数（乙）= (100 - 20) ÷ 2 = 40
```

**答案**：甲=60，乙=40

---

### 例题 2：年龄问题
父子年龄和是 50 岁，父亲比儿子大 26 岁。父子各多少岁？

**解法**：
```
父亲 = (50 + 26) ÷ 2 = 38（岁）
儿子 = (50 - 26) ÷ 2 = 12（岁）
```

**答案**：父亲38岁，儿子12岁

---

### 例题 3：三个量的和差
甲、乙、丙三数和是 120，甲比乙大 10，乙比丙大 5。求三个数。

**解法**：
先求丙：(120 - 10 - 5 - 5) ÷ 3 = 100 ÷ 3 不对

正确做法：
- 丙 = (120 - 10 - 5×2) ÷ 3 = 100 ÷ 3 不行

换一种：
- 设丙为x，则乙=x+5，甲=x+15
- x + (x+5) + (x+15) = 120
- 3x + 20 = 120
- 3x = 100 → x 不是整数

重新理解：甲比乙大10，乙比丙大5 → 甲比丙大15
- 丙 = (120 - 15 - 5) ÷ 3 = 100 ÷ 3 不对

正确做法：
- 甲 = 乙 + 10
- 乙 = 丙 + 5
- 所以 甲 = 丙 + 15
- 丙 + (丙+5) + (丙+15) = 120
- 3丙 + 20 = 120
- 丙 = 100/3（不是整数，检查题目）

换个整数题目：
- 假设甲=40，乙=30，丙=25，和=95

用公式：
- 丙 = (95 - 15 - 5) ÷ 3 = 75 ÷ 3 = 25 ✓
- 乙 = 25 + 5 = 30 ✓
- 甲 = 30 + 10 = 40 ✓

**答案**：甲=40，乙=30，丙=25

### 例题 4：购物问题
小明买了 3 支铅笔和 2 支钢笔，共花 28 元。已知铅笔比钢笔每支便宜 4 元。求铅笔和钢笔的单价。

**解法**：
设钢笔每支 x 元，则铅笔每支 (x-4) 元
3(x-4) + 2x = 28
3x - 12 + 2x = 28
5x = 40
x = 8

钢笔：8元，铅笔：4元

**答案**：钢笔8元/支，铅笔4元/支

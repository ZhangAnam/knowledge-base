# 奥数学案 O25：和倍问题

> 三年级 | 奥数应用题

---

## 📌 知识点

### 和倍问题的特征
已知两个数的和以及它们之间的倍数关系，求这两个数。

### 公式
```
小数 = 和 ÷ (倍数 + 1)
大数 = 小数 × 倍数
```

### 解题思路
1. 找准1倍数（通常是小数）
2. 画出线段图帮助理解
3. 找出和对应的份数
4. 先求1份，再求几份

---

## 📝 例题

### 例题 1：基础和倍
甲、乙两数之和是 90，乙数是甲数的 2 倍。求甲、乙两数。

**解法**：
```
甲是1份，乙是2份，共3份
1份 = 90 ÷ 3 = 30
甲 = 30
乙 = 30 × 2 = 60
```

**答案**：甲=30，乙=60

---

### 例题 2：倍数不是整数
甲、乙两数之和是 100，甲是乙的 3/5。求甲、乙。

**解法**：
```
甲 : 乙 = 3 : 5
共8份
1份 = 100 ÷ 8 = 12.5
甲 = 12.5 × 3 = 37.5
乙 = 12.5 × 5 = 62.5
```

**答案**：甲=37.5，乙=62.5

---

### 例题 3：年龄问题
父子年龄和是 48 岁，父亲年龄是儿子的 3 倍。父子各多少岁？

**解法**：
```
儿子：1份
父亲：3份
共4份
1份 = 48 ÷ 4 = 12
儿子 = 12 岁
父亲 = 12 × 3 = 36 岁
```

**答案**：父亲36岁，儿子12岁

---

### 例题 4：三个量的和倍
甲、乙、丙三数之和是 180，甲是乙的 2 倍，乙是丙的 2 倍。求三数。

**解法**：
```
丙：1份
乙：2份
甲：2×2=4份
共7份
1份 = 180 ÷ 7 不是整数

重新理解：设丙为x
乙 = 2x
甲 = 2×2x = 4x
x + 2x + 4x = 180
7x = 180 → x 不是整数

换一个题目：
甲、乙、丙三数和为84，甲是乙的2倍，乙是丙的2倍
设丙=x，乙=2x，甲=4x
x+2x+4x=84
7x=84
x=12
丙=12，乙=24，甲=48
```

**答案**：甲=48，乙=24，丙=12

## 🧪 测试题

### 应用题（每题 20 分）

1. 两数之和是 60，大数是小数的 4 倍。求两数。

2. 苹果和梨共 120 千克，梨的重量是苹果的 2 倍。苹果和梨各多少千克？

3. 学校体育用品中，足球个数是篮球的 3 倍，篮球和足球共有 48 个。篮球和足球各几个？

4. 兄弟三人年龄和是 48 岁，大哥是二哥的 2 倍，二哥是三弟的 2 倍。三兄弟各几岁？

5. 书店卖出语文书和数学书共 240 本，数学书的本数是语文书的 3 倍。两种书各卖出多少本？

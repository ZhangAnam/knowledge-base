# 奥数学案 O3-O4：基准数法与变形约分

> 三至五年级 | 奥数计算模块 A1 | 主题：O3 基准数法；O4 变形约分

---

## 本课学什么

这一课包含两个紧密相关的方法：

- **O3 基准数法**：几个数都接近同一个整十、整百、整千时，先按共同基准来算，再统一调整。
- **O4 变形约分**：先把式子改写成能相消、能约分的结构，再快速算出结果。

这两类题的共同点是：**不硬算，先整理结构。**

- 对应知识体系：

  - **O4 变形约分**

## 核心模型与策略

### 一、O3 基准数法

#### 1. 什么叫基准数

如果几个数都靠近同一个数，就把这个共同接近的数叫作**基准数**。

例如：

```text
198，199，201，202
```

它们都接近 200，就可以把 200 作为基准数。

#### 2. 基本思想

先按“每个都等于基准数”来算，再把多算或少算的部分统一调整。

```text
198+199+201+202
=200×4+(-2-1+1+2)
```

#### 3. 适用信号

- 几个数都接近同一个整十、整百、整千；
- 一组连续整数围绕某个中心数；
- 题目不难算，但直接逐个相加比较慢。

### 二、O4 变形约分

#### 1. 什么叫“先变形”

不是见到分数就立刻算，而是先把每一项改写成更有规律的形式。

常见变形：

```text
1+1/n=(n+1)/n
1-1/n=(n-1)/n
```

#### 2. 什么叫“约分”

在乘法里，如果分子和分母有相同因数，就可以约掉公共因数，让式子变简单。

#### 3. 适用信号

- 连乘中的每一项结构很相似；
- 题目里反复出现 `1±1/n`；
- 一长串乘积看起来大，实际上中间能前后相消。

## 解题方法

### O3 基准数法的步骤

1. 先找共同接近的基准数。
2. 把每个数写成“基准数 ± 差值”。
3. 先算“基准数 × 个数”。
4. 再把所有差值合并调整。

### O4 变形约分的步骤

1. 先观察每一项有没有统一结构。
2. 把 `1±1/n` 这类式子化成分数。
3. 连乘时先不急着通算，先看能否前后约分。
4. 最后只保留首尾没有消掉的部分。

### 一个共同提醒

基准数法是**整体看一组数**；  
变形约分是**整体看一串因数**。  
两者都要求先观察，再动笔。

## 易错点/关键提醒

1. **基准数选得不合理。**  
   不是随便选一个数当基准数，而要选大家都接近的那个数，最好是整十、整百、整千。

2. **差值正负写反。**  
   `198` 比 `200` 少 2，要写 `-2`；`203` 比 `200` 多 3，要写 `+3`。

3. **加减法不能直接约分。**  
   约分只发生在乘法结构中，像 `1/2+1/4` 不能直接约分。

4. **把 `1-1/n` 写错。**  
   `1-1/5=4/5`，不是 `1/4`。

5. **前后约分时只看见局部，不看整体。**  
   连乘题真正厉害的地方在于中间大量相消，不要算成一堆大分数再硬乘。

## 递进例题

### 例 1：O3 基准数法入门

计算：`98+101+99+102`

```text
98+101+99+102
=100×4+(-2+1-1+2)
=400+0
=400
```

答案：`400`

### 例 2：O3 基准数法进阶

计算：`198+199+203+201+197`

```text
198+199+203+201+197
=200×5+(-2-1+3+1-3)
=1000-2
=998
```

答案：`998`

### 例 3：O3 用中心数快速求和

计算：`48+49+50+51+52`

```text
48+49+50+51+52
=50×5+(-2-1+0+1+2)
=250
```

答案：`250`

### 例 4：O4 把每一项变形成分数

计算：

```text
(1+1/2)(1+1/3)(1+1/4)(1+1/5)
```

```text
(1+1/2)(1+1/3)(1+1/4)(1+1/5)
=3/2×4/3×5/4×6/5
=6/2
=3
```

答案：`3`

### 例 5：O4 含减号的变形约分

计算：

```text
(1-1/2)(1-1/3)(1-1/4)(1-1/5)
```

```text
(1-1/2)(1-1/3)(1-1/4)(1-1/5)
=1/2×2/3×3/4×4/5
=1/5
```

答案：`1/5`

### 例 6：O4 更长的连乘

计算：

```text
(1+1/2)(1+1/3)(1+1/4)…(1+1/10)
```

```text
=3/2×4/3×5/4×…×11/10
=11/2
```

答案：`11/2`

# 奥数学案 O1：加减法巧算

> 三年级 | 奥数计算模块 A1 | 主题：凑整、配对、带着符号搬家

---

## 本课学什么

这一课学习加减法中的常见巧算方法：凑整、配对、带着符号搬家、去括号后再合并。  
重点不是“算得快”，而是先看式子的结构，再决定怎样整理。学完后应能把一串不整齐的加减算式整理成更容易口算的样子。

## 对应知识体系与前序知识

- 对应知识体系：**O1 加减法巧算（凑整、带着符号搬家）**
- 前序知识：
  - 课内 `12`：100 以内加减法
  - 课内 `23`：万以内加减法
- 后续衔接：
  - `O3 基准数法` 会继续研究“都接近某个整十整百时怎样整体算”
  - `O2 乘法分配律的逆用` 会把这种“先整理再算”的思路推广到乘法

## 核心模型与策略

### 1. 凑整法

看到 `98、99、198、1001` 这类**接近整十、整百、整千**的数时，先把它看成整十整百，再把多算或少算的部分补回来。

```text
99+78=(100-1)+78
```

### 2. 配对法

一串数里如果有两项能正好凑成整十、整百，就先把它们放在一起算。

```text
189-45+55
```

其中 `-45` 和 `+55` 可以先合并成 `+10`。

### 3. 带着符号搬家

同级运算中，可以把加数或减数重新安排位置，但**符号要一起带着走**。

```text
a-b+c=a+c-b
```

这不是乱换位置，而是为了更方便地配对、凑整。

### 4. 去括号合并

当式子里有括号时，要先判断括号前是 `+` 还是 `-`。

- 括号前是 `+`：去括号后，符号不变。
- 括号前是 `-`：去括号后，括号里的每一项都要变号。

```text
156-(56+38)=156-56-38
156-(56-38)=156-56+38
```

## 解题方法

### 方法一：先看哪些数最容易凑整

如果有数接近 `100、1000`，或几项正好能拼成整十整百，优先整理这些部分。

### 方法二：看到减号，先想“它减的是谁”

例如：

```text
500-198-102
```

本质上是从 500 里一共减去 `198+102`，所以可以写成：

```text
500-(198+102)
```

### 方法三：去括号时先处理符号，再计算

很多错误不是算错，而是**符号变错**。去括号后再配对、凑整，会更稳。

### 方法四：算完做快速检查

- 是否比原来的大概范围合理？
- 凑整时补回了吗？
- 去括号时每项符号都检查了吗？

## 易错点/关键提醒

1. **减法不能像加法那样随便交换位置。**  
   例如 `100-36-14` 之所以能写成 `100-(36+14)`，是因为它表示“连续减去两个数”，不是因为减法本身可以交换。

2. **带着符号搬家，符号不能丢。**  
   `189-45+55` 改写时，`-45` 还是减 45，`+55` 还是加 55。

3. **括号前是减号时，去括号必须变号。**  
   `80-(25+7)` 应写成 `80-25-7`，不能写成 `80-25+7`。

4. **凑整后要补回。**  
   `99+78` 把 99 看成 100 时，多算了 1，最后要减 1。

## 递进例题

### 例 1：最基本的凑整

计算：`99+78`

```text
99+78
=(100-1)+78
=100+78-1
=177
```

答案：`177`

### 例 2：减法中的凑整

计算：`156-98`

```text
156-98
=156-100+2
=58
```

答案：`58`

### 例 3：带着符号搬家后配对

计算：`189-45+55`

```text
189-45+55
=189+55-45
=189+(55-45)
=189+10
=199
```

答案：`199`

### 例 4：连续减法先合并

计算：`500-198-102`

```text
500-198-102
=500-(198+102)
=500-300
=200
```

答案：`200`

### 例 5：去括号后再算

计算：`456-(156-44)`

```text
456-(156-44)
=456-156+44
=300+44
=344
```

答案：`344`

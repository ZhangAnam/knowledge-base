# 奥数学案 O1：加减法巧算

> 三年级 | 奥数计算

---

## 📌 知识点

### 1. 凑整法
- 看到接近整十、整百的数，先凑整
- 去掉括号后注意符号变化

### 2. 带着符号搬家
在同级运算中，可以带着符号移动位置：
```
a - b + c = a + c - b
a - b - c = a - c - b
```

### 3. 去括号
- 括号前是加号，去括号不变号
- 括号前是减号，去括号要变号

### 4. 加补数法
把接近整十、整百的数加上它的补数

---

## 📝 例题

### 例题 1：凑整计算
计算：98 + 67

**解法**：
```
98 + 67 = (100-2) + 67 = 100 + 67 - 2 = 165
```

**答案**：165

---

### 例题 2：去括号
计算：156 - (56 + 38)

**解法**：
```
156 - (56 + 38) = 156 - 56 - 38 = 100 - 38 = 62
```

**答案**：62

---

### 例题 3：带着符号搬家
计算：189 - 45 + 55

**解法**：
```
189 - 45 + 55 = 189 + 55 - 45 = 189 + 10 = 199
```

**答案**：199

---

### 例题 4：基准数法
计算：198 + 199 + 197 + 196 + 195

**解法**：
```
以200为基准：
= 200×5 - (2+1+3+4+5)
= 1000 - 15
= 985
```

**答案**：985

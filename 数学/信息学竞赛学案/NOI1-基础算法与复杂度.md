## 📝 典型例题

### 例题1：二分查找
```cpp
int binarySearch(int a[], int n, int x) {
    int l = 0, r = n-1;
    while (l <= r) {
        int mid = (l + r) / 2;
        if (a[mid] == x) return mid;
        if (a[mid] < x) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}
```

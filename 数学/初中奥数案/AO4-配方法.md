# 初中奥数 AO4：配方法

> 初中奥数 | 计算进阶

---

## 📌 知识点

### 配方法
把式子变成完全平方式

### 步骤
1. 提取系数
2. 配常数项：(x + b/2)²

---

## 📝 例题

### 例题 1
x² + 6x + 5 = 0，用配方法

**解法**：
x² + 6x + 9 = 4
(x + 3)² = 4
x + 3 = ±2
x = -1 或 x = -5

**答案**：x = -1 或 -5

---
